Delete Third Party App
 
Related Topics
About 3DCompass Web Services

Role

Removes a third party app from the compass.

Request

 Method  Path 
DELETE /resources/AppsMngt/api/custom/applications/<id>

Responses

Success

The OK response (status code 204) is without body message.

Failure

Besides the common 3DCompass Web Service Failures the specific failures are:

Status Code Status Message Reason
403 Forbidden The connected person is not admin platform.
404 Not Found The third party app does not exist.

They all return a JSON body. The format is described by Specific 3DCompass Failure Cases

Example

##CLIENT REQUEST
##------------------------------------------------------------
[DELETE] https://sdbstelladsy.ux.dsone.3ds.com/3DSpace/resources/AppsMngt/api/custom/applications/MAP-EJBQRVVNU
