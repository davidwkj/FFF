Third Party App Definition
 

This page describes the response format associated with a third party app (TPA). This format is used in the create TPA, list TPA, edit TPA and get TPA API response.

JSON Format

Key Description Value
type A constant value: "custom/applications" String
id The application identifier. A value automatically computed. String
attributes Information about the third party app A JSON structure
Key Description Value
type web|widget|native String
name The name of the application. This name is visible in the Compass. String
tooltip The information when passing over the App icon String
description A free text to describe the application. Not visible interactively. String
icon The URL of an image. String.
config Empty data for native and web app. For a widget it is the widget config file URL. String
launchinfos An URL for a web or widget app. An identifier for a native app. String
exe The custom application name which can be set for a native app. String.
error The custom error message which can be set for a native app. String.
quadrant east|north|west|south String
relationships . A JSON structure
Key Description Value
platform Information related to the platform where the third party app is created. A JSON structure
Key Description Value
data A JSON structure
Key Description Value
type A constant value: "environments" String
id identifier of the platform instance String
attributes Information about the third party app for the platform. A JSON structure
Key Description Value
public
  • false (only the granted people for the app could used it)
  • true (all people granted for the platform could use it)
 boolean

Example

Extract of a third party app web service response: 

{"type":"custom/applications",
 "id":"MAP-DLWRSZSTD",
 "attributes":{
    "type":"widget",
    "name":"CAA My Custom App",
    "tooltip":"The application displaying my 3DPart.",
    "description":"This application has been deployed for RX. Only available for the granted people.",
    "icon":"http://www.example.com/icon.png",
    "config":"",
    "launchinfos":"http://www.example.com/my_custom_app.htm",
    "quadrant":"east"
 },
 "relationships":{
    "platform":{
        "data":{
            "type":"environments",
            "id":"OnPremise",
            "attributes":{
                "public":false
            }
        }
    }
 }
}
