List Third Party Apps
 
Related Topics
About 3DCompass Web Services

Role

Retrieves the third party apps available on the platform instances the connected person is declared as administrator. The result can be limited to a given platform instance.

Request

 Method   Path 
GET /resources/AppsMngt/api/custom/applications

Query Parameters

 Name   Cardinality   Value   Description 
filter 1|0 A string. The identifier of a platform instance. Only the applications defined for this platform instance are returned. In case of invalid identifier, the request is successful with an empty list.

Responses

Success

The OK response (status code 200) is a JSON object with the data key whose the value is an array of JSON objects as described in the Third Party App Definition.

{"data" : [ { apps1 } , {appsI}, {appsN} ] }

If the connected person is granted as member (not admin) for all platform instances of the 3DEXPERIENCE, the list is empty. Same result if the filter value is not a valid platform instance. 

{"data" : [] }

Failure

There is no specific errors raised by this API. For the non specific click: 3DCompass Web Service Failures
