Sweep
Use Case

Abstract

A Sweep surface is an extruded surface generated by the sweeps of  a wire profile along a center curve and constrained by a spine curve which handles the normal to the profile plane . Sweep surface can be created with different type of limitation of each extremity along the extrusion.

Operator to be Used

To create a Sweep Surface, use the CATICGMTopSweep operator in GMOperatorsInterfaces. This operator has to be created by the CATCGMCreateTopSweep global function.

Use Case Description

The CAAGMOperatorsSweep.m module in CAAGMOperatorsInterfaces.edu illustrate how to create a Sweep surface. This use case uses the input data SweepModel.NCGM which is delivered in the InputData file directory and containing data to be passed to the operator. If you are not already familiar with topological operator use cases, go to How to Use Topological Operators.

Case 1: Create an extruded Sweep surface

With the input data below:

Fig.1 Sweep Operation: Input Data (Wire profile, center curve, Spine, Surface lying on center curve)

and the code below:



// Create the Sweep operator
//
CATICGMTopSweep * pCGMSweepOperator = CATCGMCreateTopSweep (
piGeomFactory,
&topdata,
pBodyCenterCurve, 		// Wire body defining the center curve of the sweep ( this is the curve along which the profile is swept)
pCenterSupport, 		// The surface on which the center curve is laying on.
pBodySpine, 			// Wire body defining the spine curve of the sweep: 
pBodyProfile); 			// Wire or the skin body defining the profile of the Sweep.


...
// Run the operator
//
pCGMSweepOperator ->Run();
cout << "Run the operator : OK" << endl;

// Get The result
// 
CATBody *pCGMSweepOperator = pCGMRevolutionOper->GetResult();

pCGMSweepOperator ->Release();
pCGMSweepOperator =NULL;
