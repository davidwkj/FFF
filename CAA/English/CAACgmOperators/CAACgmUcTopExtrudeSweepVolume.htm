Sweep Volume
Use Case

Abstract

A Sweep volume is an extruded volume generated by the sweeps of  a skin profile along a center curve and constrained by a spine curve which handles the normal to the profile plane. Sweep volume can be created with different type of limitation of each extremity along the extrusion.

Operator to be Used

To create a Sweep volume, use the CATICGMTopSweep operator in GMOperatorsInterfaces. This operator has to be created by the CATCGMCreateTopSweep global function.

Use Case Description

The CAAGMOperatorsSweep.m module in CAAGMOperatorsInterfaces.edu illustrate how to create a Sweep volume. This use case uses the input data SweepModel.NCGM which is delivered in the InputData file directory and containing data to be passed to the operator. If you are not already familiar with topological operator use cases, go to How to Use Topological Operators.

Case 1: Create an extruded Sweep surface

With the input data below:

Fig.1  Sweep Operation: Input Data (Skin profile, center curve)

and the code below:

cout << "\n\n==============================================================================================" << endl;
cout << "Part 3 " << endl;
cout << "==========" << endl;
cout << "Create a sweep volume extrusion " << endl;
cout << "---------------------------------- \n" << endl;

// Tag of body used by operator
//
int TagSurface = 611;

// Find body of the profile body surface from its tag
//
CATICGMObject *piCGMObjSurfProfile = piGeomFactory->FindObjectFromTag(TagSurface);
if (piCGMObjSurfProfile == NULL)
{
cout << "No object with the specified tag" << endl;
return 1;
}
CATBody * pBodySurfProfile = (CATBody *)piCGMObjSurfProfile;
cout << "\n - Retrieve the Surface Body of the profile from its tag = " << TagSurface << ": OK" << endl;

// Create the CATICGMTopSweep operator 
//
CATICGMTopSweep * pCGMSweepOperSurf = CATCGMCreateTopSweep(
piGeomFactory,
&topdata,
pBodyProfile,		// Wire body defining the center curve of the sweep ( this is the curve along which the profile is swept)
NULL, 			// the surface on which the center curve is laying on.
NULL, 			// Wire body defining the spine curve of the sweep: 
pBodySurfProfile);      // Skin body defining the profile of the Sweep.


if (pCGMSweepOperSurf == NULL)
{
cout << "Invalid Result : Get The result NULL" << endl;
return 1;
}
cout << "Create the CATICGMTopSweep Operator : OK" << endl;

// Run the operator
//
pCGMSweepOperSurf->Run();
cout << "Run the operator : OK" << endl;

// Get The result
// 
CATBody *pCGMSweepResBodyTemp = pCGMSweepOperSurf->GetResult();
if (pCGMSweepResBodyTemp == NULL)
{
cout << "Invalid Result : Get The result NULL" << endl;
return 1;
}
cout << "Get The Sweep volume body result : OK" << endl;
cout << "\n\t\t\t Tag = " << pCGMSweepResBodyTemp->GetPersistentTag() << " : OK" << endl;
