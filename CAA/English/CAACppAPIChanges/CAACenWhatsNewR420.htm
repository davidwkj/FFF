What's New in 3DEXPERIENCE R2018x
Technical Article

Abstract

New modelers, or enhancements to existing modelers are listed here, compared with the previous release. Some enhancements or maintenance reasons may also require changes regarding tools or methodologies. This article lists them, and points to the corresponding documentation.

Data Exchange Interfaces

Capability provided by the Data Exchange Interfaces modeler is not provided through API any longer, use the CATPLMBatch class to launch the DataExchangePLMBatch utility instead as illustrated by the provided code samples.
