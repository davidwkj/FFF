Uninstalling a Brand API
 

This task explains how to uninstall unused CAA Brand API.

When you stop supporting your applications for a given API V6 release level, the corresponding Brand API installation becomes of no use. Here is the way to uninstall it.
Before you begin: Note that:
  • This uninstalls all the API CD-ROMs you have installed for this release level.
  • You should uninstall the run time part of the same release level afterwards.

  1. On the Start menu, click Control Panel.

    The Control Panel dialog box opens.

  2. In Programs, click Uninstall a program.

    The Uninstall or change a program pane is displayed in the control panel.

  3. Locate Dassault Systemes Software CAA Bxxx, where xxx is the release, such as B421 for 3DEXPERIENCE R2019x.

    You can get the correspondence between the release code level and its marketing name in Authorized API Identification, Usage, Deprecation, and Stability.

  4. Right click Dassault Systemes Software CAA Bxxx, and click Uninstall/Change.

    A prompt opens to ask you whether you want to uninstall.

  5. Click Yes in this prompt.

  6. Another prompt opens to ask you whether you want to uninstall CAA files.

  7. Click OK in this prompt.

  8. An uninstallation prompt opens showing the files progressively removed, and disappears once completed.

    The Dassault Systemes Software CAA Bxxx is removed from your computer.
