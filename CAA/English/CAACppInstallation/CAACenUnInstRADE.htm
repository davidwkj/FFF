Uninstalling Native Apps Development Toolset
 

This task explains how to uninstall unused Native Apps Development Toolset release levels.

When you stop supporting your applications for a given CAA API V5 or V6 release level, the corresponding CAA RADE or Native Apps Development Toolset installation becomes of no use. Here is the way to uninstall it.

  • Uninstalling a V5 CAA RADE Release
  • Uninstalling a V6 ENOVIA Studio/Native Apps Development Toolset Release
Before you begin: Note that:
  • This uninstalls all the CAA RADE or Development Tools products and components, such as mkmk, mkODT, or mkscc, and the run time, but not the CAA APIs for that level.
  • If you just want to clean up your Visual Studio customization to use a later ENOVIA Studio/Native Apps Development Toolset release level, but still want to build and test applications on the given level, run the procedure described in Preparing for Native Apps Development Toolset Installation.

Uninstalling a V5 CAA RADE Release

If the current Visual Studio customization uses the add-ins of the CAA RADE release to uninstall, first clean-up Visual Studio, according to the procedure described for Visual Studio 2005 in Preparing for Native Apps Development Toolset Installation.

Then, or otherwise, uninstall CAA RADE:

  1. On the Start menu, click Control Panel.

    The Control Panel dialog box opens.

  2. In Programs, click Uninstall a program.

    The Uninstall or change a program pane is displayed in the control panel.

  3. Locate Dassault Systemes Software Version 5-6 Release 2014 (B24_RADE), for example for Version5-6 Release2014.

  4. Right click Dassault Systemes Software Version 5-6 Release 2014 (B24_RADE), and click Uninstall/Change.

    A prompt opens to ask you whether you want to uninstall.

  5. Click Yes in this prompt.

  6. Another prompt opens to ask you whether you want to uninstall CAA files.

  7. Click OK in this prompt.

  8. An uninstallation prompt opens showing the files progressively removed, and disappears once completed.

    The Dassault Systemes Software Version 5-6 Release 2014 (B24_RADE) is removed from your computer.

Uninstalling a V6 ENOVIA Studio/Native Apps Development Toolset Release

Uninstalling V6 ENOVIA Studio Release 2012x

  1. Clean Visual Studio

    1. If the current Visual Studio customization uses the add-ins of the ENOVIA Studio release to uninstall, first clean-up Visual Studio, according to the procedure described for Visual Studio 2008 in Preparing for Native Apps Development Toolset Installation.

    2. If the current Visual Studio customization uses the package of the ENOVIA Studio release to uninstall, remove the package according to the procedure described in Installing Native Apps Development Environment for uninstalling Native Apps Development Environment.

  2. If you have requested shortcut creation at installation time, to uninstall these shortcuts:

    1. Click the Windows Start menu.

    2. Click All Programs.

    3. On the Dassault Systemes Software V6R2012x menu, point to Tools.

    4. Right click each shortcut, and click Delete.

    5. When the Tools menu is empty, you can remove the Dassault Systemes Software V6R2012x menu.

  3. Remove the installation folder, that is, by default C:\Program Files (x86)\Dassault Systemes\B212.

Uninstalling V6 ENOVIA Studio/Native Apps Development Toolset Release Before or After 2012x

  1. On the Start menu, click Control Panel.

    The Control Panel dialog box opens.

  2. In Programs, click Uninstall a program.

    The Uninstall or change a program pane is displayed in the control panel.

  3. Locate Dassault Systemes Development Toolset for Native Apps 3DEXPERIENCE R2017x, for example for the 3DEXPERIENCE R2017x release.

    You can get the correspondence between the release code level and its marketing name in Authorized API Identification, Usage, Deprecation, and Stability.

  4. Right click Dassault Dassault Systemes Development Toolset for Native Apps 3DEXPERIENCE R2017x, and click Uninstall/Change.

    A prompt opens to ask you whether you want to uninstall.

  5. Click Yes in this prompt.

  6. Another prompt opens to ask you whether you want to uninstall everything located under the installation folder.

  7. Click OK in this prompt.

  8. An uninstallation prompt opens showing the files progressively removed, and disappears once completed.

    The Dassault Dassault Systemes Development Toolset for Native Apps 3DEXPERIENCE R2017x is removed from your computer.
