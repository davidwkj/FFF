C++ API Reference
The C++ API reference documentation is located in the References section. You can access from there:
  • The list of the frameworks exposing APIs, each of one pointing to the documentation of all its exposed interfaces, classes, global functions, etc.
  • The class/interface inheritance hierarchy.
  • Indexes, and among them, namely:
    • The index of deprecated APIs.
    • The interface and class indexes.
    • The interface/class method index.
    • The global function index.
    • The collection and notification class indexes.

    And other useful indexes about C++ types.
