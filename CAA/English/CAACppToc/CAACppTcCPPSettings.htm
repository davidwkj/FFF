Setting Repository Reference
The setting repository reference documentation is located in the References section. You can access from there:
  • The list of setting repositories, each of them pointing to the documentation of all its exposed setting attributes. See Setting Repository Reference.
  • The primary tab page in the Options dialog box where the setting repository is used is given in Index to Primary Tab Pages.
  • In addition, a Setting Attribute Master Index lists each setting attribute along with its role, the setting repository to which it belongs, and the primary tab page where it appears.

Use the class CATSysSettingRepository to handle settings in your application, and refer to the Settings section to learn about settings.
