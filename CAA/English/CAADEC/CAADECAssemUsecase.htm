Abstract

This article provides information requied for executing use case scenarios.It explains about necessary steps to be followed for usecase execution with assembly data type.

Launching usecases

To launch CAA usecases follow steps metioned below :

  1. Launch MyX-CAD application. Refer to How to Launch MyX-CAD [1].
  2. Create CAA Assembly data by using MyX-CAD application. Refer to How to Use MyXCAD Data type [2].
  3. In order to executes different scenarios of X-CAD, please refer to X-CAD Add-ins [3].
