Before Starting

The CAAPofPipeFeatureNav Use Case

CAAPofPipeFeatureNav is a use case of the CAADELShipMfgItf.edu framework that illustrates the DELMIA Heavy Industry Fluidic Fabrication Interfaces framework capabilities.

What Does CAAPofPipeFeatureNav Do?

This use case:

  • Retrieve fabrication features created by Heavy Industry Fluidic Fabrication app.

How to Launch CAAPofPipeFeatureNav?

Please refer to Heavy Industry Structure Fabrication Use Cases Overview.

The executable name is CAAPofPipeFeatureNav .

mkrun -c " CAAPofPipeFeatureNav Repository Server LoginTicket Process_Name PPR_Name"

* The input model is retrieved by importing the following files:

InstallRootFolder\CAADoc\CAADELShipMfgItf.edu\InputData\PPR_CAABSUTest.3dxml

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

The last 3D XML file contains the root PPR  Reference which is PPR_Name with the following attributes:

  • V_Name: CAA_SYS_MW_CVI System Data
  • Version: ---.000

The last 3D XML file contains the root Assembly Process Reference which is Process_Name with the following attributes:

  • V_Name: CAA_SYS_MW_CVI System Data

Where to Find the CAAPofPipeFeatureNav Code

The CAAPofPipeFeatureNav sample is made of a single source code file named CAAPofPipeFeatureNav.cpp located in the CAAPofPipeFeatureNav.m module of the CAADELShipMfgItf.edu framework: InstallRootFolder\CAADoc\CAADELShipMfgItf.edu\CAAPofPipeFeatureNav.m\ where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This sample deals with the following classes:

DELIPofUseFeatureAccess Interface being used to navigate fabrication features and its attributes.

Retrieving Cutting Features and their Attributes

Retrieve cutting features and their attributes using DELIPofUseFeatureAccess.

...
  CATListValCATBaseUnknown_var ListOfCutting;
  spFeatureAccess->GetFeaturesByType(POF_USE_FT_P_CUTTING, ListOfCutting);
  for(int nCuttingCount = 1; nCuttingCount <= ListOfCutting.Size(); nCuttingCount++)
  {
    CATICkeParm_var spParm = ListOfCutting[nCuttingCount];

    if(spParm != NULL_var)
    {
      CATBaseUnknown_var spUnkFeat = spParm->Value()->AsObject();
      CATISpecObject_var spSOFeat = spUnkFeat;
      if(spUnkFeat != NULL_var && spSOFeat != NULL_var)
      {
        CATListValCATBaseUnknown_var ListOfFeatureAttr;
        spFeatureAccess->GetAttributesFromFeature(spUnkFeat, ListOfFeatureAttr);

        CATUnicodeString sString = "[" + spSOFeat>GetType() + "]";
        ListOfResult.Append(sString);

        for(int nAttrCount = 1; nAttrCount <= ListOfFeatureAttr.Size(); nAttrCount++)
        {
          CATICkeParm_var spParm = ListOfFeatureAttr[nAttrCount];

          if(spParm != NULL_var)
          {
            CATUnicodeString sAttrName, sOutput;

            //Retrieving cutting length value from the feature.
            if(spParm->Name().SearchSubString(POF_USE_FT_P_CUTTING_LENGTH) >= 0)
              sAttrName = POF_USE_FT_P_CUTTING_LENGTH;
            else
              continue;

            //Set the output with attribute names and their values
            sOutput = sString + " : " + sAttrName + " ( " + spParm->Show() + " ) ";

  //Retrieving all the fluidic fabrication features created on this manufactured part
  CATListValCATBaseUnknown_var ListOfAllFeatures;
