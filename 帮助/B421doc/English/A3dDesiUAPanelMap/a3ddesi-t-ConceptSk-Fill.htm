Filling Regions Enclosed by Curves in Freehand Sketches

In freehand sketches, you can fill regions enclosed by sketch curves.

  1. From the Freehand section of the action bar, click Fill .

  2. Optional: Select options in the dialog box.

    Option Description


    		Color Picker. Specifies the color of the curve. The default color is black. In the Color Picker dialog box, select a color on the Color Wheel tab, the Square with Brightness tab, or the History Palette tab. The icon updates to show the selected color.

  3. Click within the region to fill.

  4. Fill additional regions as required.

    You can change options in the dialog box to affect the next region that you fill.
