Creating a Linear Pattern

You can create a linear pattern using one or more features, bodies, or reference geometry.

  1. From the section of the action bar, click Linear Pattern .

  2. Select one or more features, bodies, or reference geometry to pattern.

  3. For Direction 1 Selection :

    1. Select a linear edge or axis for the pattern direction.
    2. For Direction 1 , enter the distance between each pattern instance.
    3. Enter the number of instances for Direction 1.

  4. To create the pattern in a second linear direction, for Direction 2 Selection :

    1. Select a linear edge or axis for the pattern direction.
    2. For Direction 2 , enter the distance between each pattern instance.
    3. Enter the number of instances for Direction 2.

    • To set an equal spacing between each pattern instance in Direction 1, click .

    • To arrange the layout of instances, select from the following:
      • Array : Displays the pattern of all the possible instances in the form of rectangular array.
      • Perimeter: Displays the pattern of instances along the perimeter.
      • Primary: Displays the pattern of instances along two primary directions.
    • To reverse the direction for Direction 1, click .
    • To reverse the direction for Direction 2, click .
    • To create patterns from following types:
      • Geometry pattern: Creates a pattern using only the geometry of the feature.
      • Feature pattern: Creates a pattern using the feature definition of that geometry.
