Creating Linear Patterns of Components

Context:

To create a linear pattern of components:

  1. From the Assembly section of the action bar, click Linear Component Pattern to open the Pattern dialog box.

  2. For Components to Pattern , select one or more components to pattern.

  3. For Direction 1 :

    1. Select a plane, planar face, or linear entity (sketch line, linear edge, axis) to specify the pattern direction.
    2. For Spacing , enter the distance between instances.
    3. For Number of Instances , enter the number of instances, including the seed component, of the component.

  4. (Optional). Repeat the previous step for Direction 2.

  5. In the dialog box, click More Options to specify additional options.
