Updating Library Content

To use the Material Palette, you must populate your library collaborative space with updated material data. Any user who is logged in with the Leader role can perform the update.

After the material data is updated, you can use it in any of your products.

Note: Update the content library after each major software update. All users have access to the updated content.

This task shows you how to:

  • Change User Type
  • Update Library Content

Change User Type

To update your library collaborative space, you must be a leader. You can change your user type if you have permission to do so. If you cannot change your user type, someone with the Leader role must update the content library.

  1. From the top bar of the 3DEXPERIENCE Platform interface, hover over Home and select Manage.

  2. If required, log in using your 3DPassport.

    The 3DSpace tool opens in a Web browser.

    Note: If prompted by your web browser, allow Java Platform to run.

  3. Change your role to Leader.

Update Library Content

Before you begin: You must be logged in with the Leader role to install content in your library collaborative space.

  1. From the Manage section of the action bar, click Update Content Library .

    The Content Library Update dialog box appears.

  2. In the dialog box, click OK.

    The installation process typically takes several minutes.

    Note: A dialog box displays progress. When the dialog box closes, the update is complete.
