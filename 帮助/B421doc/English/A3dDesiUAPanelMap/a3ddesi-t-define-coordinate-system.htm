Defining a Coordinate System

You can define a reference coordinate system by selecting a vertex, edge, or face from thework area.

  1. From the Feature section of the action bar, click Coordinate System .

  2. In the Coordinate System dialog box, click Select Point and select a vertex from the work area to define the origin of the coordinate system.

  3. Select an edge from the work area to define the direction of the X axis.

  4. Select a face from the work area to define the direction of the Y axis.

    Note: The Y axis of the triad aligns normal to the face you select. The direction of the Z axis depends on the positions of X and Y axis.
