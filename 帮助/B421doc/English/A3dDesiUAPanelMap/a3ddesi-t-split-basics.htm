Splitting Bodies or Faces

You can split a body or face into multiple parts using an intersecting solid, face, plane, surface spline, or projected sketch.

This task shows you how to:

  • Split Body
  • Split Face

Split Body

  1. From the section of the action bar, click Split Body .

  2. In the Split Body dialog box, select the bodies from the work area for splitting.

  3. Select the splitting tool such as face, sketch, edge or plane from the work area.

  4. Select an edge or an axis to specify the direction for splitting.

  5. In the Split Body dialog box, select one of the following splitting methods:

    Option Description
    Project

    Splits the selected body with a projected sketch.

    Note: You can specify the splitting direction only when this option is selected.
    Intersect

    Splits the selected body using an intersecting splitting tool such as solid, face, plane, surface spline, or sketch.

  6. Select one of the following projection directions:

    Option Description
    Both Directions

    Projects the splitting tool in both directions.
    Single Direction

    Projects the splitting tool in one direction.

    Note: The projection direction options are available only for Project splitting method.

  7. You can perform following optional steps:

    1. To reverse the direction of the split, click Reverse Direction.

      Note: This option is only available for Single Direction .

    2. To remove a splitting tool if it is a surface body, clickConsume Surface Bodies .

  3. Select the splitting tool such as face, sketch, edge, or plane from the work area.

  4. Select an edge or an axis from to specify the direction for splitting

  5. In the Split Face dialog box, select one of the following splitting methods:

    Option Description

    Splits the selected body with a projected sketch.

    Note: You can specify the splitting direction only when this option is selected.
    Intersect

    Splits the selected body using an intersecting splitting tool such as solid, face, plane, surface spline, or sketch.
    Silhouette

    Creates a split line on a body along an axis.

  6. Select one of the following projection directions:

    Option Description
    Both Directions

    Projects the splitting tool in both directions.
    Single Direction

    Projects the splitting tool in one direction.

    Note: The projection direction options are available only for Project splitting method.

    1. To reverse the direction of the split, click Reverse Direction.

      Note: This option is only available for Single Direction .

    2. To remove a splitting tool if it is a surface body, clickConsume Surface Bodies .
