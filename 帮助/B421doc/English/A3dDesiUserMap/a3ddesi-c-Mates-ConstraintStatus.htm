Constraint Status in the Design Tree

Prefix icons in the design tree indicate whether the position of each assembly component is fixed, fully defined, or under defined.

Icon Constraint Status Description


Under defined Identifies components with at least one degree of freedom.
(blank) Fully defined Identifies components with no open degrees of freedom.


Fixed Identifies components that are held in position by a fixed constraint. To remove the constraint, right-click the component and select Float Component.
