Mate Options

You can edit mates options such as the mate type, mate alignment, and mate selections in the Mate dialog box.

Depending on the mate type, you can edit other options, such as the distance value for a Distance mate.

  • Options

To access the dialog box, in the design tree, select a mate and click Edit Feature .

Options



 Mate Type. Specifies the mate type. (The current mate type appears on the button.) Click the down arrow to select a different mate type.
Flip Mate Alignment. Changes the orientation between the two mated components by flipping the mate alignment.
