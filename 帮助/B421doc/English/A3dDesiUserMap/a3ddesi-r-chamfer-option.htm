Chamfer Options

You can adjust the distance and propagation to all tangent faces when creating chamfers.



Type



 Distance Angle. Creates chamfer geometry by specifying a distance and an angle.


Distance Distance. Creates chamfers by specifying with variable radii values. You can use control points to help define the chamfers.


Symmetry. Creates chamfer geometry by two identical distance values.



Reverse Direction. Changes the chamfer references face.


Tangent Propagation. Extends the chamfer to all faces that are tangent to the selected face.





Tangent Propagation selected Tangent Propagation cleared

Cap Corners. Chamfers corners where three or more chamfered edges meet.

Cap Corners selected Cap Corners cleared



Keep Features. Retains intersecting cut or boss features beyond the chamfer. Clear Keep Features to cover cut or boss features with the chamfer.





Keep Features selected Keep Features cleared



Show All Flips. Displays all flip handles in the work area.
