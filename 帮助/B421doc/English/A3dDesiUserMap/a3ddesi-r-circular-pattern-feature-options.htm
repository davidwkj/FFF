Circular Pattern Options

You can set the circular spacing to create a pattern.



Equal Spacing. Specifies equal spacing between instances of the selected geometry to pattern.


Seed at Center. Places the selected geometry to pattern at the center of the specified instances.


Align with Seed. Aligns each instance to match the alignment of the selected geometry to pattern.
Reverse axis direction. Reverses the direction of the axis.


Show Instance Selection Controls. Displays a control in the work area for each instance of the pattern. To skip an instance, click its control.
  Components to Pattern. Specifies the components to include in the pattern. Select one or more components.


Axis of Rotation. Specifies the axis to rotate about. Select an axis, cylindrical face, linear edge, or line.


Angle. Specifies the angle between the centers of the instances. You can enter a value or drag the handle to set the angle.
  Number of Instances. Specifies the number of instances in the pattern (including the seed component).
