Circular Component Pattern Options

You can specify options for circular component patterns in the Pattern dialog box.

  • Selections
  • Options

From the Assembly section of the action bar, click Circular Component Pattern .

In the Pattern dialog box, you can change the pattern by selecting one of the following options:

Linear
Circular

Selections



 Components to Pattern. Specifies the components to include in the pattern. Select one or more components.


Axis of Rotation. Specifies the axis to rotate about. Select an axis, cylindrical face, linear edge, or line.


Spacing Angle. Specifies the angle between the centers of the instances. You can enter a value to specify the angle.


Number of Instances. Specifies the number of instances, including the seed component, in the pattern.

Options

In the dialog box, click More Options to access these options.



 Reverse Axis of Rotation. Reverses the rotation direction of the pattern instances about the rotation axis.


Display Instance Selection Controls. Displays a control on each instance of the component. Click a control to remove the instance from the pattern. Click the control again to add the instance.
