Extrude Types

You can select different types of extrudes.

Solid. Creates a solid extrude.
Thin. Creates an extruded feature with a constant wall thickness. To specify the wall thickness, drag the handle or enter a value in the dialog box.



 Midplane. Specifies the wall thickness from the extrude to be equal in both directions.



Surface. Creates a surface extrude with a wall thickness defined as zero.
