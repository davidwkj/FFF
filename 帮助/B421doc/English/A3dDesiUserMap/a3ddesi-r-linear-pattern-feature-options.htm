Linear Pattern Options

You can specify the linear spacing in two directions to create a pattern.



Equal Spacing 1. Specifies equal spacing between each pattern instance in Direction 1.


Equal Spacing 2. Specifies equal spacing between each pattern instance in Direction 2.


Array. Places the seed component in an array.


Perimeter. Places the seed component at the perimeter.


Primary.


Reverse Direction 1. Reverses the direction of the pattern in Direction 1.


Reverse Direction 2. Reverses the direction of the pattern in Direction 2.


Skip Instances. Displays a control on each instance of the component. Click a control to remove the instance from the pattern. Click the control again to add the instance back in.
