Linear Component Pattern Options

You can specify options for linear component patterns in the Pattern dialog box.

  • Selections
  • Options

From the Assembly section of the action bar, click Linear Component Pattern .

In the Pattern dialog box, you can change the pattern by selecting one of the following options:

Linear
Circular

Selections



 Components to Pattern. Specifies the components to include in the pattern. Select one or more components.


Direction 1. Specifies the first direction for the pattern. Select a plane, planar face, linear edge, axis, or sketch line.


Spacing. Specifies the distance between the centers of the instances for Direction 1. You can enter a value to specify the distance.


Number of Instances. Specifies the number of instances, including the seed component, in the pattern for Direction 1.


Direction 2. (Optional). Specifies the second direction for the pattern. Select a plane, planar face, linear edge, axis, or sketch line.



Spacing. Specifies the distance between the centers of the instances for Direction 2. You can enter a value to specify the distance.


Number of Instances. Specifies the number of instances, including the seed component, in the pattern for Direction 2.

Options

In the dialog box, click More Options to access these options.



 Reverse Direction 1. Reverses the direction of the pattern in Direction 1.


Reverse Direction 2. Reverses the direction of the pattern in Direction 2.


Display Instance Selection Controls. Displays a control on each instance of the component. Click a control to remove the instance from the pattern. Click the control again to add the instance.
