Make Component Options

When creating a product from one or more selected items in the current product, you make selections and specify options in the Make Component dialog box.

  • Options
  • Selections
  • Coordinate System

To open this dialog box:

  • From the Assembly section of the action bar, click Make Component .

Options



 Coordinate System. Lets you define the coordinate system of the component. If left undefined, the assembly's coordinate system is used.

Selections



Body or Feature Selections. Creates a component from the bodies containing the selected features.
Â  Name. Specifies the name for the new component. Enter a name, or leave blank to accept the default name.

Coordinate System

Available when Coordinate System is selected.



 Origin Selection. Defines the origin of the component. Select a vertex, face, or circular edge.


X Direction Selection. (Optional). Defines the X axis of the component. Select an edge, plane, or planar face.


Y Direction Selection. (Optional). Defines the Y axis of the component. Select an edge, plane, or planar face.


Z Direction Selection. (Optional). Defines the Z axis of the component. Select an edge, plane, or planar face.


Reverse Direction. Reverses the direction of the axis.

Note: You can use the handles of the Robot to adjust the linear, rotational, and planar location of the coordinate system.
