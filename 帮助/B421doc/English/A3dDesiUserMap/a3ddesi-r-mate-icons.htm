Mate Icons

Icons are used to indicate the mate type.

Icons appear as buttons on the context toolbar when you are adding a mate between two components. For existing mates, similar icons appear in the Mate folder in the design tree.

To add a mate between two components, Ctrl + select the geometry to mate from each component. On the context toolbar, hover the mouse pointer over the buttons to preview the mates, and then click to apply the mate.

Note: Mate types that have two possible alignments have two buttons so you can preview both alignments.
