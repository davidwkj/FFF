Revolve End Conditions

You can select a different end condition for revolve Direction 1 and revolve Direction 2.

In the Revolve dialog box, specify end conditions for Direction 1 and Direction 2 .

Not all end conditions are available for every type of revolve.

Direction
1 2  
 

Off. Turns off Direction 2.


  Full Revolve. Extends the revolve feature 360º.




Blind. Extends the revolve feature to the specified angle. To specify the angle, enter a value in the dialog box. To reverse the revolve direction, click the axis handle.


  Midplane. Extends the revolve feature from the sketch plane the same amount in both directions. To specify the angle, enter a value in the dialog box.





Up To Geometry. Extends the revolve feature to the selected plane, feature, or sketch entity.




Up To Next. Extends the revolve feature to the next body.





Up To Body. Extends the revolve feature to the selected body.
