Sweep End Conditions

You can offset the sweep relative to the path at the beginning or end of the path.

Offset Options
None

Positive

Negative

To create an offset:

  • Drag the handle in the work area
  • Enter a positive or negative value in offset fields in the dialog box. A positive value extends the sweep beyond the end of the selected path. A negative value stops the sweep before it reaches the end of the path.

Sweep Offset Options




Percentage. Offsets the start or end of the sweep by the specified percentage of the length of the path.




Length. Offsets the start or end of the sweep by the specified distance.




Up to Geometry. Extends the start or end of the feature to the selected plane, feature, or sketch entity.

Note: To extend the sweep along a curve, in the dialog box, select Maintain Curvature .
