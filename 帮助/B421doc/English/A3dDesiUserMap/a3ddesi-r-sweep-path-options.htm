Sweep Profile Orientation Options

You can control the orientation of the profile as it sweeps along the path.



Tangent Propagation. Includes tangent edges in the sweep profile.


Follow Path. Keeps the profile at the same angle with respect to the path.


Keep Normal Constant. Keeps the profile parallel to the plane of the profile sketch.


Twist With Angle. Twists the profile along the path using the specified angle. Specify the twist angle by entering a positive or negative value in the Sweep dialog box.


Keep Alignment. Keeps the profile aligned to a selected direction vector. To define the direction vector, select a plane, planar face, or line for Alignment Direction Selection.
