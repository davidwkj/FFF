Assembly Section

The Assembly section provides commands for working with assemblies of parts.

The commands are listed below.

  • Flyout for Insert Component
  • Flyout for Component Pattern
New Physical Product
Opens a new tab, where you can create a new model.
See Creating a Physical Product.


Insert Existing Component
See Flyout for Insert Component.


Mirror Component
Creates opposite-hand versions of selected components by mirroring about a selected plane.
See Mirroring a Component.


Linear Component Pattern
See Flyout for Component Pattern.


Replace Component
Replaces the selected components with another component.
See Replacing Components.
Make Component
Creates a product from selected bodies and features.
See Making Components from Bodies.

Flyout for Insert Component
Insert Existing Component
Inserts an existing product as a component of the current product.
See Inserting Existing Components.


Insert New Component
Inserts a new empty product as a component of the current product.
See Inserting New Components.

Flyout for Component Pattern
Linear Component Pattern
Adds multiple instances of one or more components, spaced uniformly along one or two linear paths.
See Creating Linear Patterns of Components.


Circular Component Pattern
Adds multiple instances of one or more components, spaced uniformly around an axis.
See Creating Circular Patterns of Components.
