Freehand Section

The Freehand section provides commands for creating and modifying freehand sketches.

The commands are listed below.

Exit Freehand
Exits the active freehand sketch and commits your changes.
See Sketching Freehand Curves.
Draw Freehand
Allows you to draw free handed, without guides or visual aids.
See Sketching Freehand Curves.
Freehand Curves
Projects a concept curve onto the active sketch support.
See Sketching Freehand Curves.
Fill
Fills regions enclosed by sketch curves.
See Filling Regions Enclosed by Curves in Freehand Sketches.
Erase
Erases portions of curves.
See Erasing and Deleting Curves from Freehand Sketches.
Delete
Deletes entire curves.
See Erasing and Deleting Curves from Freehand Sketches.
Create Reflection
Reflects entities about a line or plane.
See Reflecting Curves.
Drag
Translates and rotates entities using click and drag.
See Moving Sketch Entities in Freehand Sketches.
