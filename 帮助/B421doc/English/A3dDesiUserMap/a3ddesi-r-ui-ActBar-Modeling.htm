Scale
Scales selected solid or surface bodies about a selectable origin.

Convert to Direct Edit
Makes the selected item editable without rolling back or editing the associated sketch. Displays a Robot to resize or reshape the selected item.
See Converting a Feature to Direct Edit.

Flyout for Creating Reference Elements

Reference Plane
Creates a reference plane.
See Creating a Reference Plane.
Reference Axis
Creates a reference axis.
See Creating a Reference Axis.
Reference Point
Creates a reference point.
See Creating Reference Points.
Coordinate System
Creates a coordinate system specified by various inputs.
See Defining a Coordinate System.

Flyout for Creating Extrude Features

Extrude
Creates a solid, thin, or surface extrude.
See Creating an Extrude or Extrude Cut.
Extrude Cut
Creates a cut extrude.
See Creating an Extrude or Extrude Cut.

Flyout for Creating Revolve Features

Revolve
Creates a solid extrude by revolving a two-dimensional entity about an axis.
See Creating a Revolve or Revolve Cut.
Revolve Cut
Creates a solid extrude by revolving a two-dimensional entity about an axis.
See Creating a Revolve or Revolve Cut.

Flyout for Creating Sweep Features

Sweep
Creates a solid extrude by revolving a two-dimensional entity about an axis.
See Creating a Sweep Feature.
Sweep Cut
Creates a solid extrude by revolving a two-dimensional entity about an axis.
See Creating a Sweep Feature.

Flyout for Creating Loft

Loft
Creates a solid, thin, or surface loft by moving profiles along guide curves.
See Creating Loft and Loft Cut.


Loft Cut
Creates solid, surface, and think loft cut by moving profiles along guide curves
See Creating Loft and Loft Cut

Flyout for Fillet
Fillet
Creates a rounded internal or external face on selected edges, faces, or a vertex.
See Creating Fillets.


Chamfer
Creates a beveled feature on selected edges, faces, or a vertex.
See Creating a Chamfer.

Flyout for Draft

Draft
Taper faces using a specified angle to selected faces.
See Creating a Draft Feature.


Shell
Creates an open shell or closed, hollow body from the selected body.
See Creating a Shell of Uniform Thickness.


Thicken
Thickens surface faces or bodies by a specified distance.
See Thicken a Surface.

Flyout for Linear Pattern

Linear Pattern
Creates linear patterns using features, bodies, or reference geometry.
See Creating a Linear Pattern.


Circular Pattern
Creates circular patterns using features, bodies, or reference geometry.
See Creating a Circular Pattern.

Flyout for Combine
Combine
Combines geometry to create a new part.
See Combining Bodies.


Split
Splits a body into multiple pieces.
See Splitting Bodies or Faces.

Flyout for Move Face

Move Face
Translates, rotates, or offsets faces.
See Moving Faces.
Move Body
Moves selected solid or surface bodies with respect to the Robot.
See Moving a Body.

Flyout for Delete Face

Delete Face
Deletes, heals, or replaces one or more faces from a surface body or solid body.
See Deleting Faces.
Delete Body
Deletes a solid body.
See Deleting Bodies.
