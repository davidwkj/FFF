Creating a Surface Sweep

You can use a sweep feature to add or remove material as you drag a profile along a path. Before you create a sweep feature, create the profile and the path.

Context:

Profile
You can define the profile from sketch entities, sketch regions, or planar faces. Profiles must be closed for solid sweep features. Profiles can be open or closed for thin and surface sweep features.
Path
You can define an open or closed sweep path from sketched curves contained in a sketch, or from model edges.

  1. From the Surfaces section of the action bar, click Sweep .

  2. Select profiles to sweep.

  3. Click Select Sweet Path , and select the path from the work area to sweep along.

  4. To remove material with the sweep, select Cut .

  5. To extend the path along tangent entities, select Tangent Propagation .

  6. Specify other options in the dialog box to refine the definition of the sweep feature.
