Editing Mates

You can edit mates to change the mate type, flip the mate alignment, and change the mate selections.

  1. In the design tree, select the mate and click Edit Feature .

  2. In the dialog box, do one or more of the following:

    • To change the mate type: Select a new mate type from the dropdown button .
    • To flip the mate alignment: Click Flip Mate Alignment .
