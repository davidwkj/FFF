Finding Companion Components

You can search for components that have mates with the selected component in the current product or other products. You can select from the search results to insert components.

  1. In an assembly, right-click a component and click Suggest Components.

    A search is launched to find other assemblies that contain the selected component, and determine what other components are mated to the selected component in those assemblies.

  2. Select from the search results to insert components.
