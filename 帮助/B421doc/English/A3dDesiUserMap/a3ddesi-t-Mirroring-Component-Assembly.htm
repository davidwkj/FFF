Mirroring a Component

You can create opposite-hand versions of selected components by mirroring about a selected plane. Optionally, you can constrain the original and opposite-hand components to remain symmetrical about the plane.

  1. From the Assembly section of the action bar, click Mirror Component .

  2. For Mirror Plane , select a plane or planar face.

  3. For Components to Mirror , select one or more components from the main level of the current assembly.

  4. To keep the components symmetrical to each other about the mirror plane, click Symmetric Constraint . Otherwise, the components can move independent of each other.
