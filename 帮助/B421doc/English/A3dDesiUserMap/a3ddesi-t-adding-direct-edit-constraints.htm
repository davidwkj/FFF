Adding Direct-Edit Constraints

You can add 3D constraints between direct-edit faces.

Constraints that you apply between direct-edit features are persistent. Direct-edit feature constraints are maintained during subsequent editing operations.

A constraint that you apply between a direct-edit feature and a parametric feature does not persist through subsequent editing operations. The constraint is used for positioning only.

  1. Ctrl + select two direct-edit faces.

    Valid constraint selections appear on a context toolbar.

  2. Click a constraint.
