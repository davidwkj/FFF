Creating a Chamfer

You can use the chamfer command to create a beveled feature on selected edges, faces, or vertices.

  1. From the Modeling section of the action bar, click Chamfer .

  2. In the Chamfer dialog box, under Type, select Distance Angle , Distance Distance , or Symmetry .

  3. To create a distance or angle chamfer, select an edge or face.

  4. To create a vertex chamfer, select a vertex.

  5. In the work area, drag the handles or enter values to specify the size of the chamfer.
