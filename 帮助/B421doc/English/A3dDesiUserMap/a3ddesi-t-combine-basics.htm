Combining Bodies

You can combine geometry to create a part.

  1. From the Modeling section of the action bar, click Combine .

  2. Select the type of combine to perform:

    Option Description
    Add. Defines a new part by combining the selected bodies into a larger body.
    Subtract. Removes selected bodies.
    Common. Defines a new part from the intersection of the selected Target Bodies and the selected Tool Bodies.
