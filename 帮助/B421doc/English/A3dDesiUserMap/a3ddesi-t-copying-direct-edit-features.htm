Copying Direct-Edit Features and Bodies

You can copy direct-edit features and bodies using the Robot.

Note: You can copy only solid features and bodies. You cannot copy surfaces.

Before you begin: You must have a body or feature already converted to direct-edit.

  1. Select a direct-edit face in the work area or select a direct-edit feature or body in the design tree.

  2. From the Direct Edit toolbar, click Move .

    The Robot appears on the selected feature.

  3. Press Alt and drag the handles of the Robot to copy of the direct-edit feature.

  4. Release the handle.
