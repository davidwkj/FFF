Creating a Shell of Uniform Thickness

You can use the shell tool to hollow out a model, remove faces that you select, and create thin-walled features on the remaining faces.

  1. From the Modeling section of the action bar, click Shell .

  2. In the work area, select faces to remove.

  3. To change the wall thickness, drag the handle or enter a value.

  4. Specify options in the Shell dialog box, and click .
