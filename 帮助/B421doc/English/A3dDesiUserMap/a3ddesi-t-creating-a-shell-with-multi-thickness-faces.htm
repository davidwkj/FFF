Creating a Shell with Faces of Varying Thickness

You can create shell features with different thicknesses for different faces. You can vary the shell thickness for multibody models.

  1. From the Modeling section of the action bar, click Shell .

  2. In the work area, select the faces to remove.

  3. To change the wall thickness, drag the handle, or enter a value for Thickness .

  4. In the Shell dialog box, enter a value for Custom Thickness .

  5. In the work area, select a face to apply a different thickness to.

  6. Change the wall thickness as required.

  7. Select other options in the Shell dialog box and click .
