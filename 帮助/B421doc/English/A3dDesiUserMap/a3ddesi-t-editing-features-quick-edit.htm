Editing Features with Quick Edit

You can edit parametric features without rolling back the design tree or regenerating. Quick edit opens the feature dialog box without the selection boxes.

Quick edit also displays sketch dimensions for editing. As you edit, the software shows how your changes affect dependent features.

  1. In the work area, Shift + select a face.

    The feature dialog box and handles appear.

  2. Make changes using the feature dialog box and the controls in the work area.

    The product updates as you make changes.

  3. Click in the work area to dismiss the feature dialog box.
