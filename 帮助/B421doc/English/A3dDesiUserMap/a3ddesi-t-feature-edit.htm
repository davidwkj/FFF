Editing Features

You can use Edit Feature to edit the feature's properties. Use the selections in the feature dialog box to make your changes.

  1. Do one of the following:

    • Double-click a face of the feature.
    • Click the feature selection breadcrumb and click Edit Feature .
    • Click a face of the feature and click Edit Feature .
    • In the Feature Explorer, right-click the feature and click Edit Feature .

    The feature dialog box opens and the design tree rolls back to the feature.

  2. Make changes using the feature dialog box and the controls in the work area.

  3. Click .

    Your changes are updated in the model.
