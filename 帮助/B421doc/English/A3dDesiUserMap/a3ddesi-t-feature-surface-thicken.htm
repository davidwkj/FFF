Thicken a Surface

You can thicken a body or face on a sketch object.

  1. From the Modeling section of the action bar, click Thicken .

  2. In the Thicken dialog box, select the option to thicken faces or bodies.

    Option Description


    		Thicken Faces. Select faces to thicken.
    Thicken Bodies. Select bodies to thicken.

  3. Select the faces or bodies to thicken.

  4. Specify the distance of the thickened face or body. Drag the handle or enter a value in the dialog box.

  5. Optional: Select additional options.

    Option Description
    Cut. Removes material by the value that you enter from the solid body.
    Blind. Extends the thickness outwards for the specified distance.
    Midplane. Extends the thickness for the specified distance from the sketch plane equally in both directions.


    		Blind Disabled. Disables the second thickness option.
    Blind. Extends the thickness inwards for the specified distance.
    Reverse Thicken Direction. Reverses the direction of the thickened face or body.
    Auto Merge. Merges the thickened objects with the selected bodies.
    Offset Disabled. Disables the offset option.
    Offset. Offsets the selected face or body by the specified length.
    Reverse Offset Direction. Reverses the direction of the offset face or body.
