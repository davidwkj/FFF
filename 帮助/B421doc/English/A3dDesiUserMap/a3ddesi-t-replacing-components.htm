Replacing Components

In an assembly product, you can replace one component with another.

Context:

For example, if a colleague designs a part to replace one that is used in your assembly model, you can substitute the current part with the new one.

  1. From the Assembly section of the action bar, click Replace Component .

  2. For Components to be Replaced , select one or more components to be replaced.

  3. Select a replacement component by doing one of the following:

    • In Replacement Component , select from the list of open products.
    • Click Search and then, from the top bar, search for a component.

  4. In the dialog box, set options:

    Option Description
    Solve Mates. Solves existing mates using the replacement component. The Replace Mate References dialog box opens so you can replace broken mate references with references (such as faces and axes) from the replacement component.
    Replace all instances. Replaces all instances of the component. Otherwise, only the selected instance is replaced.

  5. Click .

    The component is replaced.
