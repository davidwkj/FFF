Starting a Sketch

You create a sketch on a plane or a planar face. For more information on working with the Sketcher application, see 3D Modeling Core: Sketcher.

  1. From the Sketch section of the action bar, click Positioned Sketch .

    The pointer changes to indicate that you need to select a plane to sketch upon.

  2. Specify the sketch plane by selecting one of the following in the work area:

    • A default plane (Top, Front, or Right).
    • A planar face of the model.
    • A previously created reference plane.

    A sketch opens in a Sketcher application and a grid appears on the selected plane.

  3. From the Sketch section of the action bar, select sketch entity command such as Line , Rectangle .

  4. While you sketch in the work area, you can do one of the followings:

    Option Description
    Undo

    Reverses the last action.
    Undo/Redo Overview

    Provides you a list of actions that you can reverse or recover.
    Redo

    Recovers the last action.
    Normal View

    Displays sketch objects perpendicular to the selected plane.
    Exit App

    Closes Sketcher application after saving your sketch work. Allows you to apply features to your sketch by enabling Modeling commands.

    Note: You can identify an active sketch, inactive sketch or over defined sketch according to their color representation. A Blue color indicates an active sketch, a Gray indicates an inactive sketch and, Red indicates an over defined sketch.
