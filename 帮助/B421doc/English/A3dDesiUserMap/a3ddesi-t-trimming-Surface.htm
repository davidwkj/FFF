Trimming a Surface

You can trim sketch objects or body surfaces.

  1. From the Surfaces section of the Action Bar, click Trim Surface .

  2. In the Trim Surface dialog box, select the trim option.

    Option Description
    Standard .

    Trims the selected surfaces. You can select multiple surfaces to trim.
    Mutual .

    Trims the intersection diagonally crossed surfaces.

    The dialog box displays the options depending on your selection in this step.

  3. Click Surfaces to be trimmed and select surface bodies from the work area for trimming.

  4. Click Select items to trim with and select sketch object from the work area to trim the selected surface.

    You can select any curve, line, or a surface as a trimming tool.

  5. Set the following options:

    Option Description
    Extension type

    Extends a tool trim in a linear, tangent, or natural direction and makes all the possible regions available for your to select for trimming.
    Region selection

    Displays or hides the regions that you select for trimming or retaining.
    Make Solid

    Knits one or more surfaces to make a single solid body.
    Knit

    Knits one or more surfaces to make a single surface body.
    Knitting tolerance

    Knits the surfaces that are placed at a distance of 0.001mm to 0.1mm from each other.
