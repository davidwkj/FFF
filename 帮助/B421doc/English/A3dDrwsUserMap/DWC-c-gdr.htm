Export of a Drawing Representation

Conversion of a Drawing Representation

Two cases are possible depending on whether the target V5 CATdrawing file exists or not.

Case of the Creation of a New CATDrawing

When running the Downward Compatibility batch to convert a V6 drawing representation, in the case when the target file does not exist yet, then a new V5 CATDrawing file is created.

Case of the Synchronization of an Existing CATDrawing

When running the Downward Compatibility batch to convert a V6 drawing representation, in the case when the target V5 CATDrawing file already exists, then the existing file is replaced with a newly created CATDrawing. Any existing V5 element in the previous CATDrawing document is lost.

Note: Elements in other V5 documents pointing to the old CATDrawing document might be broken.

CATDrawing Structure

The resulting CATDrawing has no link with an external 3D part representation or a 2D Component catalog. As a consequence:

  • Generated views are converted into the interactive views.
  • Detail sheets are converted into the design sheets.

Symbols displayed in the tree are modified accordingly.

View Content

Result objects are used to convert a V6 representation to a V5 document. Every V6 view, sheet background and main view is converted into one result object. This object displays geometries, annotations and dress-up in a CATDrawing view. As a consequence, this CATDrawing document displays elements as in a print preview. It means that the following elements are not displayed:

  • Constraints
  • Construction elements

Notes:

  • You can add non-associative annotations to a converted view. You can also add new sheets and new views to this CATDrawing.
  • The drawing content is exported only as graphic objects. As a consequence, the individual objects like annotations, dimensions, dress-up objects, or 2D geometry exported from V6 are not editable in V5 as individual objects.

Copy/Cut/Paste

It is impossible to copy/cut/paste the content of a view.

Graphical Properties

Graphical properties are not converted from the V6 drawing representation to the V5 CATDrawing document.

Standards

If a drawing representation uses a standard that does not exist in the target V5 release, the batch replaces the V6 standard with a V5 standard of the same type (ISO, JIS...).

Object Linking and Embedded (OLE)

When an object is inserted in a drawing representation and displayed as an icon, this icon is converted from V6 to V5 as an image. After conversion, this OLE displayed as an icon has no more link with the Windows document.

Transfer Views in a Layout

A drafting view created with the Downward Compatibility batch cannot be exported to a 2D layout.

Export to External Formats

A view converted with the Downward Compatibility batch:

  • Can be exported into CGM, HPGL2, JPEG, PDF, PS, SVG, TIFF (print-like formats)
  • Cannot be exported into DXF, DWG, CDD, IG2 (CAD formats)

Knowledgeware

  • Formulas used to position views are deleted.
  • Formulas used to name a view are deleted. In the Properties dialog box, under View Name, the Name Editor with Formula option cannot be modified.
