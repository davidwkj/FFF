Export of a Drawing Representation

You can convert a 3DEXPERIENCE platform drawing representation into a V5 CATDrawing document, using the Downward Compatibility batch.

Note: For more information about the Downward Compatibility batch, see File Coexistence and Migration User's Guide: Downward Compatibility: Running the Downward Compatibility Batch.

Before you begin: Open a 3DEXPERIENCE platform session.

  1. From the Tools section of the action bar, click Utility.

    The Batch Monitor window opens.

  2. Double-click DownwardCompatibility from the list.

    The DownwardCompatibility dialog box appears.

  3. In the Convert list, select the target release for conversion (V5R18 SP4 or higher).

  4. Click .

    The Search dialog box appears.

  5. Define your query, and then click Apply.

    The search process starts.

  6. Click Save to save your batch parameters in an XML file.

  7. Click Run to start the batch.
