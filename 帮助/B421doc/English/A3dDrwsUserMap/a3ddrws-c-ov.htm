Overview

Drawings is a classic app that provides you powerful functionality to generate drawings from 3D parts and assembly definitions (such as flexible assemblies).

Drawings

Drawings enables you to generate views from 3D, known as generative views which are associative to the 3D. You can generate dimensions in views, which are also associative.

Drawings provides the following key capabilities:

  • Create drawings containing views which result from specifications in 3D parts, 3D products, and even .model documents.
  • Generate drawings of varying levels of complexity.
  • Apply dimensions, annotations and dress-up elements to the drawings.
