About Dimension Conversion

Dimensions can be imported in several different ways.

Options for dimension conversion are found in the DXF 2D tab of the Preferences.

Convert dimensions as can be set to the following values:

  • Dimensions: Preserves the semantic of AutoCAD dimensions in 3DEXPERIENCE as best as possible.
    • In most cases, the whole semantic of the dimension is kept. This means the position, the layout and the text are preserved. The position, color, thickness, text caption (symbol, value, tolerance, font, color) can be edited.
    • In some cases, text with redundant information (e.g. tolerance present both in the dimension field and in the dimension text) or dual dimension, or other unrecognized information is dealt with as a text with an associative link with the dimension. This dimension has a "fake value" that is blanked. To display the true dimension value, delete the associated text and enter the data in the properties of the dimension.
    Important: With the Dimensions option:
    • Problems may occur while positioning texts for Angular dimensions
    • Following attributes are not yet supported :
      • Suppression of one of the dimension lines (for half dimensioning)
      • Offset extension or suppression of extension lines
      • Arrowhead choice (the 3DEXPERIENCE default arrowhead is used)
      • Automatic suppression of arrowheads if space is not sufficient.
  • Geometry: Keeps the graphical aspect. Geometry is exploded into multiple lines, arcs, texts. (this mode increases performance when loading a model).
  • Details: Turns dimensions into details. This is an halfway notion between the previous two, in which the geometry is preserved and the dimension is easy to handle (it can be all selected at once).
