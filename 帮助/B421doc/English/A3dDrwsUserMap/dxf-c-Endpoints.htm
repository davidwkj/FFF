About End Points

End points are used to modify or stretch geometry.

Options for end points are found in the DXF 2D tab of the Preferences.

It is not easy to modify and stretch geometry of imported elements the way you can do it in native elements. A solution is to create end points when needed, but to the detriment of performance. Create end points offers you three options to fit your needs:

  • Never ensures the best performance.
  • For few entities creates end points only for hatch boundaries and mixed polylines. This is an intermediate choice between performance and edition capabilities.
  • Always creates end points for arcs, ellipses, lines, mlines, leaders and not standard polylines and splines. Use this option only when edition capabilities are required.
