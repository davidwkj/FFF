About Units

The unit for the import data can be defined in several way.

The unit option is found in the DXF 2D tab of the Preferences.

With the default option Automatic, the unit of import is determined automatically (either millimeter or inch) for the best possible resulting drawing. However, in some cases the resulting drawing is not satisfactory and requires that you select another unit in the list, then restart the import.

If you have selected Scale Factor, enter the value of the scale factor between the imported file and what you want to get from the original drawing in the fields on the right.
