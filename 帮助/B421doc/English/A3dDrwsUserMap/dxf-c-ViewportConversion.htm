About Viewports Content Conversion

Viewports content can be imported in several different ways.

Options for viewports content conversion are found in the DXF 2D tab of the Preferences.

Convert viewports content as can be set to the following values:

  • Geometry: The viewport is imported in a view. This is the default value.
  • Details: The viewport is imported as a single detail (component instance).

    This option is particularly useful when the inserting DXF data in 2D Layout for 3D Design, to use them as a background for designing in a 3D context: DXF data can be quickly moved and positioned as wanted. If required, the single details can be exploded for further snapping with 3D design.

Note: Map DXF Layers with 2D Layout for 3D Design Sheets is not compatible with Convert viewports content as Details.
