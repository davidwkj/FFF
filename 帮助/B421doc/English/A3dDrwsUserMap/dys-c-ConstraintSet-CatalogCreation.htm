Creating a Catalog

Catalog Structure

By default, the created catalog is named according to the representation reference name, while the created chapters are named according to the .

Created items are named according to the group names.

Catalog structure and names of the created catalogs, chapters, or items are not associative after creation.

Catalog Keyword

No keyword is defined while creating a catalog from a representation.

Items Representation in a Catalog Browser

The items can be identified in the Catalog Browser (thumbnail view) by their 2D representation, which is computed from the 2D representation of the linked group.

This 2D representation is associative after creation.

The 2D representation is optimized so the clip symbol representing the group is not visualized.

Note: Though the catalog is created considering the internal or external constraints of the group, they are not visualized in the thumbnail.
