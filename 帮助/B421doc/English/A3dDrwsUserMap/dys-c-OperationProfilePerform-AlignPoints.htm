Aligning Points

You can align the points by selecting the points on or before executing the Align Points command.

  • Before executing Align Points: You can select all the points using trap selection and then click Align Points. Or you can use the context menu commands Horizontal or Vertical to align the points instantly.
  • After executing Align Points: You can click Align Points and then select the points using trap selection.

Notes:

  • If you select the points using trap selection, the first created point is considered as the origin point. Whereas, if you select the points using Ctrl, the first selected point is considered as the origin point.
  • When selecting points, only free points are considered for alignment and origin selection.
  • If the selected set of points contains a mix of free points and points belonging to other geometry, points with constraints, or background points, then only free points are considered and aligned.
