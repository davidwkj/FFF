About Concatenating Curves

You can concatenate a set of existing connected curves. This capability lets you constrain the set of concatenated curves.

Related Topics
Grouping Sketch Elements Together

To concatenate the connected curves of a sketch, right-click the selected curves and choose Selected Objects > Concatenate Curve. The concatenate curve (ConcatenatedCurve.1) is created in the construction mode with limiting points (Point.5 and Point.6).



If is selected, the Group constraint is created to attach the concatenated curve and connected curves together.

There is no associativity between the concatenated curve and the connected curves used to create it. The concatenated curve supports regular sketched curves as well as use-edge curves (generated from the projection or intersection of external geometry).

A warning message is displayed if the concatenation of the curves fails.

Tip: You can hide the original curves with its limiting points to directly work with the concatenated curve.
