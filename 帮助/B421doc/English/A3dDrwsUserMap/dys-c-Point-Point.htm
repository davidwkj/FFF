Creating a Point

About the Point Command

You can create point manually. For this, move the pointer to activate SmartPick and click as you get the required option.

Isobarycenters

To create an isobarycenter, multi-select the points before clicking Point .

Note: An isobarycenter can only be created between points. In other words, if you multi-select a rectangle, only the four vertices are used for defining the isobarycenter. Associativity is no more valid.

Symbols Representing Points

The points are represented either by crosses or by points, depending on the chosen creation mode.

  • In standard mode, which is the default mode, points created on a line, for example, are represented by crosses. The points and the line are visible outside the Sketcher app. For more information, see Sketcher User's Guide: Getting Started: Basic Concepts: About Tools for Sketching.
  • The points generated by Break operations are created in construction mode, even if Standard/Construction is set to Standard. For more information, see Sketcher User's Guide: Getting Started: Basic Concepts: About Tools for Sketching.
  • It is possible to configure the point representation using the Properties (Graphic tab) command.
