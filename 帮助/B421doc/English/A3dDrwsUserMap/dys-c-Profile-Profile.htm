Creating Profiles

About the Profile Command

After clicking Profile , Tools Palette displays value boxes for defining the different point coordinates and three profile options:

  • Line: (default mode)
  • Tangent Arc:
  • Three Point Arc: the second point you click indicates a point which the profile goes through (arc second point).

About Tangent Arcs

When you sketch a profile using the pointer (that is, without using Tools Palette and typically the Tangent Arc ) you can switch from the Line mode to the Tangent Arc mode by dragging and dropping the pointer to position the tangent arc end point.

Tangent arcs are always positioned in the direction of the element previously created.
