Closing Splines Using the Close Spline Option

When creating a spline, you can close it by right-clicking the last point and selecting Close spline.

The spline is closed in such a way that it is continuous in curvature at the closure point.

It is also possible to close a spline while editing it. For more information, see Editing a Spline.
