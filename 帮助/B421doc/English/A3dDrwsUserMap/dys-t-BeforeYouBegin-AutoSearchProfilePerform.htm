Performing Auto-Search on Profiles

You can auto-search for the different elements of a profile. The unambiguous part of the profile is selected when you use Edit > Auto Search command.

Before you begin: Create an elongated hole.

  2. Click Auto Search on the context toolbar.

    This auto search uses point propagation. So, the search result consists of parts of the profile which are in contact with each other without any ambiguity.

    Note: The select is not propagated on construction geometries.

  3. Click Auto Search with Tangent Propagation on the context toolbar.

    This auto search uses tangent propagation. So, the search result consists of parts of the profile which are tangent to each other. The search is not propagated beyond points of ambiguity.

    Element selected
    Resulting auto-searched profile




    Note: The Auto Search with Tangent Propagation command is available if at least one tangent element exists with the selected element.
