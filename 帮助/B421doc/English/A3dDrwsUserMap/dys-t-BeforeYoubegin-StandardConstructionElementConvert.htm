Converting Standard into Construction Elements

You can convert standard elements into construction elements and vice versa.

Construction elements are internal to, and only visualized by the sketch. They are used as positioning references and are displayed in gray.

Before you begin:
Related Topics
Modifying Element Coordinates

  1. Select one of the standard lines you have sketched so as to convert it into a construction line.



  2. The selected line now appears as a gray dashed line, indicating that it is a new type of line. The line is converted into a construction element.

    Note: Construction elements are internal to, and only visualized by the sketch. They are used as positioning references and cannot be visualized in the work area.

  3. The construction line is converted into a standard line again.



    Tip: To convert any geometrical element into a construction element use either of the following ways:
    • Select the geometrical element to display the context toolbar and click Construction Element.
    • Double-click the geometrical element to display its edition dialog box and select the Construction element check box.
    • Right-click the geometrical element and select the Construction Element option.

    Conversely, to convert a construction element into a geometrical element, use either of the following ways:

    • Select the construction element to display the context toolbar and click Geometrical Element.
    • Double-click the construction element to display its edition dialog box and clear the Construction element check box.
    • Right-click the construction element and select the Geometrical Element option.

    These options are available in all the profile commands context menu and edition dialog boxes.
