About the Catalog Structure

Create a Catalog

  1. In the top bar, click Add > Generate Catalog.

  2. In the Catalog dialog box, enter a name for your catalog.

    Important: The name you enter for the new catalog must not already be in use for another catalog in the database.

  3. Click OK.

    The catalog is created and saved, but it is not in the edition mode.

Insert an item in a Catalog

You can insert an item in a catalog.

  1. From the Tools section of the action bar, click the Catalog Browser.

  2. Open the catalog.

  3. Right-click the required chapter node and then select Insert > Item: Feature.

  4. Back to the representation containing the item reference, select the item component from the .

    The VPM catalog window is displayed automatically and the item is inserted under the chapter you selected.
