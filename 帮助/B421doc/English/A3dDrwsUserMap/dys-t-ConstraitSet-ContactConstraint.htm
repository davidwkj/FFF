Creating Contact Constraints

You can apply a constraint with a relative positioning that can be compared to contact.

This task shows you how to:

  • Create a Contact Constraint
  • Modify or Delete Contact Constraints

Before you begin: Create two elements, for example two circles.
Related Topics
About Contact Constraints

Create a Contact Constraint

You can create specific geometrical constraints between two elements using the Contact Constraint command.

  1. From the Sketch section of the action bar, click Contact Constraint.

  2. You can either select the geometry or the command first. Click a first element. For example, click a circle.



  3. Click a second element. For example, click another circle.

    The concentricity constraint symbol appears and the constraint is created.

Modify or Delete Contact Constraints

You can modify or even delete the contact constraint using the context menu command.

  1. Make sure either Constraint or Contact Constraint is active.

  2. Right-click the constraint you want to modify or delete.

  3. Select the option corresponding to the desired operation, from the context menu.
