Creating Chamfers

You can create a chamfer between two lines trimming either all, the first or none of the elements.

This task shows you how to:

  • Trim Both Lines
  • Trim the First Line
  • No Trimming
  • Trim Both Lines Until their Intersection
  • Trim Both Lines and Create Construction Lines Until their Intersection
  • Trim Both Lines and Create Construction Lines

Before you begin:

To perform this task, create two intersecting lines.

About Chamfers

Trim Both Lines

Using the Trim All Elements in Tools Palette you can trim both lines of the chamfer.

  1. From the Sketch section of the action bar, click Chamfer .

    The possible chamfer options are now displayed in Tools Palette.

  2. Click Trim All Elements in Tools Palette.

  3. Select the two lines. Once selected, both lines are highlighted.



    The two elements are connected by a line representing the chamfer which moves as you move the pointer. This lets you vary the dimensions of the chamfer whose values appear in Tools Palette.

    Tip: You can also:
    • Select the lines first and then the command.
    • Select one or more points. Ensure that each point is formed by the intersection of only two lines. That is, each point is the end point of only two lines.

  4. Click to select where to create the chamfer.

    The chamfer with both elements trimmed is created.

    Important: If the is selected, the constraints will be created between what we call in the scenarios below the old intersection point and new end points of the lines.

    Note: When you create chamfers on multiple vertices:

    • You can see the preview of the chamfers at these vertices.
    • All the chamfers have the same dimensions. To achieve different dimensions, select each vertex separately or edit the constraints.
    • If any of the vertices is invalid (intersection of more than two lines) then chamfer is created on the valid vertices only.


Trim the First Line

Using the Trim The First Element option in Tools Palette you can trim first selected line of the chamfer.

  1. Click Chamfer .

  2. Click Trim The First Element .

  3. Select the first line and the second line.

    The chamfer with one element trimmed is created.

No Trimming

Using the No Trim option in Tools Palette you can created chamfer without trimming lines.

  1. Click Chamfer .

  2. Click No Trim .

  3. Select the first line and the second line.

    The chamfer with no element trimmed is created and the original lines are kept.

Trim Both Lines Until their Intersection

Using the Standard Lines Trim in Tools Palette you can created chamfer and the two lines are trimmed up to their intersection.

  1. Click Chamfer .

  2. Click Standard Lines Trim .

  3. Select the first line and the second line.

    The chamfer is created and the two lines are trimmed up to the two lines intersection.

Trim Both Lines and Create Construction Lines Until their Intersection

Using the Construction Lines Trim in Tools Palette you can create a chamfer and the two lines are trimmed. Two new lines are created between the intersection and the trimmed extremity of the lines, and set as construction lines.

  1. Click Construction Lines Trim .

  2. Select the first line and the second line.

    The chamfer is created and the two lines are trimmed. Two new lines are created between the intersection and the trimmed extremity of the lines, and set as construction lines.

Trim Both Lines and Create Construction Lines

Using the Construction Lines No Trim in Tools Palette you can created chamfer and the two lines are trimmed. Two new lines are created between the previous extremities and the trimmed extremity of the lines, and set as construction lines.

  1. Click Chamfer .

  2. Click Construction Lines No Trim .

  3. Select the first line and the second line.

    The chamfer is created and the two lines are trimmed. Two new lines are created between the previous extremities and the trimmed extremity of the lines, and set as construction lines.
