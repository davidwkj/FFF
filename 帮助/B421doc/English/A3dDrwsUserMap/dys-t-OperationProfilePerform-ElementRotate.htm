Rotating Elements

You can rotate elements using the Rotation command. You can duplicate element or move them using this functionality.

In this scenario, the geometry is moved keeping internal and external constraints.

Rotating elements also means re-computing distance values into angle values, if needed. However, note that, only non-fixed elements are updated.

Before you begin:
  • Create the required 2D elements and internal and external constraints on them.
  • Make sure that the Open dialog boxes with legacy appearance check box available at Me > Preferences > All Preferenes > 3D Modelling > 3D Modeling Core > Sketcher > Dialogs appearance, is selected.
Related Topics
Using SmartPick

  1. From the Sketch section of the action bar, click Rotate .

  2. In the Rotation Definition dialog box, In the Instances box, select the number of instances to be created.

  3. Keep Duplicate mode check box selected.

    The option is selected by default, which means that the 2D elements you select are copied.

    Note: Apart from the Duplicate mode check box, the Keep internal constraintsKeep external constraints check boxes are cleared, by default.

    However, after you execute this command, your preferences are stored and saved for the next usage.

  4. Select Keep internal constraints and Keep external constraintsto preserve the internal and external constraints applied to the selected elements in the rotation.

    If you clear Keep external constraints check box, any external constraint existing between the selected elements and external elements will be disregarded in the rotation.

  5. Select the geometry to be rotated.



  6. Select or click the center point for rotation.

    You can also enter coordinates in the boxes in the Tools Palette, or enter a value for the rotation angle in the Rotation Definition dialog box.



  7. Select or click a point to define the reference line that is used for computing the angle.

  8. Move the pointer to get the required rotation angle.
