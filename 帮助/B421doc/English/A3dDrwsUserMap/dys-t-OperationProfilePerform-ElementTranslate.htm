Translating Elements

You may either perform a simple translation (by moving elements) or create several copies of 2D elements.

The app provides a powerful command for translating elements. Translating elements also means re-computing distance, angle and/or length constraint values, if needed. However, note that, only non-fixed elements are updated.

This task shows you how to perform a translation on 2D elements by defining the duplicate mode and then selecting the element to be duplicated.

  • Translate Elements Keeping Internal or External Constraints

Before you begin:

Create the required 2D elements and internal and external constraints on them.

Translate Elements Keeping Internal or External Constraints

You can translate elements keeping internal or external constraints.

Before you begin: Make sure that the Open dialog boxes with legacy appearance check box available at Me > Preferences > All Preferenes > 3D Modelling > 3D Modeling Core > Sketcher > Dialogs appearance, is selected.

  1. From the Sketch section of the action bar, click Translate .

  2. In the Translation Definition dialog box, in the Instances box, keep 1 as the number of instances to be created.

  3. Keep Duplicate mode check box selected.

    By default, the Duplicate mode check box is selected, which means that the 2D elements you select are copied. If you clear Duplicate mode check box, the element is moved.

  4. Select Keep internal constraints check box.

    This option specifies that you want to preserve the internal constraints applied to the selected elements in the translation.

  5. Clear Keep external constraints check box.

    Any external constraint existing between the selected elements and external elements is disregarded in the translation.

  6. Select the elements to be translated using the trap selection.

    You can either select one 2D element or multi-select the entire 2D geometry by trapping it using the pointer as shown below.



  7. Click to indicate the translation vector starting point.

    You can define the translation length in the work area, using the pointer. For more precise results, enter a specific value for the translation length in the Translation Definition dialog box.



  8. Enter the required value in the Length box and press Enter.

    Tip: You can use SmartPick to keep lines horizontal.

  9. Click to indicate the translation vector ending point.



    A context toolbar appears showing available constraints.

    The translation is performed. You can notice that the internal constraints are preserved in the translated element (four tangency constraints), whereas the external constraint (an offset constraint) is not.

  10. Optional: Click the required constraint in the context toolbar.

    The translated geometry is constrained accordingly.

  11. Click anywhere in the work area.



    The selected geometry is translated with internal constraints.
