Trimming Elements

You can trim the unnecessary part of the element using Trim command.

This task shows you how to:

  • Trim Two Elements
  • Trim One Element

Before you begin:

Create two intersecting lines.

Related Topics
Breaking and Trimming Elements

Trim Two Elements

You can trim both the elements using the Trim All Elements in Tools Palette.

  1. From the Sketch section of the action bar, click Trim .

    The Trim options are now displayed in Tools Palette.

    By default, the Trim All Elements is selected.

  2. Select the first line.



  3. Position the pointer on the element to be trimmed.

    The second element is highlighted too, and both previewed lines are trimmed.

  4. Position the pointer on the same first element.

    The first element will be trimmed at the location where you click for the second time. The location of the relimitation depends on the location of the cursor.

  5. Click when you are satisfied with the relimitation of the two lines.



Trim One Element

You can trim first of the two selected elements using the Trim First Element in Tools Palette.

  1. Click Trim .

  2. Click Trim First Element in Tools Palette.

  3. Select the first line.



  4. Position the pointer to the second line.

    The first line selected is trimmed to the second line.

  5. Click when you are satisfied with the relimitation of the first line.



    Important:
    • In multi-selection mode, no extrapolation is done using Trim .
    • If you trim an element created from a projection or an intersection, then this element extremities are not constrained anymore to follow the extremities from the element they are issued from.
    • If the extremity point of the trimmed line is constrained (with the exception of Fix, Coincidence, Symmetry or Group constraint), or if the extremity point of the trimmed line is a geometrical element (not a construction element), then a coincidence constraint is created between this point and the trimmed line.
