Creating Mirrored Elements

You can mirror existing Sketcher elements about a line or an axis.

Before you begin: Create a Sketcher element and an axis.
Related Topics
Moving Elements by Symmetry

  1. Select the element to be mirrored.



  2. From the Sketch section of the action bar, click Mirror .

  3. Select the previously created line or axis.

    The selected element is mirrored. A symmetry constraint is created provided that you have previously selected the in the Sketch section of the action bar.



    Tip: To mirror several elements, use the multi-selection:
    • Drag the pointer and create a trap.
    • Select the symmetry axis.
