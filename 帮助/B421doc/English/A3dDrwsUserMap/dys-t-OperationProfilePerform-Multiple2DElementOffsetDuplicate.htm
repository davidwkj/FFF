Duplicating Multiple 2D Elements Using Offset

You can create one or more offset instances to profiles made of several elements.

This task shows you how to:

  • Offset an Element and Elements Tangent to It
  • Offset an Element using Point Propagation

Before you begin:

Create a rectangle, then apply the Corner command to one of its corners.

Offset an Element and Elements Tangent to It

The Tangent Propagation in Tools Palette offsets the element you select as well as the elements tangent to it.

  1. From the Sketch section of the action bar, click Offset .

    Tools Palette now displays four offset options and four different fields for duplicating elements.

  2. Click Tangent Propagation in Tools Palette.

    The Tangent Propagation offsets the element you select as well as the elements tangent to it.

  3. Select the element you want to offset. All the elements to be created appear as you drag the cursor.

    Tip: In the Tools Palette, select Both side offset to create offset on both sides.



  4. In the Instances box of Tools Palette, keep 1 as the number of offset instances you want to create.

  5. Select a point or click where you want the new elements to be located, or if you know where these elements must be precisely located, use one or more Tools Palettevalue fields for defining the location.

  6. Press Enter when done.



Offset an Element using Point Propagation

The Point Propagation in Tools Palette duplicates the element you select as well as conex elements.

  1. Click Offset .

    Tools Palette now displays four offset options for duplicating elements.

  2. Select Point Propagation .

    The Point Propagation duplicates the element you select as well as conex elements.

  3. Select the element you want to offset. All the elements to be created appear as you move the pointer.



  4. In the Instances box of Tools Palette, keep 1 as the number of offset instances to be created.

  5. Select a point or click where you want the new element to be located, or if you know where these elements must be precisely located, use one or more Tools Palette value fields for defining the location.

    Tip: In the Tools Palette, select Both side offset to create offset on both sides.



    Offset helps you to create one or more offset instances to profiles made of several elements:

    Important:
    • This is not true for generated elements ( Drafting app).
    • If the multi-selected elements do not make up a closed profile, the offset will be applied to the selected elements only. As a result, you will have as many offset elements as the first multi-selected elements.
    • Previews are not available when creating several offset instances (i.e. when the value in the Instances box of Tools Palette is higher than one).
