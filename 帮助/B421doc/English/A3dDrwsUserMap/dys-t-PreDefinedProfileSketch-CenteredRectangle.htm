Creating Centered Rectangles

You can create the rectangle, if you know its center.

This task shows you how to:

  • Create a Centered Rectangle
  • Create a Constrained Centered Rectangle

Creating Rectangles

Create a Centered Rectangle

You can create a centered rectangle.

  1. From the Sketch section of the action bar, click Centered Rectangle .

  2. Click a point in the work area or select an existing one.

  3. Move the pointer to create the centered rectangle.



Create a Constrained Centered Rectangle

You can create a constrained centered rectangle by activating the constraints options.

    Equidistant constraints are applied automatically on the opposed lines accordingly to the center point.
