Creating Polygons

You can create a polygon. This polygon can have three to 24 sides.

By default, the polygon is created with six sides and an incircle.

This is useful to create frequently used conceptual designs like nuts or bolt heads.

Before you begin: Select the from the View section of the action bar.

  1. From the Sketch section of the action bar, click Polygon .

  2. In the Tools Palette, enter the coordinates for the centre point of the polygon:

    1. In the H box, enter the horizontal coordinate.
    2. In the V box, enter the vertical coordinate.

    The center of the polygon is fixed.

    Tip: You can also select the center point by:
    • Clicking on the required coordinate in the work area
    • Selecting an already existing point

  3. To create a hexagon, keep the Number of Sides locked at the default value of six and proceed to Step 5.

  4. To create a polygon, with number of sides more or less than six, clear the selection of As default number of sides .

  5. Enter the coordinates for the Point On Circle:

    • Incircle : This is the default selection. The point lies on mid-points of one of the edges of the polygon.
    • Circumcircle : The point lies on one of the vertices of the polygon.
    1. In the H box, enter the horizontal coordinate.
    2. In the V box, enter the vertical coordinate.
    3. Optional: In the Radius box, enter the radius of the circumcircle or the in circle.
    4. Optional: In the Angle box, enter the angle at which the point lies with respect to positive horizontal axis.

      Defining the H and V values fixes the Radius and Angle values and vice versa. That is, for a given values of H and V there are unique values of Radius and Angle and vice versa.

      Tip: You can also select the point by:
      • Clicking on the required coordinate in the work area
      • Selecting an already existing point

    Point on Circle: Incircle with six sides (default selection)
    Point on Circle: Circumcircle with six sides

  6. In the Number of Sides box, define the number of sides of the polygon and lock this value by selecting As default number of sides .

    You can define a minimum of three and a maximum of 24 sides.

    Tip: You can also move the pointer to change the number of sides interactively. Move the pointer clockwise to increase the number of sides and counterclockwise to decrease.

    The number of sides you lock is saved for the session. If you leave the number of sides unlocked, the next time you create a polygon, the number of sides is reset to the default, six.

    The polygon is created.

    Important: After the creation of polygon:
    • The incircle and the circumcircle are created as construction elements. They have a concentricity constraint between them.
    • The vertices have coincidence constraints with the circumcircle. The sides have tangential constraint with incircle.
