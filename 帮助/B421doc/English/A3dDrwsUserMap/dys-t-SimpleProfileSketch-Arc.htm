Creating Arcs

You can create an arc.

Before you begin:

About Arcs

  1. From the Sketch section of the action bar, click Arc .

  2. Locate the arc's center.

    You can:

    • Click the required location in the work area.
    • Use Tools Palette to define the point coordinates in the following boxes:

      • H: horizontal
      • V: vertical
      • R: radius (for the first two points)
      • A: angle (for the first two points)
      • S: angular sector

  3. In the H and V boxes of the Tools Palette, define the coordinates of the circle's center.

  4. Repeat the operation to define the arc's start point.

    The arc's center and start point are created. The arc now appears as you move the pointer.

  5. Repeat the operation to define the arc end point or click when you get the location you want.

    The arc is created.
