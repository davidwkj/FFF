Creating Bi-Tangent Lines

You can create a bi-tangent line.

Before you begin: Create two circular elements, for example two circles.

  1. From the Sketch section of the action bar, click Bi-Tangent Line .

    Important: Ensure that your sketch is consistent before creating bi-tangent lines (you can use Sketch Analysis for this). Otherwise, the system will not compute the tangencies properly.

  2. Select the first element.

  3. Select another element.

    The bi-tangent line appears between both selected elements. Tangents are created as close as possible to where you clicked on the circle.

    Tip: The corresponding constraints are created provided you have selected in the Sketch section of action bar.

Note: You can select a point type element. The tangent line goes through this point and a coincidence constraint is created at the point.
