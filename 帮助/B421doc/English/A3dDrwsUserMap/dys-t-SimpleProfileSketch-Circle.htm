Creating Circles

You can create a circle. Once you have created one circle, you can create any other and in the meantime use the radius parameter from the circle first created.

By default, circle centers appear on the sketch. If you do not need them, you can specify this.

This task shows you how to:

  • Create a Circle
  • Create a Circle Reusing a Circle Radius Value

Before you begin:
  • Make sure that the Open dialog boxes with legacy appearance check box available at Me > Preferences > All Preferenes > 3D Modelling > 3D Modeling Core > Sketcher > Dialogs appearance, is selected.

Create a Circle

You can create a circle.

  1. From the Sketch section of the action bar, click Circle .

  2. Locate the circle's center.

    You can:

    • Click anywhere in the work area.
    • If you know where this point must be located, use Tools Palette which now displays following value boxes:

      • H: horizontal

      • V: vertical

      • R: radius

  3. In the H and V boxes in the Tools Palette, enter the coordinates for the circle's center.

  4. In the R box in the Tools Palette, enter the radius value.

    The circle is created.

    Tip: The constraints are assigned to this circle on the condition that you entered coordinate values in the boxes (you did not create the circle by clicking).

Create a Circle Reusing a Circle Radius Value

You can create a circle using the radius of the previously created circle.

  1. Click Circle .

  2. Right-click the first circle and select Parameter > Copy Radius.

  3. Click to indicate the second circle location or use Tools Palette to specify the circle center.



    The new circle is automatically created with the radius of the circle first created but not positioned.
