Creating Circles Using Coordinates

You can create a circle at a distance from a reference point.

  1. From the Sketch section of the action bar, click Circle Using Coordinates .

    In the Circle Definition dialog box, the default point coordinates that appear are the origin axis coordinates. The default circle radius is 10mm.

    Before clicking Circle Using Coordinates , if you select an existing point, this point is used as a reference point and the coordinates of the center point are set from this point.

  2. In the Cartesian tab of the Circle Definition dialog box, enter the coordinates of the circle's center.

    Note: You can also enter polar coordinates in the Polar tab, instead of Cartesian coordinates.

  3. In the Radius box, enter the radius value.

    The circle is created.

    Note: By default, circle centers appear on the sketch. If you create circles by clicking and you do not want to see the centers, you can specify this in the Preferences dialog box. For this, clear the Create circle and ellipse centers check box under Me > Preferences > All Preferences > 3D Modeling > 3D Modeling Core > Sketcher > Geometry.
