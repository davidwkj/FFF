Creating Conic Curves

You can create conic curves using arcs of parabolas, hyperbolas, or ellipses.

This task shows you how to:

  • Use Two Points and Start and End Tangent
  • Use Two Points and Tangent Intersection Point
  • Use Two Points with the Nearest End Point Mode
  • Use Four Points with a Tangency at Passing Point
  • Use Five Points

About Creating Conic Curves

Use Two Points and Start and End Tangent

You can create a conic curve using two points and start and end tangent.

  1. From the Sketch section of the action bar, click Conic .

  2. Select Two Points in Tools Palette.

  3. Click to indicate the Start Point of the curve: H=20mm, V=20mm.

  4. Type in Tools Palette for the Start Tangent point: H=30mm, V=50mm and press Enter.

    The start point and tangent has been created.

  5. Click to indicate the End Point of the curve: H=90mm, V=10mm.

  6. Type in Tools Palette for the End Tangent point: Angle=120deg, and H=70mm, and press Enter.

    Note: With a parameter :

    • If parameter = 0.5, the resulting curve is a parabola.
    • If 0 < parameter < 0.5, the resulting curve is an arc of ellipse.
    • If 0.5 < parameter < 1, the resulting curve is a hyperbola.

  7. Type in Tools Palette for the parameter: Parameter=0.3 and press Enter.

    The conic is created.



Note: You can also specify the passing point to create a conic. For example: H=50mm, V=40mm.

Use Two Points and Tangent Intersection Point

You can create a conic using two points and tangent intersection point.

  1. Click Conic .

  2. Select Two Points .

  3. Select Tangent Intersection Point .

  4. Click to indicate the Start Point of the curve: H=20mm, V=20mm.

  5. Click to indicate the End Point of the curve: H=90mm, V=10mm.

  6. Click to indicate the Tangent Intersection Point: H=60mm, V=60mm.



  7. Type in Tools Palette for the parameter: Parameter=0.3 and press Enter.

    The conic is created.

Note: You can create a conic by specifying the passing point. For example: H=50mm, V=40mm.

Use Two Points with the Nearest End Point Mode

You can create a conic using two points with the nearest end point mode.

  1. Create two lines.

    1. Create one line from H=10mm, V=0mm to H=40mm, V=60mm.
    2. Create another line from H=90mm, V=0mm to H=90mm, V=40mm.



  2. Click Conic .

  3. Select Two Points .

  4. Select Nearest End Point .

  5. Click to indicate the Start Point of the curve: H=20mm, V=20mm.

  6. Click to indicate the End Point of the curve: H=90mm, V=10mm.

  7. Click to indicate the Passing Point: H=60mm, V=40mm.

    The conic is created. The tangents at the start and end points have been defined by the lines. The start and end points taken into account are the nearest extremities of the lines during the selection.

    Note: When you redo the previous steps deactivating the Nearest End Point mode:

    • The tangents at the start and end points have been defined by the lines.

    • The start and end points taken into account are the selected points on the lines.



Use Four Points with a Tangency at Passing Point

You can create a conic using four points with a tangency at passing point.

  1. Click Conic .

  2. Select Four Points .

  3. Click to indicate the Start Point of the curve: H=20mm, V=20mm.

  4. Type in Tools Palette for the Start Tangent point: H=30mm, V=50mm and press Enter.

    The start point and tangent has been created.

  5. Click to indicate the End Point: H=90mm, V=10mm.

  6. Select Start and End Tangent .

  7. Click to indicate the First Point: H=60mm, V=50mm.

  8. Type in Tools Palette for the First Tangent point: H=90mm, V=50mm and press Enter.

  9. Click to indicate the Second Point: H=100mm, V=40mm.

The conic is created. The defined tangent at the start point has been released and the construction line representing the tangent has been removed.

Use Five Points

You can create a conic using five points.

  1. Click Conic .

  2. Select Five Points .

  3. Click to indicate the Start Point: H=20mm, V=20mm.

  4. Click to indicate the End Point: H=90mm, V=10mm.

  5. Click to indicate the First Point: H=30mm, V=50mm.

  6. Click to indicate the Second Point: H=60mm, V=50mm.

  7. Click to indicate the Third Point: H=90mm, V=30mm.

    The conic is created.
