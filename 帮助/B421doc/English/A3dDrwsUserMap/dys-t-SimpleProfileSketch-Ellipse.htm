Creating Ellipses

You can create an ellipse made from two infinite axes using the Ellipse command.

Before you begin:
  • Make sure that the Open dialog boxes with legacy appearance check box available at Me > Preferences > All Preferenes > 3D Modelling > 3D Modeling Core > Sketcher > Dialogs appearance, is selected.

  1. From the Sketch section of the action bar, click Ellipse .

  2. Locate the first point of the profile.

    • Click in the work area.
    • Use Tools Palette which now displays following value boxes:
      • Center: H: horizontal
      • V: vertical
      • Major Radius
      • Minor Radius
      • A: angle

  3. Enter the coordinates for the following points:

    1. Center



      Note: You can also click to create a first point that corresponds to the ellipse center.

    2. Major Semi-Axis Endpoint



      You just created a point on the ellipse. This point allows defining the major semi-axis.

    3. Minor Semi-Axis Endpoint

      You just created a point which allows defining both minor semi-axes.

      Tip: The constraints are assigned to the ellipse on the condition that you entered coordinate values in the boxes (you did not create the profile by clicking).
