Creating Lines

You can create a line. Once you have created one line, you can create any other and use the length from the line first created or set this first line as an angle reference.

This task shows you how to:

  • Create a Line
  • Define Line Length
  • Define Line Angle

Create a Line

You can create line.

  1. From the Sketch section of the action bar, click Line .

    Tools Palette now displays values for defining the line.

  2. In Tools Palette, type the start point coordinates: H=18mm, V=18mm and press Enter.

  3. In Tools Palette, type the end point coordinates: L=30mm, A=45deg and press Enter.

    The line is created.

    The constraints are similarly assigned to this line if you have previously selected the command from the action bar.



Define Line Length

You can create line by defining its length.

  1. Click Line .

  2. Right-click the first line and select Parameters > Copy Length.

  3. Click to indicate the start point location.

  4. Click to indicate the end point location.

A new line is created with a length of 30mm.

Define Line Angle

You can create line by defining its angle.

  1. Click Line .

  2. Right-click the first line and select Parameter > Set As Angle Reference.

    A red arrow symbolizing the reference orientation for the angle is displayed on the first line.

  3. In Tools Palette, type the angle: A=75deg and press Enter.

  4. Click to indicate the start point location.

  5. Click to indicate the end point location.

    A new line is created with an angle of 75deg in relation to the first line reference orientation.

    Note: In this task, we used Tools Palette, but you can create this line manually. To do this, move the pointer to activate SmartPick and click as soon as you get the required option.

    Important: While assigning graphical attributes to a line (for example, make it thick and red), when you change this red thick line into a construction line (from the context menu: Object.Line > Definition... or the Construction line option in the Line Definition dialog box), the line becomes a dotted gray line. Even if you then decide to make it a standard line back again (clearing Construction line), the red and thickness attributes is not assigned to the line. The line is assigned its default attributes (black).
