Creating Points Using Coordinates

You can create a point by indicating coordinates. You can use an existing point as reference for creating another point.

About Creating Equidistant Points

  1. From the Sketch section of the action bar, click Point by Clicking.

  2. Click to indicate the end point: H=20mm, V=20mm.

    The reference point is created.

  3. Click Point by Using Coordinates .

    The Point Definition dialog box appears.

  4. Select the previously created point.

  5. Select the Polar tab.

  6. In the Radius box, enter 30mm.

  7. In the Angle box, enter 30deg.

    The point is created with a 30mm radius and 30deg angle relatively to the reference point. A construction line represents the angle direction.
