Creating Points Using Intersection

You can create a point at the intersection of two intersecting elements.

Before you begin: Create a line, then a line and a spline intersecting it.

About Creating Equidistant Points

  1. Multi-select the elements to be used for intersecting.



  2. From the Sketch section of the action bar, click Intersection Point .

  3. Select an intersecting element.

    The intersecting points are created.



    Tip: The constraints appear on the condition that is active in the Sketch section of the action bar.

    Note: The Intersection Point command takes unbounded geometrical elements into account to compute intersection points
