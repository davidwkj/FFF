Creating Rectangles

You can create a rectangle.

Before you begin:
Related Topics
Creating Oriented Rectangles

  1. From the Sketch section of the action bar, click Rectangle .

  2. Locate the first point for the rectangle.

    You can:

    • Click the point's location in the work area.
    • If you want to know where this point must be located, use Tools Palette, which now displays value boxes for defining the point coordinates. These value boxes are:

      • H: horizontal

      • V: vertical

  3. In Tools Palette, enter the coordinates of one of the vertices of the rectangle.



    Tools Palette now displays two additional value boxes which are: Width and Height.

  4. Position the diagonally opposite vertex from the first point by entering the values for Width, Height.

    The rectangle is created.

    Note: In this task, we used Tools Palette but you can also create a rectangle manually. To do this, move the pointer to activate SmartPick and click as soon as you get the required coordinates or length.
