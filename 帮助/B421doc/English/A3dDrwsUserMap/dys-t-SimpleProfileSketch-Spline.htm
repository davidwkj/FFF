About Splines

  1. Indicate the control points through which the spline should pass.

    • Click the point's location in the work area.
    • Use Tools Palette which now displays following value boxes:
      • H: horizontal
      • V: vertical

  2. In the Tools Palette enter the first control point coordinates.

    The first point is created.
