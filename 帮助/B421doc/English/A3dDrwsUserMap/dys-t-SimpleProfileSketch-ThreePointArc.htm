Creating Three Point Arcs

You can create an arc using three reference points to define the required size and radius.

Before you begin:
Related Topics

About Arcs

  1. From the Sketch section of the action bar, click Three Point Arc .

  2. Locate the first arc point.

    You can:

    • Click anywhere in the work area.
    • Use Tools Palette which now displays following value boxes:
      • H: horizontal
      • V: vertical
      • R: radius

  3. In the H and V boxes of the Tools Palette, define the coordinates of the start point coordinates.

  4. Repeat the operation to define the two other points:

    1. Second Point
    2. End Point
