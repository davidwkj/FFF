Modifying Element Coordinates

Modifying sketch coordinates affects the feature defined on this sketch i.e. the associativity remains valid.

This task shows you how to modify a line by defining new coordinates.

Before you begin:

Create a line.

  1. Double-click the element to edit. You can double-click either in the work area or in the tree. For this scenario, double-click a line. The Line Definition dialog box appears indicating the line end point coordinates. Or alternatively, right-click the element and select xxx object > Definition.

  2. Enter new coordinates to change the end points and/or the length and angle.

  3. Select the Construction element check box to change the line type.

  4. Click OK to confirm the changes.

    The geometry is modified accordingly.

    Important: Multi-selection is not allowed for editing Sketcher elements.

Note: Profiles are not considered as entities when it comes to editing them. To edit a profile, you need to edit the sub-elements composing it. Edit > Properties, or Properties let you access and edit sketcher geometry properties such as the feature name and graphic properties.
