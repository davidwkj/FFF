Editing a Spline

You can edit the spline by using various options available in the Spline Definition dialog box.

This task shows you how to:

  • Modify the Spline Control Points
  • Add a Point
  • Replace a Point
  • Close a Spline
  • Remove a Point
  • Modify a Spline

Modify the Spline Control Points

You can change the control points of a spline using a graphical handle.

  1. Double-click the control point you want to edit.

    The tangency and curvature radius handles appear in gray along with the Control Point Definition dialog box.



  2. Enter new coordinates in the H (horizontal) and V (vertical) boxes respectively.

  3. Select the Tangency check box to impose a tangency on this control point.

    The tangency handle is highlighted and the Reverse Tangent and the Curvature Radius becomes available in the dialog box.

    Note: The curvature radius computation is dependent on the tangency option.

  4. Rotate the tangency handle around the control point till you get the desired angle with respect to the H direction of the Sketcher axis system.

    The tangency angle value is updated in the dialog box and simultaneously the spline curve is modified.

  5. Optional: Click Reverse Tangent to reverse the tangent direction.

  6. Select the Curvature Radius check box.

    The curvature handle is highlighted using which you can change the radius of curvature.

  7. Drag the curvature handle along the direction perpendicular to the direction of the tangent till you get the required radius.

    The curvature radius value is updated in the dialog box and simultaneously the spline curve is modified.

    Note: Both the manipulations are independent of each other. If the radius curvature handle is under manipulation, the tangency value remains constant and vice versa for the tangency handle.

  8. Click OK.

    The point is moved and an arrow appears on this point to indicate the tangency.

    Tips:
    • You can modify the spline shape by selecting a point and then dragging it.
    • You can also modify the control point by right-clicking it and selecting CtrlPoint.X object > Edit Tangency-Curvature.

Add a Point

You can add a point to a spline. There are several possibilities, depending on whether you want to add an existing point, or create a point on the sketch while editing the spline.

  1. Double-click the spline.

    The Spline Definition dialog box appears.

  2. To add an existing point, follow the steps from 3 through 5.

    To create the point on the sketch while editing the spline, follow the steps from 6 through 8.

  3. In the dialog box, select the spline point after or before which you want to add a point. Select CtrlPoint.2 for example.

  4. Choose Add Point After or Add Point Before (depending on whether you want to add a point after or before the selected point). Select Add Point After for example.

Close a Spline

You can close a given spline.

  1. Double-click the spline.

  2. In the Spline Definition dialog box, select the Close Spline check box to close a spline.

    The spline is closed in such a way that it is continuous in curvature at the closure point.

    Important: To edit closed splines using a continuity in point at the closure point: select Close spline which makes such splines continuous in curvature at the closure point.

Remove a Point

You can remove the point on a spline.

  1. Select the point that you want to remove in the Spline Definition dialog box.

  2. Click Remove Point.

  3. Click OK.

Modify a Spline

You can modify the spline by modifying its control points.

  1. Double-click the spline.

    The tangency and curvature radius handles appear in gray at the first control point along with the Spline Definition dialog box.

  2. Select the point you want to modify in the Spline Definition dialog box.

  3. Select the Tangency check box.

    This activates the Curvature Radius check box and the Reverse Tangent. The tangency handles get activated and highlighted in blue.

  4. Follow the steps 4 through 8 as explained in Modify the Spline Control Points.

    The control point is modified and as a result the spline is modified accordingly.

    Note: The handles for corresponding control point are displayed when you move the pointer over the associated control point. When the edition of spline is started, the handle is displayed at the first control point, by default. The handles remain associated with the previous control point until you move the pointer over other control point belonging to the same spline in edition.
