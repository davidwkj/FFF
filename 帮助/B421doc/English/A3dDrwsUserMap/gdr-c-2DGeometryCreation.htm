About 2D Geometry Creation

The Drawings app enables you to create 2D geometry.

Before you begin creating 2D geometry, make sure you are familiar with the following concepts:

Note the following points:

  • Construction elements contained in 2D geometry are displayed only in the active view.
  • To ensure that 2D geometry is not altered once it is created, geometry edition is only allowed in the active view. Therefore, if you want to edit or move 2D geometry, you need to make the view active which contains the geometry. You can prevent 2D geometry from being inappropriately moved (and distorted) in the active view by clearing Allow direct manipulation check box from Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Geometry tab.
  • You can create as many 2D geometry elements of a given type as needed by double-clicking the appropriate icon (instead of single-clicking it).
  • You can create detected constraints automatically by clicking Create Detected Constraints in the Sketch section of the action bar.
  • You can view the created constraints by clicking Show Constraints in the View section of the action bar.
