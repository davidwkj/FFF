About Copy/Paste

This topic explains the concepts that you need to understand before you begin copying/pasting drawing elements.

The following topics are discussed:

  • Copy/Paste in a Drawing
  • Copy/Paste a View to a 3D Shape

Copying and Pasting Elements

Copy/Paste in a Drawing

This sub topic provides more information about copy and paste operations within a drawing.

Copy/Paste Geometrical Elements

  • Geometrical elements in a drawing cannot be pasted to a 3D shape or to a sketch. They can only be pasted within a drawing.
  • If you copy an element and then immediately delete it, you are not be able to paste it anymore.

Copy/Paste Views

  • When copying and pasting views, positional links between the views (that is links which exist between a parent view and its child view, for example) are not kept. The only way you can keep positional links between views is by copying and pasting the sheet.
  • Unlike usual views, detail views cannot be copied to a 3D shape.
  • Views containing 2D components can be copied to a 3D shape, but the resulting view does not contain these 2D components (which are specific to the Drafting context). If you want to copy the equivalent 2D component to the 3D shape, you need to explode it first.

Copy/Paste Texts

When copying and pasting a text, two cases are possible depending on whether you change the feature name of the text (by right-clicking the text and selecting Properties > Feature Properties):

  • If you change the feature name, and copy a text whose customized feature name is Custom Text, for example, then the feature name of the copy (or copies) remains Custom Text.
  • If you do not change the feature name, and copy a text whose feature name is Text.1, for example, then the feature name of the copy will be Text.2 (then Text.3, etc. if you make several copies).

Copy/Paste Other Types of Elements

  • If you copy and paste a view axis, infinite lines are displayed in your view. Those lines are designed to keep constraints on the axis that were created in the first view.
  • You cannot copy and paste not-up-to-date dimensions (displayed in fuchsia by default or according to the color defined for Dimensions for which last update failed in the Elements' Analysis dialog box available via Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Display section, under Elements Analysis , selecting Activate elements' analysis check box, and clicking Configure).
  • When pasting elements from one drawing to another which uses a different standard, wherein some elements (such as dimensions) are not supported by this standard, then, in this case, such non-supported elements are not created.

Copy/Paste a View to a 3D Shape

This section provides more information about copying and pasting a view to a 3D shape.

A view copied from a drawing and then pasted to a 3D shape becomes a sketch.

  • If a 3D plane is selected before pasting the view, the sketch follows the 3D plane.
  • If no 3D plane is selected before pasting the view, the sketch follows the plane of the copied view.
