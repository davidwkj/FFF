About Loading/Opening 3D Data

When opening a generative drawing without opening its associated 3D representation, the links between them are broken. You can, however, load/open 3D data for one or several views, sheets or for a whole drawing, with or without opening its associated 3D product.

The following topics are discussed:

  • When to Load/Open 3D Data
  • Miscellaneous Remarks

Loading/Opening 3D Data

When to Load/Open 3D Data

You must load 3D data and restore broken links under the following circumstances:

  • Before performing any modification that requires 3D data, for example, before creating an associative dimension.
  • Before updating the drawing, as the update command takes into account only the loaded 3D data.

At any time during your session, you can also select Open 3D data to open the associated 3D representation.

Two commands are available:

  • Load 3D data: Loads 3D data without opening the associated 3D object representation.
  • Open 3D data: Loads 3D data and opens the associated 3D representation.

Miscellaneous Remarks

You need to consider the following remarks that apply to loading/opening 3D data.

  • From Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > View tab, under View generation, the 3D extracted representations option must not be defined As Session. If the As Session mode is selected, the Load 3D data command takes into account only the 3D representations already open in session.
  • The view must not be locked.
  • The Update command is not active before launching the Load 3D data command.
  • This command loads any 3D data pointed by the view (geometry, text attributes...) but no 2D component.
