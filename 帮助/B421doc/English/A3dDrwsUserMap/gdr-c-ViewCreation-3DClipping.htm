About 3D Clippings

The 3D clipping can be added on a generative view to define the 3D limits of the geometry to be drawn in view. This simplifies the view and gives a better visualization of the required view area. This topic provides more information about 3D clipping.

Miscellaneous

This sub-topic provides miscellaneous information about 3D clipping. A 3D clipping can only be added to an exact view (one clipping per view only).

3D clipping is not available on the following views:

  • CGR, raster, approximate views
  • Section cut views
  • Unfolded views
  • Views from 3D

The following two options have no impact on 3D clipping, regardless of whether or not they are selected:

  • Do not cut in section view (from the context menu: Front view object > Overload Properties > Characteristics > Editor Dialog Box)
  • Cut in section view

Removal/Addition of 3D Clipping

This sub-topic provides information about removing or applying 3D clipping.

Right-click the view and select View Definition > 3D Clipping. The following options are displayed:

  • Remove 3D Clipping: restores the entire view.
  • Apply 3D Clipping To: clips the selected 3D elements in another view.

Commands Available in the Context Menu

For a 3D clipping, it is possible to execute the following context menu commands from the tree:

  • Delete: You can delete the 3D clipping specification. In this case, the generative view needs an update. After updating the view, the generated geometry will not show the clipped specification.
  • Cut/Copy/Paste: You can move or duplicate the 3D clipping specification from one view to another.

Automatic propagation of the 3D clipping specification: Views created from an initial view (i.e. projection, auxiliary, section, detail views) will automatically get the same 3D clipping specification as the current initial view.

Configuration of 3D Clipping

This sub-topic provides information about configuring 3D clipping.

Cut elements are represented according to the same parameters as breakout views. You can configure these elements through Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > View section, under Geometry generation/Dress-up > View linetype, click Configure. In the Line and Thickness dialog box, choosing a type for breakout views also configures the representation of cut elements in 3D clipping views.

About 3D SPACE Clipping

The 3D volumes (spaces) are defined in the 3D and are linked to the ship definition, that is fire zones, decks etc. You can create and manage these 3D SPACEs in the Space Allocation app to be used by generative views to define the 3D SPACE clipping in the views.

Important: Only one 3D SPACE object can be associated to a generative view. The 3D SPACE object used by the generative view, has to be aggregated under the same root product as that of the 3D geometry drawn in the view.

The 3D SPACE clipping operation is compatible with all the following generative specifications:

  • All projection views
  • Offset Section View and Aligned Section View
  • All detail views
  • Breakout View
  • Broken View

Clipping Vs. Filtering

PLM filters are used by generative views to create views from filters. Whereas, the 3D SPACE clipping can be used to the views which are previously filtered.

Therefore, in short, clipping by a SPACE object is not to be considered as a filtering behavior.
