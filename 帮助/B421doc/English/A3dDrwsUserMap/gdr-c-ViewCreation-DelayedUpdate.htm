About Creating Quick Generative Views

Creating up-to-date views of assemblies involving large amount of data takes a long time. To avoid this, you can quickly create views and later update them explicitly and improve the productivity.

The following topics are discussed:

  • Simplified Result
  • Modifying Simplified Result

Related Topics
Creating Quick Generative Views

Simplified Result

A simplified result is displayed while creating the views till they are updated. This simplified result has the same behavior as that of a raster view. The raster view is generated taking into account view links, while the simplified result is computed considering what is visualized in 3D.

Note: To create views quickly, you need to go to Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Generation section, under Update, and select the Disable automatic update when view is created check box.

The following table depicts simplified results of various views during their creation:

Type of view Quick preview Simplified result Result
View From 3D



While creating a view, you can visualize a quick preview and then a simplified result is generated as a pixel image. Note that:

  • FTA annotations are not generated.
  • 2D layout elements are not created.

View From Slides



Slide is visualized in 3D

 While creating a view, you can visualize a quick preview and then a simplified result is generated as a pixel image.

The result depends on what is visualized in 3D and activation or deactivation of slides.

Unfolded View



Unfolded context is not visualized in 3D

 While creating a view, you can visualize a quick preview and then a simplified result is generated as a pixel image.

The result depends on whether unfolded context is visualized in 3D or not.

Section or Section Cut View



While creating a view, you can visualize a quick preview and then a simplified result is generated as a pixel image.

The section and section cut views are seen as projection views. Cut operation is not performed.

Detail or Quick Detail View



While creating a view, you can visualize a quick preview and then a simplified result is generated as a pixel image.

There is no difference in the representation of detail or quick detail views.

Modifying Simplified Result

You can modify the simplified results by creating various views such as broken view, breakout view, 3D clipping view and so on.

Notes:

  • To modify the views quickly, you need to go to Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Generationtab, under Update, and select the Disable automatic update when view is created check box..
  • The result of the modification depends on the selection of this check box.
  • If you do not select this check box, the simplified result is not displayed. The regular update is computed considering view properties and modification operation (clipping, breakout, etc.).

The following table depicts the simplified results of various views during their modification:

Type of view Quick preview Simplified result Result
Broken View



The simplified result displays the broken operation.
Breakout View, Apply Breakout To, 3D Clipping Box View, Apply 3D Clipping To



Existing simplified result is kept. It does not consider the breakout operation.
