Modifying a Callout Geometry

Re-creation of Callouts

This sub-topic provides information about the re-creation of callouts.

The following types of callouts can be re-created:

  • Section callouts
  • Detail callouts
  • Projection callouts

Conditions

Callouts may be re-created in one of the following conditions:

  • The callout has been manually deleted.
  • The view containing the callout has been deleted.
  • The view was cut and pasted to another drawing.
  • The view containing the callout has been modified and the callout became invalid ("red" callout).

Furthermore:

  • The projection plane of each view must be parallel in case of detail callouts and perpendicular in case of projection and section callouts.
  • The 3D origin of the reference view and the view in which the callout is to be created must be equal.
  • The orientation (first direction, second direction, and normal) of the view projection plane must be equal.
  • The views must be linked to the same product/part.
  • The view's section profile must be equal in case of detail callout re-creation.
  • Multiple callouts cannot be recreated from a single defining view.
    Important: Callouts can be re-created in an aligned section view even if some inconsistency may appear in the drawing. From a deleted view, the new callout is re-created with the same characteristics as the previous view .

Callout Types

The recreated callout type (section, detail or projection) is given by the resulting view definition. It is not defined or specified by you.

Red Callouts

This sub-topic provides information about red callouts.

Modification of Callouts

If you modify an auxiliary or a section callout in a way that the plane of the resulting view is not parallel to the previous plane view, or if you reverse the profile:

  • All the callouts of the resulting view are displayed in red.
  • You cannot edit them.
  • The change is not propagated to the detail view.

Important: If you resize a section callout constrained by the supporting lines of the profile:
  • Any detail callout does not turn red.
  • It can be edited.

Translation of Section Callouts

If you translate a section callout, the plane of the resulting section view is modified. It remains parallel to its previous orientation, yet there is a an inconsistency between the section view and the detail view defined on the previous plane view. As a consequence:

  • The detail callout becomes red.
  • Other callouts remain unchanged.
  • All callouts can be edited.

Important: If you create a detail view on the section view and translate the section callout without any visible impact on the resulting section view, the detail callout becomes red anyway and is editable.

Size of Callouts and View Scale

This sub-topic provides information about the size of callouts related to the view scale.

You can specify that the size of callout elements should not be dependent on the view scale. To do this, right-click the callout, go to Properties. In the Callout tab, select Size not dependent on view scale check box.

Types of Modifications on Different Callouts

This sub-topic provides information about direct modification of the callout geometry.

modification

You can modify a in the following ways:

Manipulation in one direction according to the callout Manipulation in two directions according to the callout

For aligned section callout profiles:

  • You can modify the length of the callout segment.
  • You cannot move the middle segment.


For aligned section callout profiles:

  • You can modify the length of the callout segment.
  • You can swivel the first and the last segment.
  • You cannot move the middle segments or change their locations.


For offset section callout profiles:

  • You can modify the length of the callout segment.
  • You can move the middle segment of the callout with three or more segments sideways.
  • The perpendicularity constraint is maintained


For offset section callout profiles:

  • You can freely move the first and the last segment.
  • You can move the middle segment of the callout with three or more segments sideways.
  • The perpendicularity constraint is maintained

Detail callout profile modification

You can modify a circular or a polygonal detail callout profile.

In a circular detail callout profile, you can perform the following edits:

  • Change the radius and location of the center using the central handle
  • Change only the radius using the manipulator on the circle
  • Change only the location of the circle using the rectangular frame
  • Replace the existing profile using Replace Profile .
Circular detail callout profile (allowed movement indicated with arrows)

In a polygonal detail callout profile, you can perform the following edits:

  • Change the location of the vertices using the individual yellow handles
  • Change the profile area spread using the square handles in white
  • Change the location of the whole polygon using the rectangular frame
  • Replace the existing profile using Replace Profile .
Polygonal detail callout profile (allowed movement indicated with arrows)

If you edit the callout profile that intersects with a broken zone, only one side of the definition view is represented. A warning message appears in a dialog box to indicate this. After the edition, if the profile is in broken zone, another warning message appears near the top bar indicating the same.

Important: During or after the modification, you can use the undo command to retain the original callout profile.

Auxiliary callout profile modification

You can perform the following manipulations to modify an auxiliary callout profile:

  • Point manipulation
  • Free manipulation
  • Replace profile
  • Reverse view direction

The above manipulations function the same way as in a section callout profile modification.

Important:
  • You cannot edit an auxiliary callout profile that intersects a broken area.
  • During edition, a callout is restricted to enter a broken area. If you do enter the broken area, a warning message appears near the top bar indicating the same. Also, the handles are restricted to move into the broken area.

During or after the modification, you can use the undo command to retain the original callout profile.
