About Modifying a View's Links

This topic provides more information about modifying the list elements pointed by a view and resolving broken links.

The following topics are discussed:

  • General Information
  • Broken Links
  • Exploded Views
  • Scenes

Related Topics
Modifying Links of a View

General Information

This section provides general information about modifying the list of elements pointed by a view.

  • To add or modify links, the reference 3D representation must be opened. If it is not opened, you will only be able to remove links.
  • All views must have the same 3D associated document.
  • You can select products or sub-products if it is an assembly view, and bodies if it is a part view.
  • If you want to modify links in your view when the related 3D document is not opened, you can only remove links.
  • You cannot perform Undo and Redo operations.

Broken Links

This section provides information about resolving broken links.

  • As long as you do not remove broken links, you cannot modify the view at all.
  • If all links in a view are broken, after you remove them all, you will be back in the Link Modification dialog box, with the whole assembly being selected.

Exploded Views

This section provides information about modifying the links of exploded views.

If you modify the links of an exploded view, all the views generated from the assembly will be modified. For more information, see Creating an Exploded View From Design Review.

Scenes

This section provides information about modifying the links of scenes.

  • If you modify the links on a scene, you can select the scene only.
  • You cannot mix scene links and products or parts' links. To select products in an assembly, start by removing any existing scene's link in the Link Modification dialog box, then select products or parts' links only.
