View Update Report Dialog Box

The View Update Report dialog box appears after you click Update, if a problem is encountered.

This dialog box provides information about the problems encountered during update.

  • General
  • View Update Boolean Error

Notes:

  • The View Update Report is displayed, among other things, when a Boolean operation fails.
  • You can re-open the View Update Report dialog box by right-clicking the view where a red cross is displayed and selecting X View Object > Update Report.

General

For each error encountered during the update process, the following information is displayed:

Severity
Indicates the severity of the problem (error, etc.)
View
Indicates the name of the impacted views.
Sheet
Indicates the name of the impacted sheets.
Description
Describes the problem encountered, and provides a solution to solve it.

View Update Boolean Error

The View Update Boolean Error dialog box appears when a Boolean error occurs while updating the drawing. This dialog box helps you find out the cause of the error. A separate error dialog box appears for each Boolean error, if multiple errors are detected.

3D Viewer
Shows the geometry involved in the error. The three elements displayed in the viewer and their color coding is as follows:
  • White: The body involved in the error and to be operated
  • Yellow: The generative tool used for the Boolean operation
  • Blue: The temporary diagnosis geometries
Inaccurate face contact after sectioning a body: Color coded error display

Error Description
Displays the details of the error. This includes:
  • The details of the geometry like the part, the body, and the view
  • The generative operation performed like section view or cut, detail, clipping, breakout, or 3D clipping
  • The full Boolean error message
3D Viewer Tools
Lets you show or hide and reframe the three elements of the viewer. Select the check boxes to show or hide the Initial 3D Body (white), the Generative tool (yellow), and the Diagnosis Geometries (blue). Click Reframe to reframe the corresponding element.
If there are multiple objects, you can browse between them.
