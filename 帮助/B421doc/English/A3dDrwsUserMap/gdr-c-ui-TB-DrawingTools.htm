About Using Drawing Tools

You can use a number of drawing tools to create a drawing. These tools are available in the following areas:

  • Tools
  • Visualization
  • Tools Palette

Tools

The tools related to geometry creation are available in the Sketch section of the action bar.

Snap to Point

This option makes your geometry (as well as 2D components) begin or end on the points of the grid. As you create geometry, points are forced to the intersection points of the grid. Note that this option is also available via Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Geometry section.

In this example, the black spline is created using the Snap to Point option. The points are on the grid.

Conversely, the highlighted spline is created when this option is not used. The points are not necessarily on the grid.



Important: You can use auto detection even if this option is selected.

Create Detected Constraints

This option creates lasting constraints. If you do not use this icon, the constraints you create are temporary: the geometry is only temporarily constrained, which means that it can subsequently be moved without being constrained.

Preserve modifications of graphical properties

Important: This command is available in the View Layout section of the action bar.

This option preserves the graphical modifications you apply to the generated geometry when the view is updated. To restore graphic modifications in a view, right-click the view frame and select View object > Restore Properties. For more information, see Managing the Graphical Properties of Generated Geometry and Dress-Up. By default, this option is selected.

Visualization

The visualization options are available in the View section of the action bar.

Grid

This option displays the grid in your session. The grid helps you to draw geometry in given circumstances. For example, the grid makes it easier to draw profiles requiring parallel lines. Note that this option is also available via Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Geometry section.

Show Constraints

This option displays the constraints. For more information, see Applying Constraints.

Display Frame

This option displays the view frame which is available for each view, as specified in each view's properties (right-click the view and select Properties, View tab, Display View Frame check box). In this case:

  • The view frame is displayed for all views for which the Display View Frame check box is selected.
  • The view frame is hidden for all views for which the Display View Frame check box is cleared.

The view frame is hidden for all views when this option is not used.

Elements' Analysis

This option lets you visualize the colors assigned to the different types of dimensions.

These displayed colors correspond to the colors customized in the Preferences dialog box. To modify these colors, go to Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Display tab. Then select the Activate element's analysis check box and click Configure to assign the desired color(s) to the desired dimension types.

Tools Palette

The Tools Palette appears whenever you select a command for which specific options or value fields are available. This enables you to immediately know when tools are available for a command.

For example, when creating a line, the values of the elements you are sketching appear in the Tools Palette as you move the pointer. In other words, as you are moving the pointer, the Length (L) and Angle (A) boxes display the coordinates corresponding to the pointer position.

Important: Value boxes in the Tools Palette:
  • By default, typing any value fills the first box.
  • When you select another box, the value in the previously selected box is locked. This allows you to move the pointer over the work area.
