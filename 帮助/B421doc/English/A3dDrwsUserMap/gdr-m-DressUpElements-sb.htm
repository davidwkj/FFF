Dressing Up Elements

The Drawings app enables you to create a number of view dress-up elements on existing 2D elements. Note that dress-up commands are actually provided with the Drawings app. Therefore, this section of the documentation actually points to the Drawings User's Guide. As such, the information detailed in this section is presented in an Drawings context.
