Related Guides

Before using Drawings, you need to understand the basic concepts and capabilities of the 3DEXPERIENCE platform.

Guide Description
3DEXPERIENCE Basics Describes the key concepts and capabilities of the 3DEXPERIENCE platform, such as the Compass, 6WTags, and 3DSearch.
3DEXPERIENCE Advanced Services Describes the user interface and tools common to all apps. This guide complements 3DEXPERIENCE Basics.
Data Exchange Interface User's Guide Describes how to import and export external files in miscellaneous formats, including DXF/DWG and CGM.
Sketcher User's Guide Describes how you can sketch 2D elements.
Enterprise Knowledge Language Reference Guide Provides reference information about the different types, methods and functions used by the Enterprise Knowledge Language (EKL) to define programs and operations specific to apps.
