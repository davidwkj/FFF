Secondary Area

The secondary area of the Sketch section provides commands for instantiating a 2D component, a fix together item, closing an arc, analyzing a sketch and so on.

  • Flyout for Closing Elements

Instantiate 2D Component
Instantiates a 2D component.
Instantiate Group
Creates a Group instance with the existing Group items.
Close arc
See Flyout for Closing Elements.
Sketch Analysis
Analyzes all the geometries in the current sketch.

Close arc
Closes a partial arc of a circle or an ellipse.

Creates the other part of an arc of a circle or an ellipse.
