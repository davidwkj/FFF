Template Section

The Template section provides commands for creating templates which are the reusable components of a drawing. This section is available in the action bar when a detail sheet is the current sheet.

Text Template
Creates text instances which can be reused as notes anywhere in the drawing.
See Managing Text and Table Templates.
Label Template
Creates labels which are custom annotations and can be instantiated or stored in catalogs.
See Working with Labels.
Table Template
Creates tabular instances which can be reused anywhere in the drawing.
See Managing Text and Table Templates.
2D Component
Creates sets of reusable 2D geometries and annotations.
See Working with 2D Components.
