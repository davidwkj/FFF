Drafting Interface in Editability Mode

When opening a drawing representation in editability (read-only) mode, a number of authorized commands and visualization modes are limited to prevent unwanted modifications.

For more information about the read-only mode, see Managing Editability.

  • Commands Available in Editability Mode
  • Visualization Modes Available in Editability Mode
  • Features Unavailable in Editability Mode

Commands Available in Editability Mode

The following commands are available in editability mode:

  • Load 3D data (in a Generative Drafting context)
  • Filter Generated Elements (in a Generative Drafting context)
  • Export to Layout view
  • Export 2D component structure
  • Auto Search
  • Update report
  • Analysis
  • Show constraints
  • Query Object Links...
  • Print...
  • Select geometrical elements
  • Dimension System Selection

Visualization Modes Available in Editability Mode

The following visualization modes are available in editability mode:

  • Display View Frame as Specified for Each View
  • Create Associative Dress-up Note: however, even if this mode is selected, no dress-up can be created in a read-only drawing.
  • Show Geometry in All Viewpoints
  • Create Detected Constraints Note: however, even if this mode is selected, no constraint can be created in a read-only drawing.
  • Analysis Display Mode
  • Snapgrid
  • Grid

Features Unavailable in Editability Mode

The following features are unavailable in editability mode:

  • The Object Properties panel.
  • Commands related to object edition.
