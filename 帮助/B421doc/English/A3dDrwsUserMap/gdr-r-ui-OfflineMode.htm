Offline Mode

The Drawings app is compatible with the Offline mode. It enables you to work for a limited period of time without connecting to the system.

For more information about the Offline mode, see Social and Collaborative Apps: Global Design Management: Collaborative Lifecycle: Working Offline.

All Drawings commands, except those listed in the table below, are available in Offline mode.
