Symbols

The various symbols used to identify the elements created in the tree are described in this reference section.

Important:
  • Right-click an element in the tree to use the traditional commands (Delete, Cut/Copy/Paste, Reframe On, etc.).
  • The Unbreak, Unclip, Remove Breakout, Apply Breakout to and 3D Clipping commands can now be found under View Definition.
  • Double-click a specification or a callout to launch the corresponding profile edition command (when available).
  • You can paste the specifications of a view to another one, whenever the specifications are compatible with the selected view (if you want to have an idea of which specifications can be pasted to a given view, check which commands are available in the View Layout section of the action bar when this view is selected).
  • You cannot paste a broken specification if broken areas overlap.
  • You cannot paste a clipping specification if the projection plane are not parallel.
