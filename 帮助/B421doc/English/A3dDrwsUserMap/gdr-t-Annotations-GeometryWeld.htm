Creating a Weld

You can create a weld. You can set text properties either before or after you create the text.

Before you begin: Open a drawing representation.

  1. From the action bar, click Weld .

    This command is not available by default. To add it to the action bar, select Me > Preferences > Customize > Commands, and drag and drop the command onto a section.

    For more information, see Customizing Options.

  2. Select a first element. For example, a line.

  3. Select a second element. For example, another line. The default weld automatically appears on the drawing.



  4. Optional: In the Welding Editor dialog box, modify the weld. For example, modify the thickness from ten to five millimeters.

  5. Optional: Modify the type of the weld by clicking the Change Type.

  6. Click OK.

    The weld is created.

Important:
  • You cannot modify the area fill corresponding to the weld.
  • Welds are not associative to the geometry. Therefore, if you modify the geometry, the weld does not move accordingly.
