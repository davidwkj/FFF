Copying View Format

You can apply one or more properties of a view to different target views.

Before you begin: Open a drawing representation.

  1. In the tree, right-click a view and select XXX object > Copy View Format.

  2. In the Copy View Format panel, clear the unnecessary target views.

    By default, all the views in the list are selected.

  3. In the Format Options list, clear the unnecessary options:

    • Under Copy Source GVS:
      • DefaultGenerativeStyle
    • Under Graphic Properties:
      • Copy Overrides
      • Copy Overloads
      • Copy Dress-ups
    • Under Operator Specifications:
      • Copy Broken
      • Copy Breakout
      • Copy3D Clipping

    Note: The above options are available depending on the properties of the source view. By default, all the available options are selected except the Dress-ups option.

  4. Click OK.

    The properties are applied to the selected target views. A message appears informing that only the applicable format options are copied.

    Note: Only those options applicable to a particular target view are applied to that target view, even if you have selected all the options.

    For example, if a target view has a different generative view style than the source view, the override copy is not performed for that target view.
