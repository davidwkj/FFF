Creating a Drawing Interactively

You can create a drawing or a specialized drawing which inherits standard drawing representation with some additional attributes.

This task shows you how to:

  • Create a Drawing Using the Compass
  • Create a Drawing Using Add > Content

Before you begin: Open a 3DEXPERIENCE platform session.

Create a Drawing Using the Compass

You can create a drawing using the Compass in the top bar.

  • From the Compass, click 3D Modeling Apps and select Drawings.

The Drawings app is loaded and an empty drawing representation is created. The first empty sheet, Sheet.1 is created in the tree.

Create a Drawing Using Add > Content

You can create a drawing using the Add > Content command.

  1. In the top bar, select Add > Content or press Ctrl+N.

    The New Content dialog box appears listing the Favorites and the Most Recently Used objects.

  2. Right-click Drafting > Drawing in the list and select Set attributes at creation.

    This allows you to define the drawing parameters before its creation. However, if you do not select this option, the drawing is initialized with the default parameters, (these parameters are: standard, sheet style, sheet orientation, generative view configuration - if applicable, product attachment).

    Important: This choice is saved as a preference and used as the default value for the next usage of drawing creation.

  3. Click Drawing.

    The Drawings app is loaded and an empty drawing representation is created. The first empty sheet, Sheet.1 is created in the tree.
