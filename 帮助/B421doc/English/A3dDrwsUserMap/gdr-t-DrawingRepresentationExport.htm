Exporting a Drawing Representation (Save As)

You can save a drawing representation in a format different from the original one:

  • .cgm

  • .dxf

  • .dwg

  • .gl2

  • .ig2

  • .jpg

  • .pdf

  • .ps

  • .svg

  • .tif

For more information about dealing with 2D IGES (*.ig2) data and DWG/DXF data, see the Data Exchange Interfaces User's Guide.

For more information about exporting drawing representations to other formats, see the 3DEXPERIENCE Basics User's Guide: Importing and Exporting.

Before you begin:

Create or open a drawing representation.

  1. In the top bar, select Share > Export.

  2. In the Export dialog box:

    1. In the Format list, select the required file type to save.
    2. In the Filename box, enter the required file name.
    3. Click to select the required path.

  3. Click OK.

    The drawing representation is saved in the specified format.
