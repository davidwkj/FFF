Loading/Opening 3D Data

You can load 3D data for one or several views, sheets or for a whole drawing, with or without opening its associated 3D product.

This task shows you how to:

  • Load 3D data
  • Open 3D data

Before you begin:

Open a standalone generative drawing representation. For this scenario, do not edit its associated 3D object. Thus, the drawing, sheet and views are seen as unreferenced in the tree.

Related Topics
About Loading/Opening 3D Data

Load 3D data

You can load 3D data.

  1. Select one or several views or sheets in the tree.

    Tip: To load 3D data for all sheets and views, select the drawing node in the tree.

  2. Right-click and select Load 3D data.

    The 3D data is loaded.

    Notes:

    • In the tree, the broken links symbol is replaced by the update status.
    • You can also update the views which have their 3D data loaded.

    When a drawing contains broken views, the 3D data is not loaded and a warning message is displayed.

  3. Optional: Click Load Root to load the whole assembly.

    The Load Root command is only available if all view links are broken.

Open 3D data

You can open 3D data.

  1. Select a view or a sheet in the tree. If needed, use the multi-selection.

  2. Right-click and select Open 3D data.

    The 3D data is opened.

    When a drawing contains broken views, the 3D data is not opened and a warning message is displayed.

  3. Optional: Click Open Root to open the whole assembly.

    The Open Root command is only available if all view links are broken.

The Product Structure Design app is displayed and the associated product is open. Back in the Drawings app, note that the selected views or sheets are updated.
