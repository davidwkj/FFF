Setting Relations Between Dimensional Constraints

You can set constraints between dimension using formulas. As a result, if you modify one of these dimensions (the driving dimension), all the other dimensions as well as the geometry will be modified accordingly.

Before you begin:

Create a drawing representation with dimensional constraints.

  1. From the Tools section of the action bar, click Formula .

    The Formulas dialog box appears. You will now select, one after the other, the dimensions to be constrained and then enter in the dialog box the formulas to be used. Here is the drawing we use in this scenario.



  2. Select a first dimension (1).

  3. Click Add Formula in the Formulas dialog box.



  4. In the Formula Editor dialog box, select a second dimension (3) and add "\4". Then, click OK.

  5. Back in the Formulas dialog box, select a first dimension (2).

  6. Click Add Formula.



  7. In the Formula Editor dialog box, select a second dimension (3) and add "*3 /4". Then, click OK.

  8. Back in the Formulas dialog box, select a first dimension (4).

  9. Click Add Formula.



  10. In the Formula Editor dialog box, select a second dimension (1) and then, click OK.

  11. Back in the Formulas dialogs box, select a first dimension (5).

  12. Click Add Formula.



  13. In the Formula Editor dialog box, select a second dimension (2) and then, click OK.

  14. Back in the Formulas dialog box, click OK.

    Note that the tree is modified accordingly.

  15. Double-click the dimension to be set as driving dimension (3).

    The Dimension Value dialog box appears.

  16. If needed, activate the Drive geometry.

  17. Enter 100mm as dimension new length and press OK.

    All the dimensions you previously constrained using formulas are automatically updated.
