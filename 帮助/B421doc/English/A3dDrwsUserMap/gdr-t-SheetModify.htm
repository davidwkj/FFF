Modifying a Sheet

You can modify a sheet by changing its standard, sheet style and orientation. This means modifying the options you selected when defining the sheet.

Before you begin: Create a drawing sheet using the ISO standard, the A0 ISO format, and the Landscape orientation in the New Drawing dialog box.
Related Topics
Sheet Properties

  1. From the View Layout section of the action bar, click Page Layout .

  2. In the Page Layout dialog box, select ANSI from the Standard list.

    If the standard file has been modified by the administrator, click Update.

    A message informs you that this action cannot be undone.

    The parameters of the chosen standard are copied into the drawing and replace the previous parameters. This may have an immediate impact on the appearance of the elements inside the drawing.

  3. Select the required sheet style.

    The sheet style defines the sheet format, size, scale and default orientation.

    Tip: You can also change the sheet style (sheet format) by right-clicking the sheet, and selecting Properties.

  4. Optional: Change the default orientation from Landscape to Portrait.

  5. In the Dress-up list, select Drawing Titleblock Sample 1.

    The Dress-up list contains the default frame and title block templates.

  6. Under Frame and Title Block, click Create to create the frame and title block.

  7. Optional: If the drawing contains several sheets, specify whether the selected options apply to the current sheet or to all the sheets.

  8. Click OK.

    The sheet is modified accordingly.
