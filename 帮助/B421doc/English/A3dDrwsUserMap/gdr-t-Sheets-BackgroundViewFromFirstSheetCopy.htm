Copying Background View from First Sheet

You can copy the background view (title block plus elements) from the first sheet of a drawing representation to the sheets created afterwards in this representation.

Before you begin: In a drawing representation, create a frame and a title block.
Related Topics
Working with Sheets

  1. Select Me > Preferences.

  2. In the Preferences dialog box, select Mechanical > Drafting.

  3. In the General tab, under Sheet Creation, select Copy sheet background from first sheet check box.

    Important: If you clear Copy sheet background from first sheet check box, only the sheet properties are copied from one sheet to another.

  4. Click OK.

    The preferences are applied.

  5. Right-click the sheet in the tree and select New Sheet .

The frame and title block created in Sheet.1 are inserted in the background of Sheet.2.
