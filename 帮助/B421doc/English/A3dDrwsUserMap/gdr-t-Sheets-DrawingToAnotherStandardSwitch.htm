Switching a Drawing to Another Standard

You can switch a drawing to another standard when you have several standards defined by your administrator.

Each drawing contains a copy of the standard it uses, and retains this version until you explicitly switch the drawing to another standard or update this copy.

Before you begin: Create new drawing representation. Make sure you specify the standard that you want to use.

About Switching a Drawing to Another Standard

  1. From the View Layout section of the action bar, click Page Layout .

  2. In the Page Layout dialog box, in the Standard list, select another standard.

    A message informs you that this action cannot be undone.

  3. Click OK to continue.

    Important: You can only switch a drawing to a standard of an equivalent or higher level than the current standard of your drawing. For more information, refer to your administrator. If you try to switch a drawing to a standard of a lower level than the current standard of your drawing, an error message appears.

  4. Click OK to validate.
