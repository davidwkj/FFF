Adding a Volume 3D Clipping to an Exact View

You can define a 3D SPACE object in the Space Allocation app, and use its definition in a generative view to clip the 3D geometry seen in this view.

This task shows you how to:

  • Add a 3D SPACE Clipping
  • Modify a 3D SPACE Clipping

Before you begin: Create a 3D SPACE object or 3D SPACE data in the Space Allocation app.

About 3D Clippings

Add a 3D SPACE Clipping

You can add a 3D SPACE clipping to a 3D volume data created in the Space Allocation app.

Before you begin: Create a 3D SPACE object.

  1. From the View Layout section of the action bar, click Add 3D Clipping .

  2. In the Clipping Object dialog box, in the Clipping Mode list, select the 3D SPACE clipping and enter the User Input.

    The User Input shows the 3D SPACE object box displaying the initial state when the 3D SPACE object is not selected, and the 3D access shortcut with the Open 3D Data. You can select the 3D SPACE object by switching to the product manually or use the Open 3D Data .

  3. Click Open 3D Data.

    It opens the product allowing you to select the desired 3D SPACE object.

  4. Select the desired 3D SPACE in the tree.

    Note: You need to select only dedicated 3D objects to be added to a view as a SPACE Clipping.

    You are taken back to the drawing representation. The 3D SPACE object box in the Clipping Object dialog box now shows the selected object.

  5. Click Apply and then click Create.

    The front view is clipped with the selected 3D SPACE object and the dialog box is closed.

    Simultaneously, the 3D SPACE Clipping.X node is added in the tree under the View Specifications node.

Modify a 3D SPACE Clipping

You can modify a 3D SPACE clipping by changing the 3D SPACE object by another one.

Before you begin: Create a 3D SPACE clipping.

  1. Double-click the 3D SPACE Clipping.x node under the View Specifications node in the tree.

  2. In the Clipping Object dialog box, click Open 3D Data.

    It opens the product allowing you to select the desired 3D SPACE object.

  3. Select the desired 3D SPACE in the tree.

    You are taken back to the drawing representation.

  4. Click Apply and then click Modify.

    The view is modified respecting the changed 3D SPACE object.
