Creating a Breakout View

You can create a breakout view from a generated view by using a cutting profile to section a view in desired dimensions to visualize the internal area.

Before you begin: Open a drawing representation containing a view from which you want to remove material. Double-click this view to activate it, and make sure it is up-to-date.

Modifying a Breakout View

  1. From the View Layout section of the action bar, click Breakout View .

  2. Click in the sheet to create the first point of the breakout profile.

  3. Click as many points as required to create the profile, and click the first point to close the profile or double-click to end the profile creation.



    The 3D Viewer appears, displaying the 3D part corresponding to the generated view and an arrow indicating in which direction the breakout will be created.

    Note: The 3D Viewer lets you position the 3D part. It also lets you visualize a view plane and use it to define the depth of the breakout.

  4. Perform either of the following operations:

    • In the 3D Viewer, drag the arrow to the required location. The breakout will not be associative with any element.
    • In the drawing, select an axis, a center line or an edge (circular or linear) of a perpendicular view. The 3D plane will be moved accordingly. This method lets you break the view through a hole or a face, for example. The breakout will be associative with the selected element.

      For example, select the center line to use it as an extremity plane, as shown below.



    • In the 3D Viewer:
      1. Click the arrow to activate the reference element selection.
      2. Select a point, a line, a circle, a face, or an axis as a reference element to define the depth of the breakout.
      3. In the Depth (in mm) box, enter a value (you can also define the depth by dragging the arrow).
      This method lets you define the depth of the breakout precisely. The breakout will be associative with the selected reference element and with the depth value.

      Note: You can select any one of the following for plane translation during reference selection:

      • Translate plane to geometry:

        Moves the plane according to the selected element. By default, this option is selected.

        If the selected element is perpendicular to the viewing plane, the breakout plane is positioned at the centre of the element. Else, it is positioned at the start or end of the element.

      • Translate plane to bounding box:

        Allows selection of plane using bounding box of the selected element. The bounding box defines two alternative reference faces displayed in white. The selected face is highlighted with cyan color.

  5. Optional: Under Display, click the following options to change the visualization accordingly:

    • Clipping tool
      • Display background and clipping tool: Shows 3D object as well as clipping tool.
      • Display clipped background and clipping tool: Shows the clipped 3D object and the clipping tool.
      • Display clipping tool: Shows only the clipping tool.
    • Transparency: to modify transparency of the clipping tool.
    • Turn viewpoint: to turn the viewpoint of 3D object and the clipping tool to Iso, Front, Rear, Left, Right, Top, or Bottom.

  6. Click OK.

    The breakout view is created.
