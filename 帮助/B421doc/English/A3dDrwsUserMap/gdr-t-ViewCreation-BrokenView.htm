Creating a Broken View

You can create a broken view, which is used to shorten elongated objects, using two profiles corresponding to the part to be broken from the view extremities.

This task shows you how to:

  • Create a Broken View
  • Create a Broken View in a View Containing a Breakout Profile

Before you begin: Open a generative drawing representation containing a view to be broken. Double-click this view to activate it, and make sure it is up-to-date.

Create a Broken View

You can create a broken view by defining a single or multiple broken profiles.

  1. From the View Layout section of the action bar, click Broken View .

  2. Click a first point corresponding to the first extremity of the first profile.

    A green dotted profile appears which allows you to position the profile either vertically or horizontally.



  3. Perform either of the following operations:

    • Click a second point corresponding to the profile second extremity. In this example, click a point so that the profile appears horizontally.
    • Move the pointer over the green horizontal profile that appears as you create a first point and, if needed, click to locate the second point on this first profile.



  4. If needed, translate the profile.

    Red zones appear. These red zones correspond to the zones out of which the view cannot be broken. Note that the orientation of the second profile is the same as the orientation of the first profile.



  5. Click a point to define the position of the second green profile that appears.

  6. Click in the sheet.

    The broken view is created.



Create a Broken View in a View Containing a Breakout Profile

You can create a broken view which already contains a breakout profile.

Before you begin: Open a generative drawing representation and create a breakout view.



  1. Click Broken View .

  2. Click a first point corresponding to the first extremity of the first profile.

  3. Click on the required area to get a vertical or a horizontal breakout.

    For this scenario, create a horizontal profile.

  4. Click a point inside the view to indicate the position of the second breakout line.

    You can select a point residing on the breakout profile, or you can also encompass the whole breakout profile with the new broken profile. For this scenario, select a point residing on the breakout profile.



  5. Click in the sheet.

    The broken view is generated.
