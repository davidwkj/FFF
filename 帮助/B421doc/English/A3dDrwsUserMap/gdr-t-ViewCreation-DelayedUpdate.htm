Creating Quick Generative Views

You can quickly create and modify views of large assemblies or large data. This prevents the automatic update, thus enabling you to complete the definition of the drawing or views. Later, as per your choice, you can explicitly perform the update (even in batch mode).

When creating views, a simplified result is displayed till the views are updated.

This task shows you how to:

  • Create Views
  • Modify Simplified Views
  • Isolate Views

Before you begin:
  • Open a 3D part representation and generate a new drawing with an empty sheet.
  • Go to Me > Preferences > Mechanical > Drafting > Generation section and select the Disable automatic update when view is created and Disable automatic updates when view is modified check boxes in the Update area.

Create Views

You can quickly create a front view, isometric view, projection view, auxiliary view, and views using wizard.

For this scenario, you will create a front view.

  1. From the View Layout section of the action bar, click Front View .

    For more information, see Creating a Front View.

  2. Select a reference plane on the 3D area geometry.

    You can visualize a quick preview.

  3. Click anywhere on the sheet to generate the view.

    The view is created quickly and a simplified result is generated, seen as a pixel image. This result and the preview are based on what is visualized in 3D area. If 3D area data is not opened, default visualization is taken into account.

    Note: The created view is seen as not up-to-date and the Update icon is activated.

    Important: With this simplified result, it is not possible to:
    • Create dimensions, annotations, and 2D objects requiring the selection of generated elements.
    • Perform the following operations:
      • Section
      • Breakout
      • 3D clipping box
    • Take into account view properties such as:
      • Overload properties
      • Axis, center lines, threads, hidden lines, fillet, 3D color and so on.

    Note: Once you perform the update, these operations or properties will be taken into account and the view will be up-to-date.

Modify Simplified Views

You can modify the simplified views by creating clippings, breaks and so on.

For this scenario, you will create a clipping view.

  1. Activate the front view created in the earlier task.

  2. Click Clipping View .

    Note: To create a clipping view by creating a profile, click Clipping View Profile . For more information about how to create a clipping view, see Creating a Clipping View/Clipping View Profile.

  3. Select a point or click in the work area to define the circle center.

  4. Move the pointer to define the circle radius.

  5. Click to end the profile creation.

    The clipping view is created. A simplified result is displayed taking into account the clipping operation.

    Note: The simplified results of clipping view and a quick clipping view show no difference.

Isolate Views

You can isolate a simplified view. However, the visualization of the view changes after isolation.

  • Right-click the view and select XXX object > Isolate.

    The view is no more displayed in gray and becomes up-to-date.
