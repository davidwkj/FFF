Creating an Exploded View From Design Review

You can generate a front view or an isometric view from a slide created in the Design Review app.

Important: It is necessary to create the slides in product markup and not in review markup.

To know more about slides, see Design Review User's Guide: Working with Slides.

Before you begin:
  • Open a product containing an assembly:
    1. Create a markup object inside a product using contextual command Insert > Markup of a product. If you have an existing markup object under a product, set it as current.
    2. Navigate to Design Review and create slides.
  • Generate its drawing representation.
  • In Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > View section, select the Inherit 3D colors check box.

About View Creation

  1. From the View Layout section of the action bar, click Isometric View .

  2. Select the slide in the 3D area

    The desired component positions and specific attributes are displayed.

    Important:
    • Make sure that you select the slide before clicking in the sheet to create the view, otherwise you will create a view of the whole assembly.
    • It is not possible to create a view from a slide linked to a sub-product. You can create a view only from a slide that is directly linked to a root product.

  3. Select the 3D geometry.

    The 2D window is automatically displayed.

  4. Click in the sheet to create the view.

    An isometric view of the slide is created.
