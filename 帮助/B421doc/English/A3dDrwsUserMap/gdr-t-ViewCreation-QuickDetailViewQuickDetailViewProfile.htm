Creating a Quick Detail View or a Quick Detail View Profile

You can quickly create a detail view using either a circle or a sketched profile as callout.

Before you begin: Open a 3D part representation, then create new drawing.

Creating a Detail View or a Detail View Profile

  1. Perform either of the following operations in the View Layout section of the action bar:

    • To create a quick detail view using a sketched profile, click Quick Detail View Profile .
    • To create a detail view using a circle as callout, click Quick Detail View .

    In the Creation Options panel, you can select the graphic properties and the operator specifications that you can propagate to this view from its reference view. If the Disable generative view style usage check box is cleared in Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Administration tab, then you can also apply the available generative view styles to this view. For more information, see Creating a View using Generative View Styles.

    In this scenario, create a quick detail view using a sketched profile.

  2. Select the points required for sketching a polygon. If you are creating a detail view using a circle as callout, click the callout center.



  3. Double-click to end the cutting profile creation.

    Note that if you do not close the profile, it is automatically closed. If you are creating a detail view using a circle as callout, drag to select the callout radius and click a point to end the selection.

  4. Optional: Select the 2D generative edge in this view to create an associative detail profile.

    A message appears at the top right corner of the screen informing that the detail view is now associated with the selected generative item. For more information, see About Detail Views.

  5. Click to generate the quick detail view.

    The associative detail view is created and the message disappears automatically once you indicate the location of the resultant detail view by clicking at the appropriate location in the sheet. Unlike a detail view , the profile is entirely closed.

    Note: The default value of scale is 2 (twice the scale of the active view). You can modify this scale from the Properties dialog box.

  6. To modify the scale, right-click the detail view, select Properties, in the View tab.

  7. Enter the required Scale and then click OK.
