Working with Views with Object Filters

You can create a view using object filters previously propagated (saved) with Product Finder and edit the filters in the current view.

For more about object filters, see the Product Finder User's Guide.

This task shows you how to:

  • Create Views with Object Filters
  • Edit the Object Filters of the View

Before you begin: Open a drawing representation and a product containing an assembly. Create an object filter (previously PLM filter). See Social and Collaborative Apps : Global Design Management: Product Finder User's Guide: Filtering data.

About Views with Filters

Create Views with Object Filters

  1. From the View Layout section of the action bar, select the Front View .

    You can use filters when creating any view type.

  2. Open the work area in the 3D tab.

  3. In the Apply a Filter panel, hover over the filter icon, and click the edit icon .

    If this panel does is not displayed by default, click the filter icon at the right hand side of work area.

    The Search - Filters related to: XXX panel appears. Where XXX is the name of the current product. The dialog box displays the list of Embedded filters and All filters (if any) associated with the product.

    • Embedded filters: lets you select a filter that is already embedded in the product.

    • All filters: lets you select a filter that has been propagated in the database, but not embedded in the product. Once you select a filter in the All filters list, it gets embedded in the product with the suffix _3dm, and can be reused by selecting Embedded filters.

  4. Click to select a filter in the All filters list.

    The filter is displayed in the Apply a Filter dialog box, with the suffix _3dm, indicating that the filter is now embedded in the current product.

    Note: If you select a filter which is already embedded in the product, a confirmation message appears asking you whether to create a copy of the embedded filter (with suffix ".n", where "n" is an integer which is incremented each time a new copy with the same name is made). Clicking OK creates a new indexed copy of the filter embedded in the product.

  5. Optional: Click to remove the filter.

    The filter is removed from the dialog box. You can search for another the filter and select it by following the steps three and four above.

  6. Now that you have applied the filter, create the view.

  7. From the product structure or in the tree, select a plane or a surface plane.

    The 2D window is automatically displayed.

  8. Click in the sheet to select a location for your view.

    The view is created displaying the 3D elements filtered by the selected filter.

    The created view has a link with the embedded filter.

Edit the Object Filters of the View

  1. Select Me > Preferences > Object Filters.

    In the Object Filters dialog box, you can see the following:

    • The name of the relevant product. You can select the product to view the filters aggregated under it.
    • The filters aggregated under the selected product with their status. A filter can have the following statuses:
      • : The filter is referenced in the current representation.
      • : The filter is referenced, but it is not modifiable. The filter object may be reserved by another user or the current user has no suitable authoring or editing permission.
      • : One or more locked views use this filter.
      • : The filter is unsynchronized because of no or multiple filters that match.

  2. Select a filter in the list and click one of the following:

    1. Upgrade selected filters to upgrade the filters to the new specifications of the original filter (the one which is listed in All filters section of the Apply a Filter dialog box and has no _3dm suffix after its name.)

      The new specifications are copied from the original filter.

    2. Replace selected filters to replace with a different filter.

      You can now select the Click here to retrieve filters related to your context to display the Search - Filters related to: XXX panel again. From this panel, select the filter to replace the existing filter.

    Both upgrading and replacing are manual operations. The standard synchronization mechanisms such as Update, Links & Relations do not synchronize the filters.
