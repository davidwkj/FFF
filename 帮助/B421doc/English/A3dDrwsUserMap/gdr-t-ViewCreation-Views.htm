Creating Views

You can create different views. If the sheet is active, the first view you create is by default a front view.

Before you begin: Create a drawing representation.

Creating Views

  1. From the View Layout section of the action bar, click New View .

  2. Click in the drawing to position the new view.

    An empty view is created, displaying a blue axis in a red frame.

    The drawing tree is updated to show the newly created view. A specific icon is added to the tree. For more information, see The Tree.

    If you change your system's regional settings to use another language, the default view name will be translated according to the language used by your system (both in the view and in the tree). However, this only applies to default view names; custom view names are not translated.

    You can now create 2D geometry in this view.

  3. Click New View again and select a projection direction to create more views.

    The views are created. As they are linked to the front view, they are projection views. The drawing tree is updated again to show the newly created views.

    Tip: If you need to switch to the third angle projection method, modify the sheet properties.

  4. Double-click the frame to activate one of the projection views. For example, double-click the front view frame.

  5. Click New View to create the top view.

    Note: The view name and the view scale are displayed in the view as per the options set by the administrator. For more information, see Managing View Texts.
