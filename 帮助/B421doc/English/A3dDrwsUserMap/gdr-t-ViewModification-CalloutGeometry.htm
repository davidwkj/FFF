Modifying a Callout Geometry

You can modify the geometrical characteristics of a callout used to create detail views, section views, and auxiliary views.

Important: Callouts represent an operation that is performed on a generative view (such as a section cut) and consequently do not behave as a geometry. You cannot isolate callouts, nor can you apply transformation or symmetry operations to callouts.

Before you begin: Open a generative drawing with callouts for section view, detailed view, and auxiliary view.
Related Topics
Creating Views

  1. Double-click the callout to be modified.

    • A context toolbar appears.
    • A set of handles appear that define the callout to be modified.

  2. Perform the modifications on the callout geometry.

    You can select the following commands from the context toolbar:

    • Replace Profile
    • Reverse View Direction
    • Point Manipulation
    • Free Manipulation

    The commands available in the context toolbar depend on the type of callout. For more information, see Types of Modifications on Different Callouts.

  3. Once the modification is completed, click anywhere in the work area.

    The modification is applied to view.
