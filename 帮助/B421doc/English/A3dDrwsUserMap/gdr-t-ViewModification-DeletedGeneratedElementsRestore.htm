Restoring Original Properties/Deleted Elements

You can restore original properties of generated elements. You can also restore deleted elements.

This task shows you how to:

  • Restore Original Graphical Properties of Generated Elements
  • Restore Deleted Elements

Before you begin: Perform the steps detailed in Changing the Graphical Properties of Generated Geometry.

About the Graphical Properties of Generated Geometry

Restore Original Graphical Properties of Generated Elements

You can restore the original graphical properties of generated geometry and view dress-up (thread, axis line, center line).

Notes: You can restore the original graphical properties of generative geometry only, and in exact views only.

  1. Right-click the view frame, and select Front object > Restore Properties.

    In the Restore Original Properties dialog box, the following options are available:

    • Overloaded Properties: Select the properties you want to restore (show, layer, symbol, color, linetype, thickness).

    • Highlight: To highlight the properties concerned by the selection, in the view.

    • Deleted Elements: To restore deleted elements.

    Note: The Preview is available only for views generated with the Exact generation mode.

  2. Under Overload Properties, select the Thickness and Color check boxes.

    You can select the Highlight check box to visualize the corresponding element in the view.



  3. Click OK.

    The original graphical properties are restored.



Restore Deleted Elements

You can restore the deleted elements.

Note: You can restore the deleted elements in all types of views.

  1. In the drawing window, select the circle in the view and press Delete.

    The circle is now deleted from the view.

    For this scenario, immediately restore the circle you deleted, but you could perform other operations before doing this.



  2. Right-click the view frame, and select Front object > Restore Properties.

    Note: As deleted elements are not visualized in the view, they cannot be highlighted.

  3. In the Restore Original Properties dialog box, under Deleted Elements, select Deleted check box, and then click OK.

    Important: Remember that this is applicable only in exact views. If the view you are working in is not an exact one, the elements you deleted will be displayed again when updating the view.

  4. Click Update current sheet to see your changes.

    The drawing is updated with the latest changes on the part, and the deleted item is restored in the view.
