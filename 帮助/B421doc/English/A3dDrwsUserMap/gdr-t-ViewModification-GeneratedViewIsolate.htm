Isolating Generated Views

You can isolate a generated view. Isolating a view amounts to suppressing associativity between an existing 3D representation and the corresponding generated view. This results in transforming a generated view into an interactive view.

Important:

  • A view that has been isolated cannot be de-isolated.
  • Center lines, threads and axis lines in isolated views are no longer associated to geometry.
  • Dimensions in isolated views are no longer associated to geometry.

This task shows you how to:

  • Isolate a Selection of Views
  • Isolate All Views in the Drawing

Before you begin: Open a generated drawing representation.

Isolate a Selection of Views

You can isolate selected views.

  1. Select the views you want to isolate, for example the Top, Bottom, Left and Right views, and right-click them.

  2. Select Selected objects > Isolate.

    Important: For the purpose of this scenario, several views are selected, but if you select only one view, the left view for example, then you need to select Left view object > Isolate from the context menu.

    The selected views are now isolated.

If you perform one or more modifications on the 3D representation, you will not be able to update the isolated views.

Isolate All Views in the Drawing

You can isolate all views in the drawing.

  1. In the tree, right-click the drawing object to display the context menu.

  2. Select Drawing object > Isolate.

    All views in the drawing are now isolated.

If you perform one or more modifications on the 3D representation, you will not be able to update the drawing.
