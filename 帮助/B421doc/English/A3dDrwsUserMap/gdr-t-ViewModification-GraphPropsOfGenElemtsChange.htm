Changing the Graphical Properties of Generated Geometry

You can change the graphical properties of generated geometry.

Note: This functionality is available in exact views only.

Before you begin: In a 3D window, create a cube with a hole. In the Drafting app, project a front view of this cube, using the exact view mode.
Related Topics
About the Graphical Properties of Generated Geometry
Restoring Original Properties/Deleted Elements

  1. From the View Layout section of the action bar, select Preserve Modifications of Graphical Properties .

    The graphical properties you apply to generated geometry will be preserved when the view is updated.

  2. Using the Graphic Properties from the Object Properties panel, apply:

    • A blue color on the left vertical element,

    • A thickness of 2.6000 mm to the right vertical element.

    The graphical properties of these elements are modified.
