Modify View Links and Resolve Broken Links

You can modify the list of pointed elements for a view and resolve broken links.

  1. Create a front view of the 3D representation.

  2. Right-click the view and select Front view object > Modify Links.

  3. In the 3D area, press Ctrl and select several 3D elements.

    Back in the drawing, the 3D elements are listed in the Link Modification dialog box.

  4. Click Add all, and select the Preview check box to see what the 3D representation looks like according to your selection.

  5. Click OK, and then click Update .

    The selected elements are displayed in the front view.

  6. Right-click and select Front view object > Modify Links.

    The Link Modification dialog box is displayed with links related to the parts.

  7. Select an element and click Remove, and select the Preview check-box to view your 3D representation.

    The preview is updated.

  8. Click OK, and then click Update .

    The front view is updated.

    Important: If you remove all the pointed elements, you are back to step 2. You can start selecting links again.

  9. In the 3D representation, press Ctrl and select an element.

    Back in the drawing, this element has been added in the Link Modification dialog box.

  10. Click OK, and then click Update .

  11. In the 3D representation, delete this element.

    A Broken link symbol is displayed in the drawing tree.

    Note: In this case, the view becomes unreferenced and cannot be updated unless the broken links are resolved.

  12. Right-click the view and select Front view object > Modify Links.

    A message is displayed, advising you to remove broken links.

  13. Click OK to continue.

    The Link Modification dialog box is displayed with the broken link highlighted so that you can remove it.

  14. Click Remove, and then click OK.

  15. Click Update .

    The drawing is updated accordingly.

Modify Links of a View

You can change the associated root context of a view by selecting one or several links in a new assembly.

Before you begin: Create a product (product A for example) containing a drawing representation and another product (product B for example).

  1. Create a front view and a projection view of product A.



  2. Right-click the view frame and select XXX objects > Modify Links.

  3. In the Link Modification dialog box, select another product (product B for example) in the 3D area and go back to the drawing.

    The selected product appears in the 3D elements to add box.

    The following options are available:

    Symbol Description


    		Adds the element to the Manage Pointed elements box.


    		Removes the element.

  4. Click Add .

    The Pointed document box now shows the newly selected product. A message appears informing that the selected links are not corresponding to the current pointed root product. All the internal links to the current root product will be isolated (dimensions, overloaded properties, ..).

    Notes:

    • When you click Add , all the current links are removed if they are not associated to the new root context.
    • When you switch from a product instance to the reference of this product instance, the links are retained. The dimensions, overload properties, annotations, and other properties are still associative. A message appears informing that the views linked to the same subassembly will be modified.

      In this process, any views which belong to the same instance, product A, but are not updated are also linked to the reference of the instance, product B.

    • In case, when similar instances are created, you can modify the links of an instance to another similar instance.

      In the Reroute operations dialog box, the similar instances are displayed in different tabs. When you select a tab, the Pointed document box shows the newly selected product.

      A message appears informing that all the views linked to the same subassembly will be modified.

    Tip: When you right-click the element in the Manage Pointed elements box, you can access Reroute to Reference and Reroute operations commands.
    Important:
    • The Reroute to Reference and Reroute operations commands are available when the Manage Pointed elements box shows only one element.
    • The similar instance tab is displayed in the reroute operations dialog box only when the product is open in the 3D editor window.

    The preview of newly selected product is seen in the preview window.

  6. Click OK and then click Update current sheet .

    The drawing is now linked to the selected product (product B in this case) and the view links are updated accordingly.
