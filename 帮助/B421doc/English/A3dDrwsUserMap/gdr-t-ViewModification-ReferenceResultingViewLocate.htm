Locating Reference/Resulting Views

You can easily locate reference views or resulting views when needed.

In drawings containing many section views, section cuts, auxiliary views or detail views, it is sometimes necessary to navigate between reference and resulting views.

This task shows you how to:

  • Locate a Reference View
  • Locate a Resulting View

Before you begin: Open a drawing representation containing reference/resulting views.
Related Topics
About View Creation

Locate a Reference View

You can locate a reference view.

  1. Right-click the detail view.



  2. Select Locate Reference View. The app window is reframed on the view where the detail view callout is defined.



Locate a Resulting View

You can locate a resulting view.

  1. Right-click the detail view callout.



  2. Select Locate Resulting View. The app window is reframed on the detail view.
