Locking a View

You can lock a view.

Locking a view may be useful if you need to modify a part without modifying the generated views. For example, when you need to modify a constraint in the 3D but not the generated views.

Before you begin: Open a drawing representation.
Related Topics
Managing Views

  1. Right-click the view to be modified (either the view frame or the view itself).



  2. Select Properties.

  3. In the View tab, under Attributes, select the Lock View check box.

    The other fields are unavailable, indicating that you can no longer edit them.

  4. Click OK to lock the view and exit the dialog box.

    The view is locked, which is indicated by a lock symbol on the view in the tree.

    Important: Once a view is locked, you will not be able to:
    • Create views from this locked view.
    • Create either annotations, dimensions, or 2D geometry in this locked view.
    • Update, delete or cut this locked view.
    • Move generated geometry, 2D geometry, annotations, dimensions and dress-up elements in this locked view.

    However, you can still move a locked view.
