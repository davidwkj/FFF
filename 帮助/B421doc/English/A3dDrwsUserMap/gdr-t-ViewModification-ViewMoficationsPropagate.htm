Propagating View Modifications

You can propagate view modifications to other views.

While doing this, the Synchronize View Definition command helps remove inconsistencies that can occur in a drawing between two views, when one view is inherited from the other, due to the non-propagation of view/section plane modifications.

This task shows you how to:

  • Propagate a Section Profile Modification
  • Propagate a View Definition Plane Modification

Before you begin: Open a generative drawing representation. Tile your windows vertically to see both your part and the related drawing.

About Propagating View Modifications

Propagate a Section Profile Modification

You can propagate a section profile modification.

  1. Create a front view of the part.

  2. Create a section view of the front view.

  3. Then create a detail view of the section view.



  4. Modify the section profile in the front view.

    The section view is updated, but the detail callout is displayed in red, highlighting the existence of an inconsistency between the detail view and the front view (which is its parent view).

  5. Right-click the detail view and select Detail X object > Synchronize View Definition to synchronize the detail view with respect to the front view.

    The detail view is synchronized, and the callout is no longer displayed in red. The inconsistency has been removed.



Propagate a View Definition Plane Modification

You can propagate a view definition plane modification.

  1. Create a new drawing using the ISO standard.

  2. Create a view from 3D on the Front view.



  3. Change the definition plane of the FrontView.1 in the 3D window.

  4. In the drawing tree, select Front View.1 object > Synchronize View Definition.

    A message asking you if you want to synchronize the view projection plane is displayed.

  5. Click OK.

    The view has been synchronized.
