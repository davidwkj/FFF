Reframing a View

You can reframe any type of view such as front view, isometric view, detail view, clipping view, etc. so as to display only required part of it.

However, note that you cannot reframe a 2D component reference.

Before you begin: Create or open a drawing representation with a view.
Related Topics
Creating Views

  1. Right-click the view frame and select Properties.

  2. In the Vew tab, under Attributes, select the Activate Clipping Frame check box.

  3. Click OK.

    A new frame that is always rectangular appears in the view.

    You can now define the position and size of your frame on the view.



  4. Click the frame to select it.

  5. Drag the handles to resize the frame as required.

    For example, resize the frame so as to display about a quarter of the view.



  6. Now, drag one of the boundaries of the frame to specify its position on the view.

    For example, move the frame so as to display only the upper-left area of the view.

    The frame is now displayed in the view as you defined it.



    Note: To remove the frame and display the view as it was originally before you re-framed it, clear the Activate Clipping Frame check box.
