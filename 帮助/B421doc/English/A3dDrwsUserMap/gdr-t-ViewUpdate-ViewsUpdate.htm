Updating Views

You need to update the representation whenever there are any changes done in the 3D product or in any of the text attribute links. This task shows you how to:

  • Update Views using Update
  • Update Views using PLM Update

Before you begin:

Open an existing product and the associated drawing representation. Modify the 3D area or modify a text attribute link.

Update symbols must appear in the tree to indicate the drawings , sheets and views that need to be updated. The Update current sheet and the Update must be active.

Note: To avoid updating a drawing when only minor changes have been performed in the 3D area, Update symbols will not be active after a change in the 3D color or 3D point symbol, when these are inherited from the 3D area (as specified in the Generation Properties of the Properties dialog box).

Update Views using Update

You can update views using Update command.

  1. Open an existing 3D product and the associated drawing representation. Modify the 3D geometry.

  2. Click Update current sheet to update all views in the active sheet.

  3. Select and right-click the views and select Update Selection to update the selected views.

  4. Type C:Force Update in the Power Input box (available in the Status Bar) to update the drawing in accordance with the 3D.

Update Views using PLM Update

You can update views using PLM update.

  1. Go to Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Assembly Design > Update section, under Automatic Update Status Computation, and check whether the computation is set to Yes or No. This defines how the PLM update status is computed.

    Setting the option to Automatic ensures that the update status is automatically computed.

  2. Click Update .

    The Confirm PLM Update dialog box is displayed, indicating that during the update process, memory is optimized so that modifications that have no consequence on data update are ignored.

    Important: You can select the Do not show this message again check box. In this case, the dialog box will not be displayed next time you launch the Update All command.

  3. Click Yes.

    A dialog box is displayed while the update is being performed.

    • All of the 3D area associated to the current drawing is updated, if necessary.
    • All of the generated views of the drawing are updated, on all sheets of the drawing.
    Important: Note that if a problem occurs while the 3D area is being updated, the update process is interrupted and the views are not updated. On the other hand, if a problem occurs while the views are being updated, the update process continues and subsequent views are updated (provided that the Stop update on first error check box is cleared in Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Assembly Design > General section, under Update behavior on errors).

Once the update is complete, the Update All and the Update icons are both disabled.

If a problem was encountered during update, an error report is displayed reporting both 3D update errors and views update errors. For more information, see View Update Report Dialog Box.
