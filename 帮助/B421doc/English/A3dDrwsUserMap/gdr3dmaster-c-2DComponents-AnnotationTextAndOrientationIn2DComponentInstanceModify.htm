About Modifiable Text in a 2D Component Instance

This sub topic provides information about modifying annotation text in a 2D component instance.

Important:
  • Once a text becomes modifiable, it is not possible to make it non-modifiable.
  • Clear Disable direct manipulation check box available through Me > Preferences > Mechanical > Drafting > Geometry tab. If this check box is selected, you cannot manipulate the component, nor modify the text strings in 2D component instances.

Attribute Link

You can add attribute links in the text content.

VPM Catalog

For 2D component instances created with a VPM catalog, if a text becomes modifiable in the catalog, you have to synchronize the external reference to make the 2D component instance text modifiable too (see Design IP Classification User's Guide: Creating a Catalog).

2D Component Reference Containing an Instance

If you create a 2D component reference (called MYREF for example) containing a 2D component instance with a modifiable text, the text will not be modifiable in instances of MYREF.

2D Components with Text Used as Symbol

To use 2D components with text as symbols, select these check boxes:

  • Apply Scale (available when you right-click and select Properties > Text tab, under Options)
  • Apply scale at creation to compensate view scale (in Me > Preferences > Mechanical > Drafting > Geometry tab)

The size of both the 2D component and its text will then be independent from the view scale.

About Text Orientation

You can fix the text orientation in a 2D component. In the 2D component instances, the text will keep the same orientation even if the 2D component instance is rotated or flipped.
