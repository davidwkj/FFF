About the Projection Methods

There are two types of projection methods used for creating projection views; first angle and third angle. You can specify the type of projection method while creating views via the wizard. This topic provides more information about the projection methods.

The following topics are discussed:

  • What Are the Projection Methods?
  • The First Angle Standard
  • The Third Angle Standard
  • How to Specify a Projection Method?

What Are the Projection Methods?

This topic explains what the projection methods are.

Projection views are created using a projection method which is based either on the first or on the third angle standard. In both cases, it is an orthographic representation comprising the arrangement, around the principal view of an object, of some or all of the other five views of that object. For more information, see No. ISO 10209-2:1993.

Note: The ISO and ANSI standards are independent from the angle projection method.

When creating views via the wizard, you are explicitly requested to specify the angle projection method you want to use.

The First Angle Standard

This topic provides information about the first angle standard.

With reference to the principal view, the other views are arranged as follows:

  • The view from above is placed underneath.
  • The view from below is placed above.
  • The view from the left is placed on the right.
  • The view from the rear is placed on the left or on the right, as convenient.

The Third Angle Standard

This topic provides information about the third angle standard method.

With reference to the principal view, the other views are arranged as follows:

  • The view from above is placed above.
  • The view from below is placed underneath.
  • The view from the left is placed on the left.
  • The view from the rear is placed on the left or on the right, as convenient.

How to Specify a Projection Method?

This topic provides information on how to specify a projection method.

The projection method is originally defined by the sheet style, as specified in the standards. For more information, see Sheet Styles.

You can change the projection method through the sheet's properties. To do so, right-click the sheet in the tree and select Properties. In the Properties dialog box, click the Sheet tab, under Projection Method, select either First angle standard or Third angle standard.
