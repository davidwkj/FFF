About Frames or Sub-frames

Frames and sub-frames can have fixed or variable sizes. You can change the size of the frames.

The following topics are discussed:

  • Variable or Fixed Size Frames
  • Frame Size
  • Frame Color Fills and Frame Margins

Variable or Fixed Size Frames

Each frame can either have a variable or a fixed size.

Variable size frames adapt to the text length, whereas fixed size frames always remain as they are, no matter what the text length is. So if you choose a fixed size frame and the length of your text exceeds the frame size, then the text will extend beyond the frame.

Frame Size

The size of frames depends on the following factors:

  • Whether it is used as a frame or a sub-frame
  • The height between the characters top and bottom or cap and base
  • Margins

Thus, a frame or a sub-frame may look different although the text it is applied to is identical.

Frame Color Fills and Frame Margins

You can create frame color fills and frame margins only for global frames.

Frame Color Fill
You can add color fill to the frames of annotations.


Notes:

  • You can fill only the text and balloon annotations.
  • You can fill color only if the selected annotation has a frame.

Frame Margins
You can manage the horizontal and vertical margins of a text annotation.


Notes:

  • The Margin H and Margin V boxes are available only if the selected annotation frame type supports either the horizontal, vertical, or both margins.
  • Only the margins of variable frames can be managed.
