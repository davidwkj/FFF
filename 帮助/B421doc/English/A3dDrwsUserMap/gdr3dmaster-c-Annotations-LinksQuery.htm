Querying Annotation Links

You can query the annotation links with Query Object Links that is available in the context menu of all the annotations containing links.

It displays the Query Link dialog box showing all the links of the selected objects.

The following columns are displayed in the dialog box giving specific information about the links.

  • Type: It shows the type of the link between the object and the corresponding reference. The following types are possible:
    • Positional link
    • Positional link on leader
    • Orientation link
    • Orientation link on leader
    • Instantiated link template
    • Attribute link
    • Welding specification

  • Value: It shows the value of the attribute referenced by a link of the following type:
    • Instantiated link template
    • Attribute link

    For other types of link, this column remains empty.

  • From: It shows the object that contains the link, displayed as a path from its container. In case of Drawings, the container is the view while in case of 3D Tolerancing & Annotation, it is the annotation set.
  • To: It shows the object pointed by the link, displayed as a path inside its PLM component. If the object is not found in a session, it shows the status of the object. This status can be:
    • Not loaded
    • Broken
    • Isolated
  • In: It shows the PLM component containing the pointed object displayed in the column To. It is shown only if the link is external and can be resolved. Otherwise this column is empty.

Re-frame the window: When you select this check box and choose a link in the Query Link dialog box, it focuses and highlights the annotation and the object pointed by it.

However, the annotation and the object pointed by it are highlighted only if they exist in the active representation. If none of these objects exist in the same representation, they are not highlighted.

Similarly, the annotation and the object pointed by it are focused only if they exist in the active representation and in the current sheet. If none of these objects exist in the same representation and in the same sheet, they are not focused.
