About Modifying a Table

This topic provides more information about editing and modifying the table using keyboard shortcuts. The keyboard shortcuts help to speed up table edition.

You can use the following keyboard shortcuts:

Keyboard shortcut Action
Left, Right, Up, Down,

Home, End, PageUp, PageDown

To change the cell selection.
Shift + (Left, Right, Up, Down), Home, End, PageUp, PageDown To extend the selection.
F2 To switch from one edition mode (full or quick) to the other.
Del To clear the content of the selected cells.
Shift + Del To clear the format of the selected cells.
