Creating Constraints via SmartPick
Creating Constraints

SmartPick dynamically detects the following geometrical constraints:

  • Support lines and circles
  • Alignment
  • Parallelism
  • Perpendicularity
  • Tangency
  • Concentricity
  • Horizontality
  • Verticality
  • Middle point

You can choose whether or not to create the constraints detected by SmartPick. To create the constraints, you can use either of the following ways:

  • Select the Create detected and feature-based constraints check box in Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Geometry tab.
  • From the Sketch section of the action bar, select Create Detected Constraints .

Furthermore, note that:

  • When a constraint is detected by SmartPick, you can temporarily:
    • Deactivate this constraint by keeping the Shift key pressed.
    • Lock this constraint by keeping the Ctrl key pressed.
  • In a Drawings context, you cannot create geometrical constraints with extremities and midpoints of any generated line element, except with generated curves.
Important: Smartpick on background does not detect FTA construction geometries.
