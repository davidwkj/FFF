Creating Dimensions

General Dimension Creation Rules

This sub-topic provides information about dimension creation rules.

First of all, bear in mind that dimension creation in Drawings and 2D Layout for 3D Design follows the general rules which apply to geometry creation in V6: the geometry to dimension must be contained within a box whose dimensions are 2.e+6mm (the coordinates can vary from -1,000,000 mm to +1,000,000 mm). Therefore, it is impossible to create dimensions for elements exceeding these dimensions.

You can create (and therefore modify) dimensions on one element, or on two elements.

Except for driving dimensions and angle dimensions, a dimension value cannot be equal to 0.

Dimension Attributes

When dimensioning, you can modify a number of attributes.

While dimensioning (before clicking to validate the dimension creation), you can modify the dimension, modify the dimension type, create/modify angle dimensions, and use the dimensioning tools available in the Tools Palette.

While dimensioning, or just after having validated the dimension creation, you can modify the dimension, modify the dimension value text position, modify the dimension overrun/blanking (either one or both), add text before/after the dimension value or modify the dimension type.

Dimension Properties

Once a dimension has been created, you can also modify its properties in the Properties dialog box.

The following tabs may be available, depending on the type of dimension selected:

  • Chamfer Properties
  • Dimension Line Properties
  • Dimension Texts Properties
  • Extension Line Properties
  • System Properties
  • Tolerance Properties
  • Value Properties

Dimension Options

Dimension creation will also be impacted by the options defined in Me > Preferences > Mechanical > Drafting > Dimension section.

For more information, refer to Dimension.

Useful Creation Tips

You may find the following tips useful when creating dimensions or dimension systems.

  • When creating: To activate temporarily the Dimension following the cursor check box, press Ctrl.
  • When creating and editing: To activate temporarily the Snap value between symbols (SHIFT toggles) check box, press Shift.
  • When creating an angle dimension: If Dimension following the cursor check box is selected, you can swap the angle sector of the first dimension according to the pointer position by pressing Ctrl and Shift. If Dimension following the cursor check box is not selected, you can swap to the complementary angle sector by pressing Ctrl and clicking on the dimension line.
  • When creating an associative dimension on cylindrical geometry: The view plane has to be perpendicular or parallel to the cylinder axis.

Before You Begin Creating Dimensions

Before you begin creating dimensions, make sure you are familiar with:

  • The Tools in the Tools Palette. For more information, see About Dimensioning Tools.
  • SmartPick, an easy-to-use tool designed to assist you when creating dimensions. For more information, see Sketcher User's Guide: Using Smartpick.
  • Multi-selection. For more information, see the 3DEXPERIENCE Advanced Services: Selecting Objects.
