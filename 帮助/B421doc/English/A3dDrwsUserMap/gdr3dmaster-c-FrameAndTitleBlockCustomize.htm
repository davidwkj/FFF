Creating a Customized Frame and Title Block

About Customizing a Frame and a Title Block

Using the Page Layout dialog box, you can define a sheet's standard, its orientation and also insert a sheet frame and title block in a representation.

You can configure a frame and title block:

  • By creating a new frame and title block:

    With this, you can create a template of your frame and title block and instantiate when required.

  • By managing the default frame, title block and revision table: To configure the default frames and title blocks scripts, you must be familiar with Visual Basic.

About Sheet Dress-up Style

The default sheet dress-up styles for the frame, title block and revision table are available and can be selected in the Page Layoutdialog box.

However, note that only one sheet dress-up style can be selected at a time and all the actions of all the categories are displayed accordingly. The following dress-up styles are available by default:

  • Drawing Titleblock Sample 1 and Drawing Titleblock Sample 2: Two default styles that provide complete dress-up of a sheet frame, title block and a revision table.
  • Dress-up DS 1: The default dress-up style that provides several categories such as a frame, title block and revision table. Using this style, you can configure the frame, title block and revision table by creating templates. You can also instantiate a frame, title block and a revision table using this style.

Dress-up DS 1

For each category, a list of actions is displayed along with the preview. However, note that this preview does not show the current sheet's content.

Frame: With this category, you can create and manage a sheet frame in the current sheet. The following default actions are available with the Frame.

  • Instantiate: Checks to get the first sheet, which shares the same sheet format and orientation as the current sheet. If a suitable sheet is found, all the elements contained in its background are instantiated in the input sheet background. If no suitable sheet is found, an error message is displayed.
  • Remove: Deletes all the elements contained in the sheet background.
  • Resize: Resizes the frame and translates the existing title block and revision table (if any) and removes the existing frame and instantiates the new frame to keep their position relative to the resized frame.

Title Block: With this category, you can create and manage a title block in the current sheet. The title block is created using a table positioned in the lower right corner of the frame. The following default actions are available with the Title Block.

  • Instantiate: Checks to get the first table located directly in the sheet whose name starts with the prefix TitleBlock_.
    • If a suitable table is found, it is instantiated in the current sheet. The instantiated table is positioned in the current sheet at the lower right corner of the sheet frame.
    • If no suitable table is found, an error message is displayed.
  • Remove: Removes the first table located directly in the current sheet whose name starts with the prefix TitleBlock_.
  • Update: Retrieves the first table in the current sheet whose name starts with the prefix TitleBlock_ and updates the content of its cells which match a specific naming convention with a computed value.

Revision Table: With this category, you can create and manage a revision table in the current sheet. The revision table is created in the upper right corner of the frame. The following default actions are available with the Revision Table.

  • Instantiate: Checks to get the first table located directly in the sheet whose name starts with the prefix RevisionTable_.
    • If a suitable table is found, it is instantiated in the current sheet. The instantiated table is positioned at the upper right corner of the sheet.
    • If a suitable table is not found, an error message is displayed accordingly.
  • Remove: Removes the first table located directly in the current sheet whose name starts with the prefix RevisionTable_.
  • Add Revision: Retrieves the first table in the current sheet whose name starts with the prefix RevisionTable_ and adds a new line corresponding to a new revision.

Note: Once you change the value in the list, that value is saved as a user preference for the next usage.

About Default Template Initialization

Using the default template, you can quickly initialize the current sheet with the frame, title block and revision table.

The title block and revision table are created directly inside the sheet, whereas the frame is created in the sheet's background. This is done when you click Instantiate a default template in each category.

  • Frame: It creates a sheet frame in the background of the current sheet, based on the format of the sheet.
  • Title Block: It creates a table directly in the current sheet, adding the prefix TitleBlock_ to its name. The table contains link templates and cells with names that are matching with the keyword convention for a computed value.
  • Revision Table: It creates a table directly in the current sheet, adding the prefix RevisionTable_ to its name. The table contains cells with names that are matching with the keyword convention for a computed value.

Keyword Convention for Title Block and Revision Table

The default templates used for title block and revision table contain the text that is filled with a computed value. This value depends on the keyword which follows a specific naming convention.

The cell text must have a name starting with a specific keyword to declare that the cell content must be filled with the corresponding value. The keyword definition is not case sensitive.

The following table lists the keywords and their computed values used in the default title block and revision table.

Keyword Computed value
SheetIndex Index of the current sheet, among the sheet of same type (design or detail)
SheetCount Count of all the sheets of the same type as the current sheet (design or detail)
SheetFormat Format of the input sheet, for example, A0
SheetScale

Scale of the input sheet, for example, 1:1
Revision Revision identifier, incremented from the previous revision line in the table, for example, B
Date Current date, retrieved from the system
UserId Id of the currently logged user
Title Title of the product of first generative view, or the active root product, if any. If the cell is not a text template, an attribute link is created on the product, if any. If possible, replace text with associated VPMReference.
