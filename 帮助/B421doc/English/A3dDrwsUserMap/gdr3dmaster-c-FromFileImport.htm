About Importing CSV Files

A .csv file is a text file, that is comma delimited file. You can import a table from a .csv file.

Note that the table import is based on system options. In a CSV (Comma Separated Values) file, each line corresponds to a line in the table. Within a given line, each cell is separated by a specific character that depends on the decimal separator specified in the system options. This decimal separator determines the character that will be interpreted as the cell separator:

  • If the decimal separator is a dot, then the comma (,) is used as a column separator.
  • If the decimal separator is a comma, then the semi-colon is used as a column separator.

About Importing IG2 (2D IGES) Files

You can import an IG2 file into an existing layout representation.

Note: Clear the Map IGES 2D Layers with 2D Layout for 3D Design Sheets check box from Me > Preferences > All Preferences > 3DEXPERIENCE Platform Common Preferences > Compatibility > IGES 2D section, under Import, to import the IG2 file in the current sheet.

When you import an IG2 file, the following actions take place:

  • A new view (auxiliary view) is created in the current sheet.
  • The name of the current sheet is not modified.
  • The format and standard of the current sheet are not modified.
  • By default, the view is placed in the xy plane.

Otherwise, the creation of views and sheets depend on the layers. For more information, see Converter for IGES User's Guide: Dealing with IGES 2D Data: IGES 2D Concepts: About Layers

Notes:

  • If an IG2 file contains more than one view, the entities of all IG2 views are created in the current auxiliary view.
  • If an IG2 file contains more than one sheet, the entities of all IG2 sheets are created in the current auxiliary view.

Important: While importing an IG2 file, all the IG2 settings are taken into account. However, the drafting standard of the current sheet remains unaltered.

Sheet Background Color

Note: The background luminosity value ranges between 0 and 255.

  • If the background luminosity is lower than 100, the black color from IG2 is changed to white (so, both black and white are white).
  • If the background luminosity is between 100 and 200, no color is changed.
  • If the background luminosity is higher than 200, the white color from IG2 is changed to black (so, both black and white are black).
