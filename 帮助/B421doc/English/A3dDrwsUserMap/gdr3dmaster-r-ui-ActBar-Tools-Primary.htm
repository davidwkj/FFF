Tools Section

The Tools section provides commands for browsing a catalog, measuring between items, and creating frames and title blocks.

Catalog Browser
Instantiates a 2D component from a catalog.
See Instantiating a 2D Component from a Catalog.
Non Persistent Measure Between
Measures the distance between the two items.
See Measuring.
Non Persistent Measure Item
Measures the items.
See Measuring.
