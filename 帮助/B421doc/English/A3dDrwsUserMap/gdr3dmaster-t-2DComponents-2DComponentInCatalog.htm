Creating a Catalog

You can create a catalog from a representation containing one or more detail sheets.

For more information about catalog creation, see Design IP Classification User's Guide.

  • Insert a 2D Component/Label in a Catalog

About the Catalog Structure

Create a Catalog

You can create a catalog containing 2D components, text templates or labels.

  1. In the top bar, select Add > Generate Catalog.

  2. In the Catalog Browser panel, enter a name for your catalog.

    Important: The name you enter for the new catalog must not be already in use for another catalog in the database.

  3. Click OK.

    The catalog is created and saved, but it is not edited.

Insert a 2D Component/Label in a Catalog

You can insert a 2D component or a label in a catalog.

  1. From the Tools section of the action bar, click the Catalog Browser.

  2. Open the catalog.

  3. Right-click the required chapter node and then select Insert > Item: Feature.

  4. Back to the representation containing the 2D component reference, select the 2D component in the tree.

    Note: In case of label, select the desired label in the detail sheet.

    The Design IP Classification is automatically displayed. You can see that the 2D component/label is inserted under the chapter you selected.
