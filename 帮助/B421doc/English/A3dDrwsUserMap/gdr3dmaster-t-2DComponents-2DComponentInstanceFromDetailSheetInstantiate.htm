Exploding a 2D Component from a Detail Sheet

You can individually explode a 2D component that was instantiated from a detail sheet and then modify it as per your requirement.

Before you begin: Open a representation.
Related Topics
About Exploding a 2D Component Instance

  1. Right-click the component that was previously instantiated from the detail sheet and select 2D Component.x (Instance) object > Explode 2D Component.

    The component is now exploded. You can therefore modify the geometry and/or graphical properties on one or more elements of this component.

  2. Click the text and reposition it.

  3. Select one line on the top of the hexagon and use the Graphical Properties to change its color to red.
