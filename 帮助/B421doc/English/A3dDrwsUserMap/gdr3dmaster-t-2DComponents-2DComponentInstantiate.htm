Instantiating a 2D Component

2D Component lets you createYou can reuse a 2D component. You can also position the 2D component during its creation, add a leader to a 2D component and replace the reference of a 2D component instance.

In this particular case, we will instantiate a 2D component previously created in a detail sheet.

This task shows you how to:

  • Instantiate a 2D Component
  • Position a 2D Component During Creation
  • Add a Leader to a 2D Component
  • Replace the Reference of a 2D Component Instance

Before you begin: Open a drawing representation and create a 2D component reference in a detail sheet.

Instantiate a 2D Component

You can instantiate a 2D component.

  1. Double-click the view in which you want to instantiate the 2D component.

    This view is now active.

  2. From the Sketch section of the action bar, click Instantiate 2D Component .

    Tip: You can position several 2D components on the sheet and specify the scale and angle properties for all these components. For this, double-click the Instantiate 2D Component .

    Note: You cannot change the 2D component reference when doing this. All 2D component instances created using this method will have the same angle, scale and origin. You can modify these later on.

  3. Go to the detail sheet and select the 2D component.

    The drawing sheet automatically displays the sheet containing the starting view.



  4. Position the component using Smartpick.

  5. Click in the main sheet.

    The 2D component instance is created. Furthermore, the 2D component instance is added in the Component Instances node, under the appropriate view node in the tree.

  6. Optional: Select the component and use the displayed handles to modify the component.

Position a 2D Component During Creation

You can position a 2D component while creating it.

  1. Instantiate a component and click in the sheet to define its position.



    The Tools palette lets you modify the origin, position and orientation of the 2D component instance.

  2. In the Tools Palette, click Change Component Origin .

  3. Click a point on the component, which you want to use as the component origin.



    This new origin will move to fit into the position you indicated for the component. You can see that the component origin has moved to the defined position.

  4. Click Change Component Position in the Tools Palette.

  5. Click a point to define a different position of the 2D component.



    The component origin moves to the selected point.

    Notes:

    • This command moves the position of the 2D component to the selected point.
    • This modification in change of component position is applied to the currently selected 2D component instance which is created during the command execution.

  6. Click in the view.

    The 2D component instance is created.

  7. Optional: Flip the component according to either the x or y axis.

    Click Flip component horizontally , to flip the component on the horizontal axis of the detail.

    Click the Flip component vertically , to flip the component on the vertical axis of its reference.

Add a Leader to a 2D Component

You can add a leader to a 2D component instance.

  1. Create a 2D component instance.

  2. From the customized section of the action bar, click Add Leader .

    This command is not available by default. To add it to the action bar, select Me > Preferences > Customize > Commands, and drag and drop the command onto a section. For more information, see Customizing an Action Bar by Adding Commands.

  3. Select the required 2D component.

  4. Select the element you want to associate to the 2D component, or click in the sheet.

    The leader is added.

    Note: The leader is associated to the selected element from the origin of the 2D component.

Replace the Reference of a 2D Component Instance

You can replace the reference of a 2D component instance.

  1. Create a 2D component instance.

  2. Right-click the instance and select 2D Component.X (Instance) object > Replace Referencefor this instance.

  3. Select another instance or a 2D component in a local detail sheet.

    The 2D component reference is replaced.
