Creating a 2D Component Reference

You can create a detail sheet and then position a 2D component on this sheet. This 2D component will then be instantiated on a design drawing sheet.

Differentiating the design sheet from the detail sheet lets you assign a structure to the representation. This means separating the drawing elements from the reusable components.

Before you begin: Open a drawing representation.

Instantiating a 2D Component

  1. In the tree right-click the DrawingXXX node and select New Detail Sheet.

    The detail sheet is created and listed in the tree.

    Tip: You can add a new detail sheet after this one, either by repeating the above step, or by right-clicking the newly created detail sheet in the tree, and selecting New Detail Sheet.

  2. Specify a name for the sheet in the Sheet Browser at the top of the work area.

    If you change it, the new name is updated in the tree.

  3. From the Template section of the action bar, click 2D Component .

  4. Click anywhere on the sheet to define the 2D component view position.

    The 2D component view is created at the indicated location.

  5. Create a 2D component.

    For example, we create two circles in the detail sheet.



    Important: Note that you can customize both the design and the detail sheet background colors. For more information, see the 3DEXPERIENCE Advanced Services

  6. Optional: Right click the 2D component and select Add leader .

    A leader is added to the 2D component.
