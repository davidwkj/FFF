Modifying Annotation Text and Orientation in a 2D Component

You can modify the text and orientation in 2D component instances. It helps you to visualize the annotation text in a better way.

This task shows you how to:

  • Modify the Text in a 2D Component Instance
  • Specify Annotation Orientation

Before you begin:
  • Open a drawing representation.
  • Make the detail sheet active, and then the 2D component view (double-click the sheet and view in the tree).
  • Insert text in the 2D component.
  • Create a 2D component instance.

Modify the Text in a 2D Component Instance

You can modify the annotation text in a 2D component instance.

  1. From the catalog or detail sheet, right-click the 2D component reference text.

  2. Select Modifiable in instance.

    You are now able to modify this text in existing instances and/or in newly-created instances.

  3. In a sheet containing an instance, double-click the 2D component instance text.

  4. Modify the text and then click OK.

    The text is modified in this 2D component instance.

Specify Annotation Orientation

You can specify the annotation orientation to read it properly.

  1. Right-click the text and select Properties > Text tab.

  2. In the Orientation box, select Horizontal to orientate the text horizontally with respect to the sheet.

  3. Create two 2D component instances:

    • One without modifying the orientation

    • One with orienting the 2D component instance during creation (set the angle with the sheet to 130deg, for example).



    Note that the text orientation defined in the 2D component reference is kept for both the instances.
