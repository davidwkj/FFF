Accessing and Customizing Preferences

You can access and customize preferences for Drawings to suit your working habits.

  1. From the top bar, select Me > Preferences, and then click the All Preferences tab.

  2. Select the Mechanical > Drafting category in the left-hand box.

    The options for Drawings are organized in tabs.

  3. Select the tab corresponding to the options to be customized.

    Options in this tab... Allow you to customize...
    Drawing the standard to be used when creating drawings, and options related to standard upward compatibility.
    General general options, such as sheet creation, view creation, view axis display, and paper unit.
    Display display options, such as tree, and elements analysis.
    View view generation options, such as geometry generation, dress-up, view generation, clipping view and view from 3D.
    Generation generation options such as update, view, generated geometry, and dimension generation.
    Geometry geometry options, such as grid, geometry creation and manipulation, constraints creation, constraints display, and 2D component.
    Dimension dimension options, such as dimension creation, move and line-up.
    Handles handle options, such as handle reference size and zoomable, paper grid, rotation and dimension handles.
    Annotation and Dress-Up annotation and dress-up options, such as annotation creation, table and area fill.
    Administration administration options, such as drawing management, page setup directory, style, generative view style, and annotation.
    DXF2D general, import and export options for DXF.

  4. Set these options according to your needs.

  5. Click OK to save the options and exit the Preferences dialog box.

    Important:
    • In Me > Preferences> General > File-based Design Import, the Drafting tab offers options for file-based design import related to the Drawings app. For more information, see Native App's User Preferences User's Guide: Display: File-based Design Import: Drafting.
    • In Me > Preferences > General > Compatibility, the V4/V6 DRAW tab offers options for the migration of V4 2D DRAW elements. For more information, see V4 / V6 DRAW
