Performing an Advanced Search

You can perform an advanced search to look for objects based on the definition of multiple criteria. This task shows you how to use the advanced search command in the Drawingsapp.

For more information about advanced search, see Selecting Using the Find... Command in Advanced Mode.

Before you begin: Open a drawing representation containing annotations.

Finding and Replacing Text
