Modifying a Balloon

You can modify a created balloon.

Before you begin: Open a representation (drawing or a layout). Create a balloon.

  1. Right-click the balloon you want to modify and select Properties.

  2. In the Properties dialog box, click the Text tab.

  3. Select the balloon frame properties in the Frame list. For this scenario, select the fixed-size Circle .

    Important: By default, balloons are assigned a variable-size circle which adapts to the balloon text length. However, you can:
    • Select None to display the balloon without a frame.
    • Select Circle to assign a fixed-size frame to the balloon.

  4. Click OK to validate.

    The balloon size is modified.

  5. Double-click the balloon.

    An immersive dialog box appears.

    Important: The Autofit check box is active when the size of the balloon frame is fixed.

  6. Modify the balloon text.

    Notes:

    • You can add some string or remove the attribute link. The balloon text is immediately updated according to the text editor content modification.
    • You can also modify the anchor point and thereby the position of the balloon.
    • You can also isolate a balloon when it contains an attribute link, meaning that the text of the balloon does not reflect anymore the change of the reported item.

  7. Select the Autofit option to adapt the size of the text to that of the balloon frame.

  8. Click OK.

    The text is enlarged to fit within the balloon frame.

    Important: In the case of large texts, the Autofit option reduces the text size.
