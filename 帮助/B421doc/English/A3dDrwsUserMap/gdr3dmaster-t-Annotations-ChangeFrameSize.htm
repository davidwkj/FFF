Changing Frame Sizes for Fixed Frames

You can have multiple fixed size frames for annotations in the same . You can change the size of the frames and manage several fixed sized frames using standard files. Changing the size of the frame in the standard file changes it in the entire drawing.

Before you begin: You must be an owner to perform this task. Select Me > Preferences > Standards to open the Standard Definition dialog box.

  1. In the Category list, select drafting and in the File list, select the required standard .xml .

  2. Under the standards, select Frame.

  3. In the List Of Frames:

    1. Select any of the frames and change its parameters in the boxes that appear below the list.
    2. Select the required type of the frame and click .

      The same frame is duplicated in the list. The new frame is automatically named in a sequential order. You can now edit this frame by changing its defining parameters and its name.

      You can add as many frames as required.

    3. Select the frame to be removed and click .

      The selected frame is removed.

      You have to retain at least one frame of each type. Removing an already referenced frame from the standard, switches the frame to the first defined frame.

The frame sizes are changed. You can now use these frames in the .

To change the frame in the work area, you have the following methods:

  • Select the respective frame under the Text section in the Object Properties panel.
  • Right-click the annotation and select Properties. In the Text tab, under Frame, select the required frame.
