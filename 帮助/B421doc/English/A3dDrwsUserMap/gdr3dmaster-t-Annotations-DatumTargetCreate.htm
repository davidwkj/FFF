Working with Datum Targets

You can create a simple datum target or a movable datum target. You can also convert a non-movable datum target into a movable one.

The can be set either before or after the datum target is created.

This task shows you how to:

  • Create a Datum Target
  • Convert a Non-movable Datum Target into a Movable Datum Target
  • Create an Offset between the Anchor Point and the First Point of the Leader
  • Position a Datum Target on an Existing Annotation

Before you begin: Open a drawing representation with geometrical elements.

Create a Datum Target

You can create a datum target on a projection view.

  1. From the Annotation section of the action bar, click Datum Target .

  2. Select an attachment point for the datum target leader.

  3. Select a location for positioning the datum target (anchor point).

    Note: If the value selected in the Associated Datum box changes, it is instantly reflected in the datum target.

    Tip: You can click Reset to reset the Area Size box to an empty string, the to A1 and the Area Size diameter symbol to the datum target style value.

  4. Enter the required values in the Area Size and Associated Datum boxes.

    By default, A1 is the value in the Associated Datum box.

  5. Select the required area shape icon.

    For example, select the Circular area of contact symbol, if you specify the circle as a datum target.

  6. Click in the dialog box or click in the sheet to validate.

    The datum target is created.

Convert a Non-movable Datum Target into a Movable Datum Target

You can convert a non-movable datum target into a movable one and vice-versa.

A movable datum target indicates that the datum target is not fixed at its basic location and is free to translate.

  1. Select the datum target.

    All the handles of the datum target are displayed.



  2. Select the red handle.

    The datum target is instantly changed to a movable datum target. The handle is oriented to the inside of the movable datum, indicating that it is movable. Slide text handle is also displayed. You can use the text handle to change the position of e datum target.



    Note: The red movable handle is added on the left or right of the datum depending on the position of the first leader with respect to the datum. For more information about handles on movable datum target, see About Working with Datum Targets.

Create an Offset between the Anchor Point and the First Point of the Leader

You can create an offset between the anchor point and the first point of the leader.

  1. Select the datum target.

    The manipulation handles appear.

  2. Drag to the required length.

    You can also drag the white handle to create an offset.

    The line is added or the offset is created between the anchor point and the first point of the leader.

Position a Datum Target on an Existing Annotation

You can position a datum target on an existing annotation on a specific point at its anchor position. You can also snap datum target position to any predefined place.

Before you begin: Create a datum target.

  1. Click Datum Target .

  2. Click in the sheet to define the leader anchor point.

    The datum target leader is placed at the indicated location.

  3. Move the pointer over the reference annotation (in this case, the reference datum target).

    The attachment points are displayed. The nearest attachment point is highlighted to show that the annotation will be positioned on it if that point is selected.

  4. Select the required attachment point to position the datum target.

    The leader is removed and the datum target is positioned on the selected attachment point.

  5. Enter the required values in the Area Size and Associated Datum boxes.

  6. Click in the dialog box or click in the sheet to validate.

    The datum target is created. If required, you can change the position of the datum target.



  7. Optional: Drag the datum target to change it's position.

    The datum target is snapped to the selected position and the attachment point is modified.
