Modifying a Datum Target

You can modify a datum target by editing it.

Before you begin: Open a drawing representation. Create a datum target.
Related Topics
Working with Tolerances

  1. Double-click the datum target you want to modify.



    Note: During datum target modification, the dialog box is initialized with datum target values.

  2. Modify any of the datum target values. For example, enter B instead of A.

  3. Click in the dialog box or click in the empty space to validate.

    The datum target is modified.
