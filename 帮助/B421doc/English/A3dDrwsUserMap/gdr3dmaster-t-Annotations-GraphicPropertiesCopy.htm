Copying Graphic Properties

You can copy the graphic properties of a text element to existing texts. This is true for any type of Drawings element. For our scenario, we will take free text as an example.

Before you begin: Open a drawing representation and create a text.

  1. Select one or more free texts to be modified graphically.



  2. In the Object Properties panel, under Graphic Properties, click Copy Object Format .

  3. Select the text to be used as a graphical reference for selected texts.

    The graphical properties assigned to the text used as a reference are now copied onto the selected free texts.
