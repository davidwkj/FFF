Adding a Leader to an Existing Annotation

You can add a leader to an annotation that was previously created. You can position the leaders freely, or using snapping (the leader is oriented perpendicular to the reference element).

For this scenario, you will add a leader to an existing text, however, this functionality is available with other annotation types as well.

Before you begin:
  • Open a drawing representation.
  • Select Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Handles tab and select the Snap by default (SHIFT toggles) check box.
  • Create a hexagon and create an annotation, a for example.

About Creating a Text With a Leader

  1. Right-click the annotation to which you want to add a leader and select Add Leader.

    The leader is created following the mouse pointer.

  2. Perform either of the following operations:

    • Move the leader so that its orientation is snapped on any of the privileged direction.
    • Position the leader freely: Click where you want to position the leader head.

      You can now move it to the desired location using the pointer. You can position the leader breakpoint anywhere on the reference element.

    • Orient the leader perpendicular to the reference element: Press Ctrl while clicking where you want to position the leader head.

      The leader is created. It is snapped, and oriented perpendicular to the element to which it is attached.

      Release Ctrl and the pointer.
