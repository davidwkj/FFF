About Querying Annotation Links

Querying Annotation Links

You can query annotation links and find out the objects linked to an annotation. You can also query links contained in a text template or a table template.

  1. Right-click the text "Front view Scale: 1:1" and select Query Object Links .

    The Query Link dialog box displays the name and specifications of the linked objects. In our example, the view name and scale are linked to the front view.

  2. Click Close to close the dialog box.

Zoom on the linked object

You can have a close look at the object linked to an annotation.

  1. Right-click the text "Note 1: Circle..." and select Query Object Links .

  2. In the Query Link dialog box, select the Re-frame the window check box and then select the linked object you want to zoom (in this specific case, you can only select Circle.2).

    The app zooms on the object that is linked to the text, i.e. the circle.

  3. Click Close to close the Query Link dialog box.
