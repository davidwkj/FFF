Working with Multiple Bills of Materials

You can create a multiple bill of materials (BOM) table using the Multiple BOM command. The bill of materials that are created are associative and also consistent with the product structure.

This task shows you how to:

  • Create Multiple Bills of Materials
  • Position Multiple Bills of Materials

Before you begin:
  • Open an assembly product containing several sub-products.

    Important: You must keep the product tab open till you complete creating multiple BOM tables.

  • Generate a drawing of the assembly.

About Creating Multiple Bills of Material Tables

Create Multiple Bills of Materials

  1. From the Annotation section of the action bar, click Multiple BOM .

    The Multiple Bill of Material dialog box appears displaying No Selection in the Root Context box.

  2. Select the product in the 3D area.

    The Multiple Bill of Material dialog box displays the name of the selected product in the Root Context box.

    Notes:

    • The Root Context box automatically displays the name of the root product if all the views in the current drawing point to the same root product.
    • Alternatively, you can select a particular view of the current drawing that is pointing to the root product. In this case, the Root Context box automatically displays the name of the root product.

    The Configuration box displays All Level+Summary and the root product gets listed in the Selection List.

  3. Select the required configuration from the list.

    If you change the configuration to User Selection, you can add the required sub-assemblies in the selection list to create the bill of materials table.

  4. Select the required style in the Top Level BOM and Summarized BOM boxes.

    By default, the Default style is selected.

  5. Change the type of BOM for the selected product. For example, click T to make it unavailable.

    T denotes the top level BOM and S denotes the summarized BOM. You can change the type of BOM by selecting or clearing appropriate attributes.

    Note: You cannot make both the attributes T and S unavailable for single product available in the selection list. If you try to do so, an appropriate message appears in the upper right corner of the screen.

  6. Optional: Click to remove the product from the Selection List.

    The product is removed from the list. You can redefine the root context by following the steps from 2 through 4.

    If you remove a product from the list, or if there is only one product in the list and that is removed, then the Configuration option automatically changes to User Selection and you can add the product again by clicking Add.

  7. Click Add.

    As soon as you click Add, the drawing document is tiled vertically along with the root product.

  8. Select the required product(s) from the work area.

    The selected products are added in the Selection List . By default, for each added product, both the BOM attributes T and S are selected.

  9. Click in the Multiple Bill of Material dialog box.

  10. Click anywhere in the sheet to place the bill of materials.

    Multiple bill of materials are created.



Position Multiple Bills of Materials

You can position the bills of materials accurately by selecting a point (used as an anchor point), a line, a table, or a frame and title block as a reference.

Before you begin: Create a line in the active view.

  1. Follow the steps 1 through 9 to create a set of bills of material. See, Creating Multiple Bills of Materials.

  2. Select the line you created.



    The reference line is highlighted and the BOM is displayed along with the red orientation handles and a context toolbar showing the selected mode of BOM creation.

  3. Optional: Select the red handle to invert the orientation of the BOM in the corresponding direction.

  4. Optional: Use the context toolbar commands to change the creation mode of the BOM.

    By default, it displays Create a table aligned on nearest extremity mode. That is, it creates the BOM aligned on the nearest extremity of the selected line. For more information about the table creation modes, see About Creating a Table.

    Important:
    • If the size of the reference is smaller than the size of the created BOM, then, only two table creation modes are displayed, aligned on reference and aligned on the nearest extremity.
    • When the reference size is bigger than that of the created BOM, Creates a table constrained on reference mode is displayed. In this case, the BOM is automatically resized (either by adjusting columns' height or width) to match the selected reference geometry. This mode is unavailable when multiple tables are generated with the Multiple BOM command. If there is a single table created using Multiple BOM, this mode is available.

  5. Optional: Hover over the context toolbar and drag and place it to the required position.

    The pointer changes to that lets you change the position of the context toolbar as required.

  6. Click in the sheet to validate the BOM creation.
