Modifying Annotation Positioning

You can assign new position to existing annotations. You can also modify the position of the views using the Positioning dialog box.

Before you begin: Open a drawing representation containing annotations.
Related Topics
About Annotation Positioning

  1. Multi-select the annotations to be newly positioned. In this example, multi-select text.



  2. From the Annotation section of the action bar, click Element Positioning .

  3. In the Positioning dialog box, under Align, select Align to top .

    For more information, see More About Annotation Positioning.



  4. Under Space, select Space from left to right and enter the Space value as 30mm.



    Important: Note that when you select Space, the modification does not appear similarly on the drawing. This modification only appears when you enter the new Space value in the Positioning dialog box or when you select a Space value.

  5. Under Distribute, select Distribute horizontally .



  6. Under Move, select Move vertically to top and enter the Move value as -10mm.



    Important: Note that when you select Move, the modification does not appear similarly on the drawing. This is only the case once you enter the new Move value in the Positioning dialog box or when you select a spacing option.
