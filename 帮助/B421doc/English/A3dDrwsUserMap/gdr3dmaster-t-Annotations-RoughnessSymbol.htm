Creating a Roughness Symbol

You can create a standard roughness symbol including roughness value, direction of lay, production method, and so on.

Before you begin: Open a drawing representation.
Related Topics
About Roughness Symbol

  1. From the Annotation section of the action bar, click Roughness Symbol .

  2. Select an attachment point of the roughness symbol.

    The roughness symbol position and orientation will be associative to this point.



  3. In the Roughness Symbol dialog box, enter the required values in the various boxes.

  4. Click OK.

    The roughness symbol is created.

  5. Optional: Drag the roughness symbol to modify its position.

    Note: An extension line may be displayed between the roughness symbol and the element to which it is attached (providing this element is linear), depending on where you drag the roughness symbol.
