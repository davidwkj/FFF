Creating an Associated Text

You can create a text that will be associated to an existing element.

You can set text properties either before or after you create the text.

Before you begin: Open a representation and create two diameter dimensions.

About Making an Existing Annotation Associative
Creating Texts

  1. From the Annotation section of the action bar, click Text .

  2. Select the element to which you want to associate the text.

    For our scenario, select the dimension line. In this particular case, you can associate the text to the dimension line by clicking it.

  3. Enter the text to be created in the Text Editor dialog box or directly in the frame. For example, enter "diameter".

    The text is created below the dimension value and associative position and orientation links are created automatically. The text is associated with the position and the orientation of selected sub-part of the dimension.



  4. Click anywhere in the sheet to end the text creation.

  5. Optional: Select the dimension line and drag as required.

    The text remains associated with the position and the selected sub-part of the dimension.



    Important:
    • If you select the dimension value, the text is created below the dimension value with the Anchor Point positioned at the Top Center, and the Anchor Mode at Global, by default.

      When you select a dimension line, the text is created below the dimension line with a 1 mm offset.



    • If you select the dimension line, the text is created below the dimension line, with the Anchor Point positioned at the Top Center, and the Anchor Mode at Global, by default.

      When you select a dimension value, the text is created below the dimension value with an offset equal to "Value > Vertical Offset" + 1 mm. If the "Value > Vertical Offset" is superior to 0; by default the offset is 1 mm.



    • If you select one of the extension lines, the text is created at the middle of the length of the extension line, with the Anchor Point positioned at the Top Center and the Anchor Mode at Global, by default.

      However, note that the text is positioned according to the length of the extension line during the text creation. It may not remain at the middle of the extension line if its length changes.



      In all the above cases, a rigid associative positional link is created which prevents unintended move of the text.
