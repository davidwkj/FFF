Finding and Replacing Text

You can locate a string of characters and then replace it.

Strings can be found and replaced in the following elements:

  • Balloons
  • Datum features
  • Datum targets
  • Dimensions
  • Texts
  • View texts
  • Table cells

Before you begin: Open a representation containing text.

Performing an Advanced Search

  1. In the Search Options list, select Find and Replace Text.

    Tip: To directly access the Find & Replace Text dialog box, press Ctrl+H.

  2. In theFind & Replace Text dialog box, enter the text you want to find.

  3. Click Find Next.

    Important: The following message appears in the dialog box: Searching within current sheet. If you selected several elements in the current sheet, then the text to be found is searched only in annotations present in those selected elements and the following message will appear: Searching within selected annotations. Elements other than supported annotations are ignored.

  4. If needed, click Find Next again to search for other instances.



  5. Enter the text you want to use as a replacement text and click Replace.

    For example, enter Second as the Replace with what? text.

    To replace all instances of the text, click Replace all.

    You can also match case, find whole words only or re-frame the window.

  6. Click Close when done.



    Note: It is impossible to replace a string in an attribute link. Also, you cannot replace a string in a locked annotation.
