Creating Constraints via SmartPick

You can detect, create and visualize constraints using SmartPick. In this scenario, you are going to create two constrained parallel lines.

Before you begin: From the View section of the action bar, select Show Constraints (to visualize detected and created constraints) and from the Sketch section of the action bar, select Create Detected Constraints .
