Creating a Frame and Title Block Template

You can design and create a frame, title block, and revision table as per your requirement and save the format as a template or use a default template (script) to create the template to be instantiated in a representation.

Before you begin: Open or create a representation.
Related Topics
Creating a Customized Frame and Title Block

  1. Right-click the sheet and select Edit Background.

    The sheet background is in editing mode.

  2. Create the desired sheet frame.

    You can or design a different sheet frame.

  3. From the Annotation section of the action bar, click Working Views.

    You are back to the working views and you can view the sheet frame in the background.

  4. In the detail sheet, create a table or a table template to be used as a title block.

  5. Right-click the table and select Properties > Feature Properties.

  6. Specify the name as TitleBlock_1 and click Apply and then click OK.

    Important: You must specify the name of the title block with the prefix TitleBlock_.

  7. Create a table again to be used as a revision table.

  8. Right-click the table and select Properties > Feature Properties.

  9. Specify the name as RevisionTable_1 and click Apply and then click OK.

    Important: You must specify the name of the revision table with the prefix RevisionTable_.

  10. In the top bar, select Share > Save > Save with Options.

    Your representation is saved along with the sheet frame, title block and revision table. You can now instantiate this template.
