Creating/Modifying Angle Dimensions

You can create angle dimensions and perform the following kinds of modifications on such dimensions: create new angle sector or turn an angle sector into a supplementary sector.

Before you begin:
  • By default, Angle dimension is not available in any of the action bars. To make it available, you need to customize the action bar. For more information, see Customizing an Action Bar by Adding Commands.
  • Open a drawing representation with angles to dimension.

  1. From the Annotation section of the action bar click Dimensions .

  2. Select both lines to be dimensioned, one after the other.

    The angle dimension appears in the sector associated to both selected lines.



  3. Drag the angle dimension line to the desired quadrant (or sector).



  4. To move the dimension to a new sector, perform either of the following:

    • Right-click the angle dimension and select Angle Sector. Then either select a given angle sector or the Complementary angle sector.

    • Hold down Ctrl and click the dimension line.



  5. Click anywhere to create the angle dimension.

    Tip: To edit the angle sector of an existing angle dimension, right-click the angle dimension and select Dimension_name object > Angle Sector.

  6. From the customized section of the action bar, click Angle Dimension .

  7. Select two lines in the rectangle to create an angle dimension.



  8. Click to create the angle dimension, then right-click the dimension and select Half Dimension.

  9. Click in the sheet to create the dimension.



    Note: The orientation of the half angle dimension depends on the order in which you select the lines to be dimensioned. If you need to modify the dimension, use Re-routing Dimensions .
