Creating Chamfer Dimensions

You can create a chamfer dimension.

This task shows you how to:

  • Create Chamfer Dimensions Manually
  • Create Chamfer Dimensions Using Chamfer Detection

Before you begin: In a drawing representation, create three lines. For more information, see Sketcher User's Guide: Sketching Simple Profiles: Creating Lines.

About Chamfer Dimensions

Create Chamfer Dimensions Manually

You can create chamfer dimensions manually.

  1. Go to Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Dimension section, clear the Detect chamfer check box.

  2. From the Annotation section of the action bar, click Chamfer Dimensions .

    In the Tools Palette, choose from the following:

    • The format of the dimension:
      • Length x Length
      • Length x Angle
      • Angle x Length
      • Length
    • The representation mode:
      • One symbol
      • Two symbols

    Tip: You can also access these options using the context menu at any time during the chamfer dimension creation.

  3. Select the Length x Length format and the One symbol mode .

  4. Select the element to be dimensioned.

  5. Select a reference line or surface.

  6. Perform any one of the following:

    • Click in the sheet to create the dimension.

      The chamfer dimension is computed with an implicit second reference line that is perpendicular to the first one.



    • Select a second reference line or surface.

      The chamfer dimension is computed according to both the selected reference lines.



    Important: In a Drawings context (i.e. in the case of a generative view), you must explicitly select the second reference line.

    In any case, the dimension is associated with all the selected elements.

Create Chamfer Dimensions Using Chamfer Detection

You can create chamfer dimensions using chamfer detection.

Note that chamfer detection is provided as a help in selecting chamfers. However, depending on the geometrical configuration, it may not detect all chamfer types. If your chamfer is not detected, you can still create the chamfer dimension manually as explained below.

  1. Go to Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems Drafting > Dimension section, select the Detect chamfer check box.

  2. Click Chamfer Dimensions .

    In the Tools Palette, choose the format of the dimension and the representation mode. For more information, see step 3 in Creating chamfer dimensions manually.

  3. Select the Length x Length format and the One symbol mode .

  4. Hover over the element to be dimensioned.

    Note that, depending on the position of the pointer, the auto-detection indicates a different order for taking reference elements into account when creating the chamfer dimension:

    • 1 indicates the element to be dimensioned.
    • 2 indicates the line used as the first reference.
    • 3 indicates the line used as the second reference.

  5. Click when you are satisfied with the order shown by the auto-detection agent. For example, click to accept the 3 - 1 - 2 order.

    The chamfer dimension is computed according to the first and the second auto-detected reference lines.

    Important: At this stage, if you are not satisfied with the order you just accepted, or if your chamfer is not detected, you can still click to select the first reference line, and, optionally, the second reference line. This amounts to creating the chamfer dimension manually.

  6. Click in the sheet to position the chamfer dimension.



    The dimension is associated to all the auto-detected elements.
