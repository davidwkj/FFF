Creating a Datum Feature

You can create a datum feature.

Before you begin: Open a drawing representation.
Related Topics

  1. From the Annotation section of the action bar, click Datum Feature .

  2. Select the attachment point of the datum feature.



  3. Select a point to be used to position the datum feature (anchor point).

    An immersive dialog box appears with the value A pre-selected in the Datum Feature box and simultaneously previewed on the .

    Important: During datum feature creation, all the existing datum features of a drawing or a layout are parsed to retrieve the highest letter of all the letters used. The value of the datum feature is initialized to value of the next letter. For example, if a drawing contains datum features such as, A, B, D, then the next datum feature is initialized to E.

    Note: If the value selected in the Datum Feature box changes, it is instantly reflected in the datum feature.

  4. Optional: In the immersive dialog box, change the value.



  5. Click or click in the sheet to validate the inputs provided in the dialog box.

    The datum feature is created, and an extension line is automatically created on the datum feature.
