Modifying a Datum Feature

You can modify a datum feature by editing it.

Before you begin: Open a drawing representation.

Create a datum feature with a value of A.

Related Topics
Working with Tolerances

  1. Double-click the datum feature you want to modify.

  2. In the immersve dialog box, modify the datum feature value. For example, enter B instead of A.

  3. Click or click in the sheet to validate.

    The datum feature is modified.

  4. Optional: Drag the datum feature to move it.



    Important: Depending on the type of element to which the datum feature is attached, you may not be able to move the datum feature as wanted. For example, if the datum feature is attached to a dimension line, you can only move the datum feature along the dimension line direction.
