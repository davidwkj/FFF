Modifying the Dimension Overrun/Blanking

You can modify dimensions extension line overrun and/or blanking either together or separately.

Before you begin:
  • In a drawing representation, create a distance dimension, for example.
  • Select Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Handles tab, under Dimension Handles, select the Modification check boxes corresponding to the Modify overrun and the Modify blanking.
Related Topics
About Dimension Overrun/Blanking

  1. Drag the overrun handle(s) to a new position.





    Tip: To modify one extension line only, press Ctrl and drag the desired handle.

  2. Drag the blanking handle(s) to a new position.





  3. If you need to be more precise, double-click the handle.

  4. In the Blanking Edition dialog box, enter the required value to modify the blanking.

    Important: You can also modify the overrun/blanking of only one extension line of the dimension.

  5. Double-click the overrun handle(s).



  6. In the Overrun Edition dialog box, enter the required overrun value and clear the Apply to both sides check box.

    The overrun is applied to one side only.
