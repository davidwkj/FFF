Specifying the Dimension Value Position

You can automatically position a dimension's value outside the area between extremity symbols when this area is too small, or you can explicitly position the value inside or outside the area between extremity symbols.

Related Topics
Creating Dimensions

  1. Right-click the dimension to be modified and select Properties.



  2. In the Value tab, underValue Orientation, in the Position list, the following options are available:

    • Auto: Positions the value inside the area between extremity symbols whenever this is possible; otherwise, positions it outside.

      Note: This option will be unavailable if you modify the position of the dimension value text using the pointer. You can restore the original position of the dimension value by right-clicking the dimension and selecting Restore Value Position from the context menu. In this case, note that Text Before and Text After the dimension value is not taken into account.

    • Inside: Positions the value inside the area between extremity symbols.

    • Outside: Positions the value outside the area between extremity symbols.

  3. Select Auto and click OK.

    If you change the dimension from now on, and the value does not fit inside the area between extremity symbols, the value is automatically positioned outside.



    Tip: If you switch between Auto, Inside, and Outside, you can use the Restore Value Position from the context menu to make sure that the dimension value is properly positioned.
