Modifying the Dimension Value Text Position

You can modify the position of the dimension value text using the pointer.

Related Topics
Creating Dimensions

  1. Select the dimension value text.



  2. Drag the value text to the new position.

  3. Click to validate the position.

    Tip: Note that as a useful help, you can press Shift use Snapping temporarily (as long as you keep the button pressed).



    At any time, you can restore the original value text position. To do this, right-click the dimension you positioned and select Restore Value Position.

    Important: When using the Restore Value Position option from the context menu, note that Text Before and Text After the dimension value are not taken into account.
