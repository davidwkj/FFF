Interrupting Extension Lines

You can interrupt manually one or more extension lines of one or more dimensions.

Before you begin: .
Related Topics
About Interrupting Extension Lines

  1. Right-click a dimension and select Dimension.x Object > Create Interruption(s).

    Important:
    • You can also select the interruption command first, and then the dimension.
    • You can multi-select several dimensions either using Ctrl or the pointer.

  2. In the Tools Palette, select one of the following:

    • Interrupt one extension line : Creates the interruption on one extension line.
    • Interrupt both extension lines : Creates the interruption on both extension lines.

  3. Click to indicate the first point defining the interruption to be created.

  4. Click to indicate the second point defining the interruption to be created.

    Important: If you have chosen to create the interruption on one extension line, the interruption is automatically created on the extension line which is closest to where you click.

  5. To remove the interruption you created, right-click the dimension and select Dimension.x Object > Remove Interruption(s).

  6. In the Tools Palette, select one of the following:

    • Remove one interruption : Removes a single interruption on an extension line.
    • Remove all interruptions : Removes all interruptions on an extension line.
    • Remove all interruptions : Removes all interruptions on both extension lines.

      In this case, select the Remove One Interruption.

  7. Click to indicate the extension line from which the remove the interruption is to be removed.

    The interruption is removed from the extension line which is closest to where you click.
