Copying Geometrical Tolerances

You can copy an existing geometrical tolerance and edit the content to create a new one.

Before you begin: Open a drawing representation.

  1. Click the geometrical tolerance you want to copy.



  2. Right-click and select Copy.

  3. Select the element to which you want the geometrical tolerance to be associated with.

  4. Right-click and select Paste.

  5. Move the copied geometrical tolerance to position it as desired.

  6. Double-click the copied geometrical tolerance.

  7. In the Geometrical Tolerance dialog box, modify the required values. For more information, see Creating a Geometrical Tolerance.

  8. After you are done entering values in a given box, press Tab to move to the next box.

    The geometrical tolerance is updated as you define values for each box.

  9. Click OK or click anywhere in the drawing to validate.
