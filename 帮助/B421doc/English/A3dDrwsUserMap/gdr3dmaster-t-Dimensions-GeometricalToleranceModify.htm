Modify Geometrical Tolerances

This task shows you how to modify a geometrical tolerance.

  1. Double-click the geometrical tolerance you want to modify.



    The Geometrical Tolerance dialog box appears, with the existing values pre-entered.

    Important: You can reset the current values in the Geometrical Tolerance dialog box by clicking Reset.

  2. Modify the required values. See Creating a Geometrical Tolerance.

  3. Click OK or click in the sheet to validate the geometrical tolerance modification.



  4. , and then create a geometrical tolerance on it.



  5. Click the extension line and move the dimension.

    You can see that the geometrical tolerance follows the dimension.



  6. Click the geometrical tolerance and move it.

    You can see that it has no impact on the position of the dimension.

    Important: The behavior is the same if a geometrical tolerance is created on the dimension value.

Create Associativity Between the Dimension and the Geometrical Tolerance

You can create associativity between the dimension and the geometrical tolerance.

  1. , and then create a geometrical tolerance on it, by selecting the dimension line.

  2. Move the dimension.

    Notes:

    • The positioning and the length of the tolerance leader remain constant when moving the dimension, whatever the dimension type.



    • The length of the tolerance leader is recomputed as long as the dimension is moved. This is because the ratio between the lengths of the leader's projection and the extension line's projection remains constant.



    • The positioning of the leader is also recomputed so that the distance between the anchor point of the tolerance and the extension line remains constant.



Important: When a tolerance is created on an angle dimension, the position and the length of the leader remain constant if the dimension is moved.
