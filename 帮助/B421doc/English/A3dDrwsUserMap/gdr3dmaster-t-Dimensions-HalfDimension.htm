Creating Half Dimensions

You can create half dimensions on distances, angles, diameter cylinders, radius cylinders that are created using two selections, diameter edges and diameter tangents. However, you cannot create half dimensions on cumulated dimensions.

About Creating Half Dimensions

  1. From the Annotation section of the action bar, click Dimensions .

  2. Select the first element in the view. For example, an edge.

  3. Select the second element in the view. For example, an axis.

    A half dimension is created and only one extension line is displayed. The dimension line is shortened with a specific overrun, gap, and length. The value is not aligned with the center of the dimension line. The attributes mentioned in the Dimension Parameters determine the graphic display of the dimension.

    Note: When you create a dimension from an axis to a geometry or from a geometry to an axis using interactive commands, a half dimension is created by default when no user preference is available.



  4. Right-click the required dimension and select Half Dimension.

    The half dimension changes to a standard dimension.



    This selected dimension type is stored as a preference and used as the default value until a half dimension is selected again.
