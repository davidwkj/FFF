Creating a Minimum Distance Dimension

You can create a dimension measuring the absolute minimum distance between two curves, or between a point and a curve.

Related Topics
About Minimum Distance Dimensions

  1. From the Annotation section of the action bar, click Dimensions .

    Tip: You can also use Length/Distance dimension to define a Minimum distance dimension type.

  2. In the Tools Palette, select Force dimension on element from the Tools Palette.

  3. Select the two elements (curves, or point and curve) between which you want to measure the minimum distance.

  4. Right-click the dimension and select Minimum Distance.

  5. Position the minimum distance dimension with the pointer, then click to validate its creation.



    The minimum distance dimension is created.
