Creating Partial Curvilinear Length Dimensions

Partial curvilinear length dimensions are defined using points. You can create dimensions for the curvilinear length of a curve portion, i.e. to measure the partial length of a curve.

This task shows you how to:

  • Create partial curvilinear length dimensions using existing points
  • Create partial curvilinear length dimensions using points created on-the-fly

Before you begin:

In a drawing representation, create two points on a curve. These points will be used to define the extremities of the curve portion to dimension.

Related Topics
About Creating Partial Curvilinear Length Dimensions

Important: You can also use spline control points, or points created in free space. In the case of points in free space, the partial curvilinear length dimension is computed according to the normal projection of these points on the curve. So, when creating such points, you need to make sure that they are projected on the curve, as shown below for example.

Create partial curvilinear length dimensions using existing points

Partial curvilinear length dimensions are created using the existing points on the curve.

  1. From the Annotation section of the action bar, click Dimension .

  2. Select the curve on which you created the points.

    A preview of the dimension is displayed. By default, this preview shows an overall curve dimension.

  3. Right-click and select Partial Curvilinear Length.

  4. Right-click again, and select a representation mode for the dimension line:

    • Offset displays the dimension line as an offset of the measured curve.

    • Parallel displays the dimension line as a translation of the measured curve.

    • Linear displays the dimension line as linear.

    For this scenario, select Parallel.

  5. Select the existing point on the curve that defines the first extremity of the curve portion to dimension.



  6. Select the point that defines the second extremity of the curve portion to dimension.



  7. Position the dimension line and click in the drawing to validate the dimension creation.

    The semi-arc symbol displayed over the dimension value symbolizes a curvilinear length dimension (whether partial or not). You can now handle the dimension just like any other dimension.

  8. Move one or both points, on the line or in the free space.

    The dimension is recomputed (if you moved the point in the free space, it is recomputed according to the normal projection of the points on the curve).

    Important: If you move a point in such a way that it cannot be projected on the curve anymore, the dimension needs an update.

Create partial curvilinear length dimensions using points created on-the-fly

Partial curvilinear length dimension is created using the extremities of the curve.

  1. Click Dimension .

  2. Select the curve.

    A preview of the dimension is displayed.

  3. Right-click and make sure Partial Curvilinear Length is selected.

  4. Right-click again, and select a representation mode for the dimension line.

    For this scenario, select Offset.

  5. Select the point on the curve that defines the first extremity of the curve portion to dimension.

    You can click the curve, or in the sheet. Note that the indicated point cannot go further than the extremity of the curve itself.

  6. Select the point that defines the second extremity of the curve portion to dimension.



    Note that two points, as well as two coincidence constraints, have been created on the curve, at the projection point of where you clicked.

  7. Position the dimension line and click in the drawing to validate the dimension creation.

    The semi-arc symbol displayed over the dimension value symbolizes a curvilinear length dimension (whether partial or not). You can now handle the dimension just like any other dimension.
