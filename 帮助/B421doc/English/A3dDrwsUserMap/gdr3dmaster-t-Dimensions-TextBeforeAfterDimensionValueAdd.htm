Adding Text Before/After the Dimension Value

You can insert text before or after the dimension value.

Before you begin:
  • In a drawing representation, create a distance dimension, for example.
  • Select Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Handles section, and select the Modification check box for the Insert text before and the Insert text after.

  1. Click the dimension to be modified.

    The dimension is highlighted and two handles appear, both before and after the dimension value.

  2. Click the handle before the dimension value, for example.



  3. In the Insert Text Before dialog box, enter the text that you want to add before the dimension value, L= for example.

  4. Click OK.

    The text is automatically inserted before the dimension value.

    Important: Note that the last five texts created before or after the dimension value are automatically added to the drop-down list in the dialog box and can therefore be selected again from this list.

  5. Click in the sheet to validate.

    Important: when using the Restore Value Position option from the context menu, note that Text Before and Text After the dimension value are not taken into account.

  6. You can repeat steps 2 to 5 to add text after the dimension value.
