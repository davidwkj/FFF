Creating Cumulated Dimension Systems

You can create a cumulated dimension system that can comprise distance or angle dimensions.

Related Topics
About Dimension Systems

  1. From the Annotation section of the action bar, click Cumulated Dimensions .

  2. Click a first point on the view.

  3. Click a second point on the view.

    A first dimension is created within the cumulated dimension system.



  4. Click a third point on the view.

    A second cumulated dimension is created in the system. You can create as many cumulated dimensions as desired.



    Notes:

    • During dimension creation, the whole system follows the pointer if Dimension following the cursor check box is selected in Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Dimension section. Therefore, If you move one dimension line as you create a cumulated dimension, all the lines move accordingly.

    • If the cumulated dimensions are set with the value oriented along the dimension line, set the Dimension Parameters in the standards.

  5. Click in the sheet to end the cumulated dimension system creation.
