Lining Up Dimension Systems

You can organize the view dimensions by lining up the dimension system.

Before you begin: Open a drawing representation.
Related Topics
Working with Dimension Systems

  1. Create a cumulated dimension system including several dimensions as shown below.



  2. From the Annotation section of the action bar, click Dimension to create a dimension.



  3. Right-click your dimension system and select Line-up.

  4. Position the pointer on the dimension you just created to align the dimension system on it.



    appears.

    Important: For a dimension system only Offset to reference, Align stacked dimension values and Align cumulated dimension values are taken into account.

  5. In the Line Up dialog box, modify the Offset to reference value to 5mm.

  6. Click OK.

    The dimension system is aligned on the dimension with an offset of 5mm.

    Important: The dimension system is aligned according to its dimension system properties.
