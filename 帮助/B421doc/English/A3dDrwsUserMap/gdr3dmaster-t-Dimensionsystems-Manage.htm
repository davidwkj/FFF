Working with Dimension Systems

You can insert a dimension into a dimension system, remove a dimension from a dimension system or transfer a dimension to another dimension system using dedicated context menu commands.

This task shows you how to:

  • Group Dimensions
  • Regroup Dimensions
  • Ungroup Dimensions

Before you begin: In a drawing representation, create a stacked dimension system including several dimensions.
Related Topics
Modifying a Dimension System

Group Dimensions

You can group two or more dimensions to form a new system if they are compatible.

  1. Create and select several dimensions (distance or angle).



  2. Right-click and choose Selected objects > Group Dimensions.

    The dimensions are grouped together and a new dimension system is formed.

    Notes:

    • If a dimension is already a part of another dimension system, then, it is removed from that system before getting grouped with other dimensions or systems to form a new system.
    • If the existing dimension system is emptied by Group Dimensions, then the system is deleted.

Regroup Dimensions

You can regroup an existing dimension system by accommodating one or more dimensions in it.

Before you begin: Create a stacked dimension system and several other compatible dimensions.

  1. Select the individual dimensions.

  2. Right-click them and choose Selected objects > Regroup Dimensions.

  3. Select the dimension system (stacked, in this scenario) to add the selected dimensions to.

    The dimensions are added to the selected system and the system is aligned automatically taking the new dimensions into account.

    Important:
    • If the dimension system is of cumulative or stacked type, then the dimension to be added must have one of its dimensional elements same as the common element of the system.
    • If the dimension system is of cumulative type, then the dimension to be added must also be of cumulative type.
    • If the dimension system is of chained type, then any one of its dimensions should have one dimensional element same as an element of the dimension to be added.
    • If a dimension is already a part of a system, then, it is removed from that system before getting added to the other system.
    • If the existing dimension system is emptied by Regroup Dimensions, then the system is deleted.

Ungroup Dimensions

You can remove one or more dimensions from a dimension system by ungrouping them.

  1. Select the dimension system.

    A group handle is displayed and the whole system is highlighted. A context toolbar appears showing the active Group mode.

  2. Select Elements from the context toolbar to select individual dimensions of the dimension system.

  3. Press Ctrl and select required dimensions from the system.

  4. Right-click the dimensions and choose Selected objects > Ungroup Dimension(s).

    The selected dimensions are removed from the dimension system and the system is aligned taking the rest of the dimensions into account.

  5. Optional: Remove all the dimensions from the dimension system using the following steps:

    1. Select Group from the context toolbar.
    2. Right-click the system and choose Selected objects > Ungroup Dimension(s).

      All the dimensions are removed from the system and the system is deleted.
