Modifying a Dimension System

You can modify a dimension system or a dimension within a dimension system.

In this example we create a stacked dimension system.

This task shows you how to:

  • Move a Dimension System
  • Move a Dimension
  • Align a Dimension System
  • Restore a Value Position
  • Add a Dimension into a System
  • Delete a Dimension from a System
  • Copy a Dimension System

Working with Dimension Systems

Move a Dimension System

You can move and organize the dimension system.

  1. Click over the dimension system to select it.

    A group handle is displayed and the whole system is highlighted. A context toolbar appears showing the active Group mode.

  2. Drag the whole system below the drawing.



  3. Click in thesheet to end the stacked dimension system selection.

Move a Dimension

You can move and organize the dimension lines.

  1. Right-click the Dimension system and select Properties.

  2. In the System tab, set the Offset mode to Free, then click OK.

    For more information on the Properties tab, see System Properties.

  3. Click the dimension system to display the group handle and select Elements mode to allow the selection of individual dimensions of the system.

  4. Select several dimensions and drag them above the dimension system.

  5. Click in the sheet to end the dimension selection.



    Important: You can only move a dimension in a stacked dimension system.

Align a Dimension System

You can align the dimension or a dimension system on to another dimension in the same view.

  1. Click over the dimension system to select it.

  2. Select Group from the context toolbar to select the whole dimension system.

  3. Right-click the dimension system and select Align into System .

    The dimensions are aligned into the system as before.

Important: You can perform an Align into System on a whole system and also on several dimensions after multi-selecting them. To do so, set the system's offset mode to Free (see System Properties), then click Elements to cancel the selection of the whole dimension system. The dimensions you selected are aligned into a system of their own as shown below.

Restore a Value Position

You can restore the value position of the dimension after it is modified.

  1. Select the dimension system.

    Note: Select Group from the context toolbar if the system is not selected.

  2. Right-click the dimension system and select Properties.

  3. In the System tab, under Dimension values alignment, set the Align stacked dimension values at Reference line and Values Offset at 10mm.

  4. Click Apply to visualize the dimension values alignment.



  5. Select Elements from the context toolbar to cancel the selection of the dimension system.

  6. Move some of the dimensions.



  7. Select Group from the context toolbar to activate the dimension system again.

  8. Right-click the dimension system and select Restore Value Position.

    The dimension values are back to their initial position.

  9. Select Elements from the context toolbar to cancel the selection of the dimension system.

  10. Drag several dimension values to modify their position.



  11. Multi-select the dimensions.

  12. Right-click and select Restore Value Position.

    The dimension values are centered back in the system.

Add a Dimension into a System

You can add a dimension into a dimension system, i.e. chained, cumulated, or stacked.

  1. Select the dimension system.

    Note: Select Group from the context toolbar if the system is not selected.

  2. In the System tab, set the Offset mode to Constant, then click OK.

    For more information on the Properties tab, see System Properties.

  3. From the Annotation section of the action bar, click Stacked Dimensions

  4. Select the system you want to insert a dimension to.

  5. Select the geometry to dimension.



    The new dimension is inserted into the system and is automatically aligned into the system.



Delete a Dimension from a System

You can delete a dimension or a dimension system from your view or drawing.

  1. Select the dimension system.

  2. Select Elements from the context toolbar to select a single dimension.

  3. Select a dimension. Note that only the selected dimension is highlighted, not the whole system.



  4. Press Del.

    The dimension system is updated accordingly and automatically aligned.

Important: If you want to delete the whole dimension system, select Group and press Del.

Copy a Dimension System

You can copy a geometry and its dimension system in to another view.

  1. Select the dimension system.

    Note: Select Group from the context toolbar if the system is not selected.

  2. Select the geometry and its associated dimension system and click Copy.

  3. Create new view and select it.

  4. Click Paste.

    The geometry and its dimension system is pasted in the view.

Important: If the dimension system offset mode is set to Free, modifying one or several dimensions of the system will not impact the alignment of the system. If the dimension system offset mode is set to Constant, an automatic line-up is applied to dimension lines and values in case you perform any of the following action:
  • Adding a dimension to a system
  • Deleting a dimension to a system
  • Updating the 3D geometry
  • Applying a style to a system
For more information about the Constant and Free Offset mode, see System Properties.
