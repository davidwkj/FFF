Exporting Data to 3D XML

You can use the Export command to export a product structure into a 3D XML file without sharing the unresolved product structure or the configuration dictionary (Variability Space).

Before you begin: Selected one or more objects in explore, search result or folder editor window.

  1. In the top bar, select Share > Export.

  2. In the Format list, select either with authoring, which allows exchanging this data on another site via import, or for review, which allows you to perform a DMU review on the exported file.

    Note: For more information, see Digital Review User's Guide: Setting Up Your Session: Importing 3D XML Files.

  3. Enter the Path and Name for the exported file.

    You may also browse for the directory using Browse .

  4. Optional: Edit the properties of the 3dxml file by clicking on the <to be edited> text in any row.

  5. Click Preview to see a list of the objects to be exported.

    If one of the selected entities is not supported for export, none of the selections are exported, and the 3D XML will not be generated.

  6. Optional: Under the Options list, select the required options and check boxes.

  7. Click OK to export.
