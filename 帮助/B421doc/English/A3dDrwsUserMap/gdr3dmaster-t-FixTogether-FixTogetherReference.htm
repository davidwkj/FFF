Creating a Group Reference

You can create a detail sheet and then position a group reference on this sheet. This group can then be instantiated in a design sheet.

Before you begin:
  • Open a representation.
  • Create a detail sheet.

  1. Right-click the created detail sheet in the tree and select Set As Current.

    The details sheet is now in editing mode.

  2. Create a group of 2D geometry in the detail sheet.

  3. The Group constraint is created and displayed on the geometry with the clip symbol.



    You can create more than one group references and then create a catalog.
