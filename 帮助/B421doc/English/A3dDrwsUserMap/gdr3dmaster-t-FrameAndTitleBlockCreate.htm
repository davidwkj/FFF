Creating a Default Frame and Title Block

You can insert a default frame and title block template in the background of a sheet.

This operation is performed using a macro. A few macros are provided by default.

Before you begin: Open or create a representation.
Related Topics
About Frames and Title Blocks

  1. If you are working on a regular sheet, from the View Layout section of the action bar, click Page Layout .

  2. If you are working on a detail sheet:

    1. Right-click the detail sheet in the tree, and select Edit Background.
    2. From the View Layout section of the action bar, click Page Layout .

  3. In the Page Layout dialog box, in the Sheet Dress-up list, select the required dress-up.

    For this scenario, choose Drawing_Titleblock_Sample1.

    By default, the Dress-up DS 1 is selected.

    Note: Once you change the value in the list, that value is saved as user preference for the next usage.

  4. Click Frame And Title Block to see the entire list.

    It displays a list of actions to be performed and a preview of the frame and title block. However, note that this preview does not show the current sheet's content.

    • Create: Creates the frame and the title block.

    • Delete : Deletes the frame and the title block.

    • Resize: Resizes and updates the frame, the title block and the working views.

    • Update : Updates the frame and the title block, as well as the fields in the title block (information related to the 3D product and the sheet).

    • Check by: Completes the "Checked by" field and automatically updates the verification date.

    • Add a revision block: Adds a revision block.

    Important: Information which is unavailable in the 3D product is substituted by "XXX" in the .

  5. Select Create in the list.

  6. Click OK.

    The frame and title block is created in the drawing.

    Important: The title block contains associative text (text with attribute links) whenever possible. However, paper size, weight, sheet number and number of sheets are not associative. You need to use the Update action of the Page Layout dialog box to update the content of these fields.

    Tip: When adding sheets, if you want the frame and the title block to be created in new sheets, go to Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > General tab, and select the Copy sheet background from first sheet check box. This will insert the frame and the title block from the sheet you previously created in the new sheet.
