Creating a Customized Frame and Title Block

You can insert a customized frame and title block template that you have created earlier or insert the default frame/title block/revision table template in a sheet. You can also define additional actions or dress-up sets to specify the other required information to be added in a sheet.

Before you begin: Open or create a representation and create a detail sheet.

  1. If you are working on a regular sheet, from the View Layout section of the action bar, click Page Layout .

  2. If you are working on a detail sheet:

    1. Right-click the detail sheet in the tree, and select Edit Background.
    2. From the View Layout section of the action bar, click Page Layout .

  3. In the Page Layout dialog box, in the Sheet Dress-up list, select Dress-up DS1.

  4. Select the required frame template using the following steps:

    1. Click Frame to view the frame templates list.

      It displays a list of actions to be performed and a preview of the frame. However, note that this preview does not show the current sheet's content.

    2. Optional: Click Instantiate a default template .

      The default frame is added to the current sheet.

    3. Click Search for an existing template .

      A message appears at the top prompting you to select or query an object using Search.

    4. From the top bar, in the Search Options list, enter a search string for the representation you saved earlier and click Search .

      A list of all the objects corresponding to your search criteria appears in the Search Results panel. Select the required representation in this list or the representation which is being edited (if any).

    5. Select the required representation.

      Immediately its name appears in the Frame box.

    6. Select Instantiate in the list of actions and click Apply.

      The frame is added to the current sheet.

  5. Select the required title block template using the following steps:

    1. Click the Title Block to view the title block templates list.

      It displays a list of actions to be performed and a preview of the title block. However, note that this preview does not show the current sheet's content.

    2. Optional: Click Instantiate a default template .

      The default title block is added to the current sheet.

    3. Click Search for an existing template .

      A message appears at the top prompting you to select or query an object using Search.

    4. From the top bar, in the Search Options list, enter a search string for the representation you saved earlier and click Search .

      A list of all the objects corresponding to your search criteria appears in the Search Results panel. Select the required representation in this list or the representation which is being edited (if any).

    5. Select the required representation.

      Immediately its name appears in the Title Block box.

    6. Select Instantiate in the list of actions and click Apply.

      The title block is added to the current sheet.

  6. Select the required revision table using the following steps:

    1. Click the Revision Table arrow to view the revision table templates list.

      It displays the list of actions to be performed and a preview of the revision table. However, note that this preview does not show the current sheet's content.

    2. Optional: Click Instantiate a default template .

      The default title block is added to the current sheet.

    3. Click Search for an existing template .

      A message appears at the top prompting you to select or query an object using Search.

    4. From the top bar, in the Search Options list, enter a search string for the representation you saved earlier and click Search .

      A list of all the objects corresponding to your search criteria appears in the Search Results panel. Select the required representation in this list or the representation which is being edited (if any).

    5. Select the required representation.

      Immediately its name appears in the Revision Table box.

    6. Select Instantiate in the list of actions and click Apply.

      The title block is added to the current sheet.

  7. Click OK.

    The frame, title block and the revision table are inserted in the sheet.
