Performing Actions on the Frame and Title Block

You can perform various actions on the frame, title block and revision table such as removing or updating the title block, resizing the frame, and so on.

Before you begin:
  • Go to Me > Preferences > Mechanical > Drafting > General tab and clear the Copy sheet background from first sheet check box if selected already.
  • Create a template containing two sheet frames using the following format:

    Sheet1:

    • Sheet Style: A0 ISO
    • Orientation: Landscape

    Sheet2:

    • Sheet Style: A0 ISO
    • Orientation: Portrait
  • Open a drawing representation and create a frame (Sheet Style: A0 ISO and orientation: Landscape), a title block and a revision table using a template representation.

For this scenario, resize the frame, update the title block and add revision to the revision table.

  1. Resize the sheet frame using the following steps:

    1. From the View Layout section of the action bar, click Page Layout .
    2. In the Page Layout dialog box, under Sheet Style, change the orientation of the sheet from Landscape to Portrait and click Apply.

      The sheet orientation changes but the sheet frame remains as it was before.

    3. In the Frame action list, click Resize.

      The sheet frame is resized with respect to the changed sheet frame. The title block and the revision table are also repositioned as per the resized sheet frame.

  2. Update the title block using the following steps:

    1. Add a new sheet by right-clicking the representation in the tree and selecting New Sheet.

      The new sheet is added.

    2. Activate the first sheet by double-clicking it.
    3. From the View Layout section of the action bar, click Page Layout .
    4. In the Title Block action list, click Update.

      The title block is updated displaying increased sheet number.

  3. Add a new revision to the revision table using the following steps:

    1. Click the Revision Table list.

      It displays a list of actions to be performed and a preview of the revision table. However, note that this preview does not show the current sheet's content.

    2. Select Add Revision and click Apply.

      A new line corresponding to a new revision is added to the existing revision table.

  4. Remove the sheet frame/title block/revision table using the following steps:

    1. Click the Frame/Title Block/ Revision Table list.

      It displays a list of actions to be performed and a preview of the selected category. However, note that this preview does not show the current sheet's content.

    2. Select Remove from the list of actions and click Apply.

      The sheet frame/title block/revision table is removed from the current sheet.

  5. Click OK.
