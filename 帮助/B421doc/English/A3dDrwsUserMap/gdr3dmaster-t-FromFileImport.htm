Inserting From Files

You can insert files of certain types into a representation.

This task shows you how to:

  • Insert Files
  • Drag and Drop Files

Before you begin: Create or open a representation.

Insert Files

  1. From the Annotation section of the action bar, click Insert From File .

  2. In the Import From dialog box, define the path of the file and select the convenient file format in the list.

  3. Select a file in the File name box.

  4. Click Open.

    Note: While importing a .csv file:

    • You need to define the position of the table.
    • If you clear the Do not split table on several sheets check box from the Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems > Drafting > Annotations and Dress-Up section, and if the indicated location requests it, the created table is split on several sheets.

The file is imported into the representation.

Drag and Drop Files

You can drag and drop files onto the window. It is similar to importing files in a using the Insert From File command.

Important: You can drag and drop only the files with extensions, .dxf, .dwg, .cgm, .ps, .gl2, .svg and .csv.

  1. Select the required file.

  2. Drag it from its location and drop it to the window.

    Notes:

    • If you are dragging a .csv file, you need to define the position of the table, once you dropped it onto the window.
    • You can drop multiple files at the same time. They are imported one after another.

The file is imported into the representation.

Note: Importing a .svg file using the above described methods imports the file as 2D elements. To import a .svg file as an image, use the Insert Picture command. .
