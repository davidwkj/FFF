Instantiating a Label

You can instantiate a label from a label template.

A label template is a label instance located in the main view of a detail sheet, on which a frame and/or a leader might have been added.

This task shows you how to:

  • Instantiate a Single Label
  • Instantiate Several Labels

Before you begin:
  • Create a label template.
  • Double-click the detail sheet.

Instantiate a Single Label

You can select a reference label and create a single instance of the label on a selected target.

  1. From the Annotation section of the action bar, click Instantiate Label .

  2. Select a label template (reference label) to instantiate.

    Tip: You can instantiate a label also using either of the following ways:
    • Right-click the label template and select Instantiate Label.
    • Instantiate a label from a catalog.

  3. Perform either of the following operations:

    • Click anywhere in the free space to indicate the label anchor point (if the label template is created without leader).
    • Indicate the leader anchor point (if the label template is created with leader).
    • Click in the free space to define the position of the label.

    Note: You can position the label on any existing annotation by selecting its attachment point.

    The label instance is created.

    Important: As soon as the label is created, if it contains some annotations with link templates, they are routed on the elements used as targets. If there is any remaining unsolved link template, the Route Link Template dialog box appears after the label instantiation, using which you can resolve the link template.

  4. Select the object to which the compatible link template is to be routed or resolved.

    The link template is resolved and the label instance is created with the relevant information. If required, you can use the contextual command Route Link Templates to reroute the existing template.

Instantiate Several Labels

You can select one or several targets before selecting the reference label and create several label instances.

  1. Select the target elements (annotations, points, etc.) by trapping them using the pointer.

  2. Follow the steps from 1 through 4 as explained in Instantiate a Single Label.

    Label instances are created at the selected target elements.
