Edit a Label

You can edit a label by modifying its content (a text or a table) if it is defined as modifiable in instance in the reference. You can also manipulate a label using its handles.

  1. Double-click the label or right-click the label and select Label.x Definition.

    The label is in edition mode.

  2. Right-click the text in the label and select Modifiable in Instance.

    The text becomes modifiable in instance, i.e. it can be edited after it is instantiated.

  3. Right-click the text again and select Set Master Annotation.

    The annotation is set as the master annotation and all the font and text properties are now handled by this master annotation and no more by the label itself. The leader is now attached to the master annotation.

    Note: The Set Master Annotation contextual command of a text or a table contained in a label is:

    • Available for labels located in the main view of a detail sheet, when they are in edition.
    • Available only when a single annotation is selected.
    • Not available in the context menu of a text or a table which is already the master annotation of the label.

  4. Optional: Right-click the text and select Clear Master Annotation.

    The annotation is no more master annotation. The leader is attached back to the label.

    Note: The Clear Master Annotation contextual command of a text or a table contained in a label is:

    • Available for labels located in the main view of a detail sheet, when they are in edition.
    • Available if at least one annotation is selected.
    • Available only when a master annotation has already been defined for the label.

  5. Double-click the text.

    The Text Editor and the Insert Link Template dialog box appear.

    Note: The Text Editor dialog box lets you edit the annotation only when the annotation in the label is declared as Modifiable in instance. You can modify the properties of all such annotations of the label using Object Properties. However, you cannot edit label properties using Object Properties, once the label is in edition mode.

  6. Optional: Edit the text or insert the link template and click Ok in the Text Editor dialog box.

  7. Instantiate the label.

    The label instance is created in the design sheet of a .

  8. Optional: Select the label using any of its sub parts including annotations declared as modifiable in instance.

    Handles are displayed, using which you can move or drag the label as per your requirement.

  9. Optional: Rotate or drag the label to the required position.

  10. Optional: Change the symbol shape of the yellow anchor point or add a breakpoint to the leader.

    The label is modified as per your requirement. For more information about adding a breakpoint to the leader, see Moving and Positioning Leader Breakpoints.

Replace a Label Content

You can replace a label content with a 2D component reference or the content of another label.

  1. Right-click a label and select Label.x object > Replace Content.

    Tips:
    • You can replace the content of several labels at once by multi-selecting the labels before executing Replace Content.
    • You can also add content to an empty label by double-clicking the label and selecting Replace Content.

  2. Select a 2D component reference or another label to replace the content.

    If you select a 2D component reference, a message box appears prompting you whether you want to delete the 2D component reference used to define the label content.

    The label content is replaced.

  3. Optional: Click Yes to delete the 2D component reference or click No to keep the reference as it is.
