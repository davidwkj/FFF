    • If no selection is made, the scope of the legend table is taken considering the root of the current . Accordingly, a message appears that the legend table is created for all the .

  5. In the Legend Table dialog box, under Title, enter the required title for the legend table in the Table box.

    You can also define the title or heading of the columns of the table in the Title area. These titles are displayed in the two header rows of the table.

    Note: The Table box is empty, by default. However, once you enter the title, it is memorized (as a preference), and set as the default value for the next usage.

  6. Optional: Enter the required heading in the Image, Legend, and Count boxes or keep the default headings.

  7. Optional: Under Count, select Add Total Count check box.

    This enables the Count box and a column is added in the legend table to display the total legend count.

  8. Optional: Select the Add Count per View check box.

    When this check box is selected, a column is added in the legend table for each view for the selected scope. The name of the corresponding column is automatically set according to each respective view name. If a view does not contain any symbol of a specific type, the corresponding cell contains the following symbol: "-".

  9. Under Format, select a Table Style.

    You can select a table style from the available styles.

  10. Optional: Select the Reverse Table check box to reverse the table rows from the bottom to the top.

    This inversion is done after the actual creation of the table. This check box is cleared by default.

  11. Optional: Select the Split Table check box to split the table if it is too large.

    By default, this check box is cleared.

  12. Click OK.

    Note: If no label or symbol is found in the defined scope or there is no legend to be reported, an error message appears and the command ends without creating any table.
