Visualizing the Print Area

You can visualize the print area (providing it is activated), so as to re-position or re-dimension it for example.

Before you begin: Open a drawing representation. Right-click a sheet and then select Properties. On the Sheet tab, make sure Activate is selected in the Print area section.
Related Topics
Printing a Sheet

  1. Right-click the sheet node in the tree and select Sheet.X object > Print Area > Edit Print Area.

    This zooms onto the print area, which is outlined as a purple dashed box, with an X cross at its center.

  2. Use the handles at the corners of the box to re-dimension the print area.

  3. Drag the dashed box or the central X cross to re-position the print area.



    You can then exit the print area visualization mode by pressing the Escape key or by clicking elsewhere in the drawing. You can check the sheet properties to make sure that the coordinates, width or height have been updated.

  4. Change the sheet properties as wanted, and then click OK to validate.
