Managing a Sheet

You can add, delete, rename, or edit sheets. A sheet is opened by default when a is created, and it is displayed in the Sheet Browser and in the tree.

This task shows you how to:

  • Add a New Sheet
  • Delete a Sheet
  • Edit a Sheet or its Background
  • Rename a Sheet

Before you begin:
  • Open a representation.
  • From the View section of the action bar, click Sheet Browser to view the sheet browser which is docked at the top of the work area.

Add a New Sheet

You can add a new sheet in a using the Sheet Browser.

  1. In the Sheet Browser, click Add.



    Tips: You can also right-click Add and select New Sheet. Or you can right-click the node in the tree and select New Sheet.

    The new sheet is created and is automatically set as the current one. It is listed in the tree.

    Note: You can add a new sheet after this one, by clicking Add. This new sheet is assigned with the same standard, format, and orientation as the first created sheet (default option).

  2. Optional: In the tree, double-click the sheet to activate it or select it in the Sheet Browser.

Delete a Sheet

You can delete a sheet in a .

  • Right-click the sheet in the tree or in the Sheet Browser and select Delete.

    The sheet is deleted.

Edit a Sheet or its Background

You can edit a sheet or its background in a .

  1. To edit a sheet's background, right-click the sheet in the tree or in the Sheet Browser and select Edit Background.

    You can now start creating a frame and title block for example.

  2. To edit a sheet, right-click the sheet in the tree or in the Sheet Browser and select Edit Sheet.

    You can now start creating views.

Rename a Sheet

You can rename a sheet in a . Only a current sheet can be renamed.

Before you begin: Ensure that the sheet you have to rename is the current sheet.
  • You can follow the methods given below:
    1. Double-click the sheet in the Sheet Browser.
    2. Edit the name of the sheet.
    3. Click anywhere in the work area or press Enter.

    Or

    1. Right-click the sheet in the tree and select Properties.
    2. In the Sheet tab, edit the name of the sheet in the Name box.
    3. Click OK.

The sheet is renamed in the Sheet Browser and the tree.

Note: If the Ensure uniqueness of sheet names check box is selected in Me > Preferences > All Preferences > 3D Modeling > Mechanical Systems Administration section, under Naming Uniqueness, then you cannot name two sheets alike. If you do so, the name of the latest sheet is changed automatically. A message appears on the upper-left corner of the work area to prompt the same.
