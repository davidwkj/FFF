Reordering a Sheet or a View

You can reorder sheets, detail sheets or views in the tree of a representation.

Important:

  • Only views that belong to the same sheet can be reordered.
  • Only sheets that belong to the same can be reordered.
  • Detail sheets are reordered separately, after the regular sheets.

Before you begin: For this scenario, create four sheets and two detail sheets in a new representation.

This scenario describes the command in the Drawingsapp context, but it is identical to the 2D Layout for 3D Designapp.

  1. In the tree, select one or more sheets or views you want to move.

    In this scenario, Sheet.3 and Sheet.4 are to be moved.

  2. Right-click the selected sheets and select Selected objects > Reorder.

    The pointer changes to let you select a new place for the sheets 3 and 4.

    • : Select a sheet with the downward pointer to move the sheets 3 and 4 under the selected sheet. By default, this pointer is displayed.
    • : Select a sheet with the upward pointer to move the sheets 3 and 4 above the selected sheet. To display this pointer, hold down Ctrl.

  3. For this scenario, select Sheet.1 with the downward pointer.

    The sheets 3 and 4 are reordered under the Sheet.1.
