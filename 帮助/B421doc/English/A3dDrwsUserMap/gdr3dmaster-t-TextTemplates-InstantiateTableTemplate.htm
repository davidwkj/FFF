Instantiating Table Templates

You can instantiate the table template stored in a current or a catalog. This task shows you how to:

  • Instantiate Table Templates Stored in a Catalog
  • Instantiate Table Templates Stored in the Current
  • Route the Link Templates

Before you begin: . Open the in which you want to instantiate table templates.

Instantiate Table Templates Stored in a Catalog

You can instantiate a table template stored in a catalog.

  1. From the Tools section of the action bar, click Catalog Browser .

    dialog box appears.

  2. In the Catalog Browser panel, select the catalog and then the chapter that contains the table templates you want to use.

    The table templates are displayed in the right-hand pane, in the Item name column.

  3. Right-click a table template and select Use Item > Instantiate Table.

  4. Select either of the following options to locate the instantiated table:

    • Select a reference table.
    • Select a horizontal or a vertical line.

      The table is created and a balloon is displayed to position the table in the required way. Simultaneously, the Route Link Template dialog box appears.

      The table cells containing the link templates are highlighted in the table.

    • Indicate the table anchor point by clicking in the sheet at the required location.

      The table appears at the indicated location along with the Route Link Template dialog box.

  5. in the table.

Instantiate Table Templates Stored in the Current

You can instantiate the table template stored in the current .

  1. From the Annotation section of the action bar, click Instantiate Table .

    You can also instantiate an existing table by right-clicking it and selecting Instantiate Table.

  2. In the detail sheet, click the table template you want to instantiate.

  3. Double-click the sheet which contains the element you want to select as a reference.

  4. Select a reference table, a horizontal or a vertical line or indicate a point in the sheet to locate the instantiated table.

    Note: You can select several reference elements (lines) prior to selecting the Instantiate Table command. If several lines are selected, the first point of the line is considered as the position point and the tables are created in the . If several tables are selected, they are ignored and removed from the selection.

  5. in the table.

Route the Link Templates

You can resolve the link templates using targets. However, if a relevant package is not loaded in the session, a message is displayed informing to load the corresponding package.

Before you begin: Consider the following table template example:

  1. Select a table cell.

    The selected table cell appears in blue. Simultaneously, the Route Link Template dialog box appears.

  2. Select the target to resolve the link. For example, in this scenario, select the area fill.

    The link is routed and the table cell displays the value of the corresponding attribute.

  3. Successively select another appropriate target to route the remaining link (in this example).

    If any one of the link templates has not been routed associatively, the cell remains red. Additionally, corresponding message is displayed in the top right corner.

    However, if all the links in a table cell are routed associatively, the cell turns green.



  4. Select another cell and follow the steps from 1 through 2.

    The link template is immediately routed to the selected object, and the corresponding cell is unselected and highlighted in green if the link is routed associatively.

    Notes:

    • You can also instantiate a table template without routing the link templates it contains.
    • You can also query links in the table templates by right-clicking a table cell and selecting Query Object Links.
