Storing Text and Table Templates in a Catalog

You can store a text template and/or a table template in a catalog. This task shows you how to:

  • Create a Catalog
  • Store a Text/Table Template in an Existing Catalog

Before you begin: Create a text template and a table template and make sure the containing them is propagated to the database.

Create a Catalog

You can create a catalog.

  1. In the top bar, select Add > Generate Catalog.

  2. In the Catalog dialog box, enter a name for your catalog and click OK.

The catalog is created. All the detailed sheets of the current drawing are turned into chapters of the catalog. The chapters contain all the text templates (with or without link templates), table templates (with or without link templates) and/or 2D components of the .

Note: In each chapter containing text templates or table templates the type of the link template is specified, whenever possible. The preview displays an additional blue frame when a table contains at least one link template.

Store a Text/Table Template in an Existing Catalog

You can store a text template or a table in an existing catalog.

  1. Open an existing catalog.

  2. Right-click the chapter in which you want to insert the text/table template.

  3. Select Chapter > Insert Item: Feature.

  4. From the detail sheet, select the text template/table template.

The text template/table template is inserted in the selected chapter.

Important: For more information about catalogs, see the Design IP Classification User's Guide.
