Customizing an Action Bar by Adding Commands

You can customize an action bar by adding or removing commands according to your needs. You can also add the commands which are not directly available in any of the action bar or menus.

  1. Select Me > Preferences > Customize.

  2. In the Customize dialog box, select the Sections tab to view the list of commands.

    The Categories list allows you to filter the listed commands by category. For more information, see Customizing Options.

  3. Select the required section and click Add commands....

  4. In the Commands list, select the required command.

  5. Click OK.

    The command is added to the selected section of the action bar. For more information, see Customizing Options.

  6. Optional: To remove a command from a section, in the Customize dialog box, select the Sections tab.

  7. Select the required section and click Remove commands....

  8. In the Commands list, select the required command to be removed.

  9. Click OK.

    The command is removed from the selected section of the action bar
