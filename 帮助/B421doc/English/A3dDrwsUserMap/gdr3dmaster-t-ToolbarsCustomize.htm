Customizing Properties Boxes

You can customize the appearance of some boxes in the following properties: Styles, Graphic Properties, Text Properties, Dimension Properties and Position and Orientation.

  1. Right-click the properties box you want to customize.

    A context menu appears.

  2. If necessary, scroll down this context menu to display the customization options you can apply:

    • Set text width

    • Set list width

    • Set list height

    • Icons display

    • Set step

    • Set precision

    Important: The options available depend on the selected box. For more information about what options are available for each box, see About Customizing Properties.

  3. Click the required option.

    Depending on the option you selected, the corresponding dialog box appears.

  4. Enter the appropriate value in the dialog box:

    • Set text width: Sets the width used to display the box in the properties, in number of characters to be displayed (based on 'W').

    • Set list width: Sets the width used to display the list, in number of characters to be displayed (based on 'W').

    • Set list height: Sets the height used to display the list, in number of lines to be displayed (up and down arrows will make it possible to scroll within the list).

    • Icons display: Defines whether icons should be displayed in this box, or only in the list, when the list is collapsed.

    • Set step: Sets the step which is used to increment or decrement the value in the related box, using the available arrows.

    • Set precision: Sets the precision used to display a numerical value in this box, in number of digits after the separator.

  5. Click OK to validate.

    The properties box is modified.
