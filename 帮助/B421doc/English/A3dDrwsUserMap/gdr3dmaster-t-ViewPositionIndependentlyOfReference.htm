Positioning a View Independently of its Reference View

You can position a view independently of its reference view.

Note: For this scenario, we will position a right projection view independently of its parent front view.

During creation, views are by default positioned according to the parent (reference) view. A projection view moves along a line passing through its own axis and the reference view axis.

Before you begin: Open or create a containing geometry.
Related Topics
Positioning a View

  1. Right-click the frame of the view that you want to position independently of its reference view and select View Positioning > Position Independently of Reference View.

  2. Drag the view to the required location.



Important:
  • You can perform the Position Independently of Reference View command on one particular view at a time (you cannot select several views in this case).
  • If you want to reposition independently positioned views according to their reference views, repeat the scenario described above, by selecting the Position According to Reference View command from the displayed context menu.
