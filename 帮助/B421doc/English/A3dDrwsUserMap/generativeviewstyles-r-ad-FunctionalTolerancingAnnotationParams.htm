3D Tolerance & Annotation Parameters

3D Tolerancing & Annotation parameters are located in the FunctionalTolerancingAndAnnotation node of the generative view style XML file (available through Me > Preferences > Standards > generativeparameters category > DefaultGenerativeStyle.xml ).

The parameters specify the 3D typed Note Object Attribute specifications to be extracted as hatched areas to represent some technological information in 2D generative view. They are detailed in the table below.

  • NoteObjectAttribute

NoteObjectAttribute

Parameter name Value Description
TechnologicalAreaGeneration > TechnologicalAreaGenerationMode Yes/No Specifies whether the technological area should be extracted.
TechnologicalAreaGeneration > TechnologicalAreaForFTA > NOAExtractedType String

Specifies the Note Object Attribute to be used as the 3D technological specification. You can specify one Note Object Attribute type definition.
TechnologicalAreaGeneration > TechnologicalAreaForFTA > NOAPatternName String Specifies the pattern name to be used by the view update process for the initial creation of the 2D generated hatched area.

Note: Views generated without using generative view style with these parameters will not have 3D typed Note Object Attribute extracted as hatched area.
