Creating a Resource Table

You can create a resource table and associate resources with this table. After associating the table with a collaborative space, the resources of the table will be available to the users.

  1. From the Compass, click Social and Collaborative Apps , and select Data Setup.

  2. In the Resource Tables tab, right-click the desired Resource Set ID and select Create a New Resource Table.

  3. Enter the resource set name and click OK.

  4. Select the row to which you want to assign a resource and double-click the Unset resource cell.

    Notes:

    • You can also right-click the Unset resource cell and select Choose the assigned resource to allocate the resource.

    • Items in a resource table marked with a + icon or can have multiple sub items.

  5. Type the name of the resource in the Search box and click and click Search.

  6. Select the resource from the list.

    The resource is assigned.
