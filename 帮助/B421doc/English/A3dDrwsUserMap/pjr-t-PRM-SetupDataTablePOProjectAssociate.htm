Associating a Resource Table with a Collaborative Space

If you want your resources to be used by other users, you must associate it with a collaborative space.

Note: The collaborative spaces displayed in the tree depend on your credentials. To know more, see: Installation and Administration: 3DEXPERIENCE Platform: 3DEXPERIENCE Platform - Administration: Customization: People and Organizations: Managing People, Organization and Security Data.

Before you begin: You must have created a resource table.

  1. Click the resource table in the Resource Tables tab.

  2. Drag and drop it to the Collaborative Project bindings node.

    This icon... Indicates that...
    No table is bound to the selected collaborative space.
    A table is bound to the selected collaborative space.

    The resource table and the collaborative space are associated.

    Notes:

    • To unbind a resource table, right-click the resource table in the Collaborative Project bindings tree and select Unbind the resource.
    • If the collaborative space already contains a table with the same resource set Id as the dragged one, the association does not occur.
