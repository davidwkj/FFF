2D Layout for 3D Design Resources

A resource is a PLM object (for example, a catalog or a business rule) or another type of file (for example, a bitmap or a spreadsheet). In Data Setup, resources are grouped in resource sets. A resource set is a PLM object that implements a single resource table. This topic lists available resources in Drafting and 3D Master Resources and Standard Resources.

You must be assigned the role of an Owner to set these resources and bind to your collaborative space.

Drafting and 3D Master Resources
Logical Name Resource Type Description
Standards for Drafting - OBSOLETE XML Document Allows to customize the Standards used for Drafting. This is an obsolete resource.
Standards for 3D Master - OBSOLETE XML Document Allows to customize the Standards used for 2D Layout for 3D Design, 3D Tolerancing & Annotation and Generative Functional Tolerancing. This is an obsolete resource.
Generative View Styles - OBSOLETE XML Document Allows to customize Generative View Styles used by 2D Layout for 3D Design and Drafting.This is an obsolete resource.
General Tolerances Sheet Allows to add customized general tolerances used for 3D Tolerancing & Annotation and Generative Functional Tolerancing.

Important:

  • When you add the customized tolerances in the resource excel sheet, follow the below guidelines:
    • In the first column, define the standard and dimension type, e.g. ISO.LinearSize.

      This decides for which conventional standard (ISO, ANSI-ASME, or JIS) the customized standard is going to appear.

    • In the second column, define the general tolerance standard and class, e.g. ISO 2768 - f.
    • In the third and the fourth column, and every third column after, define the nominal values range in increasing order.

      The variation lower limit and upper limit for the nominal value range need to be defined in the column next to the larger nominal value column.

    • The serial column number need to be specified till the last column of the sheet (with at least one filled cell value).
  • The customized general tolerances does not appear under General tolerance class available in Me > Preferences > Mechanical > Tolerances tab.
  • When you open a 3D model with dimension having general tolerance values which are not defined in the resource sheet, only the standard name is displayed in the tooltip. When you edit the dimensions, they are not proposed in the list.

Accessing Resources

When you bind a resource table to your collaborative space, you can directly access its resources from the server, whenever necessary. To access the other standards and generative view styles on your computer, you have to switch to offline mode.
