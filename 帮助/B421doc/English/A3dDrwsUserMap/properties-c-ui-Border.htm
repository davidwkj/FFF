Border Properties

Border lets you modify table borders.

Select the Border tab to access properties.

Important:

These properties are only available when the entire table is selected.

  • Lines and Curves
  • Presets
  • Border

Lines and Curves

Color

Defines the color for all table borders.

Line Type

Defines the line type for all table borders.

Thickness

Defines the line thickness for all table borders.

Presets

None

Removes borders from the selected cells.

Outline

Adds borders around the selected cells.

Inside

Adds borders inside the selected cells.

All

Adds borders around and inside the selected cells.

Border

Use the icons around the preview and customize borders for the selected cells.

You can apply slash and/or back-slash diagonal borders on one or several cells. The cell border type used for newly-inserted cells is based on the cell border type of the preceding row or column in the table.

Notes:

  • When merging cells, the border type of upper-left merged cell is displayed. You can apply the diagonal cell border type on merged cells (which are then considered as a single cell).
  • When editing cell properties, note that a number of properties do not apply to the selected cell, but to the table and all its cells.

    Namely, on the Text properties:

    • You can apply the diagonal cell border type on merged cells. In this case, merged cells are considered as a single cell.
    • X and Y position
    • Reference
    • Orientation
    • Blank Background

    On the Graphic tab (Lines and Curves section):

    • Color
    • Linetype
    • Thickness

    On the Border tab (Lines section):

    • Color
