Chamfer Properties

Chamfer lets you modify chamfer properties.

Select the Chamfer tab to access properties.

  • Value Format
  • Numerical Display
  • Representation Type

Value Format

Offers the following options:

Distance x Distance
Displays the chamfer format with two distances.
Distance x Angle
Displays the chamfer format with one distance and an angle.
Distance
Displays the chamfer format with one distance.
Angle x Distance
Displays the chamfer format with an angle and one distance.

Numerical Display

Tip:

For a full description of the selected tolerance type, click information icon on the left of the box.

Main description
Defines the dimension format for the main value.
Display
Displays the factor applied to the dimension: one, two or three factors.
Dual description
Defines the dimension format for the dual value.
Display
Displays the factor applied to the dimension: one, two or three factors.

Representation Type

The following options are available:

Two Symbols
Defines a chamfer representation with two symbols.
One Symbol
Defines a chamfer representation with one symbol.
