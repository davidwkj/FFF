Dimension Line Properties

The Dimension Line lets you modify dimension properties.

Select the Dimension Line tab to access properties.

  • General
  • Symbols
  • Foreshortened

General

Representation
Defines the dimension line representation:
  • Regular
  • Two Parts
  • Leader one Part
  • Leader two Parts
Color
Defines the dimension line color.
Thickness
Defines the dimension line thickness.
Extension
This property is only applicable to the following dimensions:
  • Radius dimension
  • Diameter one-symbol dimension

Defines the dimension line extension type:

  • From standard
  • Till center
  • Not till center

Leader
Angle
This property is only available with the Leader one Part or Leader two Parts properties of Representation. Defines the dimension line leader angle.
Length
This property is only available with the Leader two Parts property of Representation. Defines the dimension line leader length.
Second Part
This property is only available with the Two Parts or Leader two Parts properties of Representation. Defines the second leader part properties.
Reference
Defines the second leader part reference orientation according to the:
  • Screen
  • View
  • Dimension Line
Orientation
Defines the dimension value orientation. Different options are available for different types of references, described as follows:
Reference Orientation
Screen Horizontal and Vertical
View Horizontal and Vertical
Dimension Line Parallel, Perpendicular, and Fixed Angle
Angle
This property is only available with the Fixed Angle property of Orientation. Defines the dimension value orientation angle.

Symbols

Symbol 1, Symbol 2, Leader Symbol
Defines the dimension line symbols.

Symbol 2 is available with the a dimension line containing a second symbol.

Leader Symbol is available with the Leader one Part or Leader two Parts properties of Representation.

Shape
Defines the dimension line shape:
  • No Symbol
  • Open Arrow
  • Outlined Arrow
  • Filled Arrow
  • Transparent Arrow
  • Slash
  • Outlined Circle
  • Filled Circle
  • Transparent Circle
  • Crossed Circle
  • Outlined Triangle
  • Filled Triangle
  • Plus
  • Cross
  • Double Filled Arrow
Important:
Color
This property, in Leader Symbol, is not editable and reuse the color property defined for Symbol 2. Defines the dimension line color.
Thickness
This property, in Leader Symbol, is not editable and reuse the color property defined for Symbol 2. Defines the dimension line thickness.
Reversal
Defines the orientation of dimension line shape:
  • Auto
  • Inside
  • Outside
  • Symbol 1 In/Symbol 2 Out
  • Symbol 1 Out/Symbol 2 In

Foreshortened

Defines whether a radius dimension is transformed into a foreshortened radius or not. This property is available for radius dimension.

Important:

In the case of foreshortened radius dimensions created with V6 onwards:

  • The dimension value remains underlined when it is moved outside the dimension line.
  • The dimension value can be moved either on the short segment or on the long segment of the dimension.
  • The symbol is reversed when moving the dimension value over the extension line or the geometry.
Text position
Positions the dimension text along the following:
  • Short Segment
  • Long Segment
Orientation

Defines the orientation of the dimension text relative to the dimension line:

  • Parallel
  • Convergent
Angle
Defines the foreshortened angle.
Ratio
Defines the foreshortened ratio between the short and the long segment.
Point Scale
Defines the foreshortened extremity point scale.
Position extremity point manually.
Defines whether the foreshortened extremity point can be move by the user.
