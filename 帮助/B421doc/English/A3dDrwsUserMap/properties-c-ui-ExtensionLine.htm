Extension Line Properties

The Extension Line lets you modify dimension properties.

Select the Extension Line tab to access properties.

  • Extremities
  • Funnel

Color
Defines the extension line color.
Thickness
Defines the extension line thickness.
Displays first extension line
Defines whether the first extension line is displayed.
Displays second extension line
Defines whether the second extension line is displayed.
Slant
Defines the slant angle for the extension line. This angle is set between 90 degrees and -90 degrees excluded. This property is only applicable on linear dimension line, and the line linking extension line anchor points (blanking excluded) has to be parallel to the dimension line.

Extremities

Overrun
Defines the extension lines overrun.
Blanking
Defines the extension lines blanking.

Funnel

Funnel
Defines whether a funnel is applied to the extension line or not.
Height
Defines the funnel height.
Angle
Defines the funnel angle.
Width
Defines the funnel width.
Funnel mode

The following options are available:

External
Defines an external funnel.
Internal
Defines an internal funnel.
Funnel side

The following options are available:

Left or Bottom
Defines the funnel side: left or bottom.
Right or Top
Defines the funnel side: right or top.
Both Sides
Defines the funnel on both sides.
