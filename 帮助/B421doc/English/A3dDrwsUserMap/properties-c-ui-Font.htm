Font Properties

The Font lets you modify font properties for Coordinates Dimension, Datum, Datum Target, Dimension, Flag Note, Geometrical Tolerance, Note Object Attribute, Roughness, or Text.

Select the Font tab to access properties.

  • Attributes
  • Character
  • Preview

Font
Defines the font name.
Style
Defines the font style.
Size
Defines the font size.
Underline
Defines the underline effect, this property is not compatible with Overline property:
  • None
  • Underline
Color
Defines the font color.

Attributes

Strikethrough
Applies the strikethrough effect.
Overline
Applies the overline effect. This property is not compatible with Underline property.
Superscript
Applies the superscript effect.
Subscript
Applies the subscript effect.

Notes:

  • You can either underline or overline a text.
  • When you are using a font stroke for annotations, the character thickness is set to 1 for regular style and 3 for bold style.

    You can customize standard files to remove this parameter from the Thickness box so that it cannot be applied to annotations characters.

Character

Ratio
Defines the character ratio.

Note: The Ratio option is not available for TrueType fonts.

Slant
Defines the character slant according to the font capabilities.
Spacing
Defines the character spacing.
Pitch
Defines the character pitch according to the font capabilities.

Notes:

  • The Slant and Pitch options are available only for stroke fonts.
  • The pitch of some stroke fonts cannot be modified. In that case, the Pitch is disabled.
  • Set appropriate spacing for fonts which allow no or little spacing between characters. This avoids superimposition of characters.
  • In case of stroke fonts that are Asian fonts, the ISO 646 characters are always shown using variable pitch.

Kerning
Defines that the character should be kerned according to the font capabilities.

For more information, see About Fonts.

Note: The Kerning option is available only for OpenType fonts.

Preview

Previews the selected properties and the font type.

It also indicates the font type.
