Label Properties

Label lets you modify the label properties.

Select this tab to access Label content properties.

Content

Color
Defines the color that is used for all the sub-elements of the label.

  • When No Color is defined, the values of each sub-element of the label are taken into account for display.
  • When you define this property (by choosing required color from the list), it overrides the properties of all the sub-elements of the label for display.

Linetype
Defines the linetype that is used for all the sub-elements of the label.

  • When No Linetype is defined, the values of each sub-element of the label are taken into account for display.
  • When you define this property (by choosing required linetype from the list), it overrides the properties of all the sub-elements of the label for display.

Thickness
Defines the thickness that is used for all the sub-elements of the label.

  • When No Width is defined, the values of each sub-element of the label are taken into account for display.
  • When you define this property (by choosing required thickness value from the list), it overrides the properties of all the sub-elements of the label for display.
