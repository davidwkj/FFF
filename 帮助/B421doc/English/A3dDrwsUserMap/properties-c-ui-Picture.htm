Picture Properties

Picture lets you modify picture properties.

Select the Picture tab to access properties.

  • Position
  • Stretch picture to fit cell size
  • Size
  • Scale
  • Original Size
  • Crop from
  • Compression

Position

This property appears for a picture inserted in a view. It replaces the Stretch picture to fit cell size area which is displayed for a picture inserted in a table cell.

Horizontal
Defines the horizontal positioning on the sheet, taking the lower left corner of the image into account.
Vertical
Defines the vertical positioning on the sheet, taking the lower left corner of the image into account.

Stretch picture to fit cell size

This property appears for a picture inserted in a table cell. It replaces the Position area which is available for a picture inserted in a view.

Stretches the picture to fit the cell size, a stretched picture is scaled (stretched or shrunk) uniformly.

Clear this property to display the picture in its original size.

When a non-stretched picture is smaller than its cell, then the picture is displayed at one of the nine possible positions.

Size

Width
Defines the ratio between the original width and the represented width.
Height
Defines the ratio between the original height and the represented height.

When you modify the size, the scale is automatically modified at the same time.

Reference
Defines the orientation reference in:
  • The sheet
  • The view for design sheets
  • The 2D component for and detail sheets
Orientation
Defines the picture orientation to choose between:
  • Horizontal
  • Vertical
  • Fixed Angle: Enables to specify an angle in the selected reference through Angle.

    When Fixed Angle is selected, the Angle is set to 45.000 deg by default.

    If a value corresponding to a common orientation is identified such as 0°, 90°, etc. it is presented as Horizontal, Vertical next time the Properties dialog box is opened or when changes are applied.

Angle
Defines the angle that is related to the sheet, view or 2D component depending on the Reference property of the picture.
The angle that it displays is related either to the sheet or to the view/2D component/symbol, depending on the Reference property of the picture, which cannot be modified by this field. Please note that a support of pictures is also provided for X and Y positions in this Object Properties panel.

Scale

Width
Defines the width in percentage of the picture.
Height
Defines the height in percentage of the picture.
Lock aspect ratio
Preserves the ratio between width and height when using the corner handles.

However, when using the border manipulators, the picture can be distorted.

Original Size

Width
Defines the original width of the picture in millimeters.
Height
Defines the original height of the picture in millimeters.
Reset
Resets the size of the represented picture to the size of the original picture.

Crop from

Top
Defines the top part of the picture to be hidden.
Bottom
Defines the bottom part of the picture to be hidden.
Left
Defines the left part of the picture to be hidden.
Right
Defines the right part of the picture to be hidden.

Compression

(Compression format)
Select the required option:
None
Does not compress the picture - keeps its original size.
PNG
Compresses the picture without loss in quality.
Important:

The PNG format is used by default when inserting a picture whatever its original format.

JPEG
Compresses the picture with a loss in quality.
(Compression rate)
Displays the compression rate in percentage and the size of the picture in bytes according to the chosen format once it is saved on your hard drive. The compression rate and picture size vary according to your picture.
