Reported Properties

Reported lets you modify the bill of material properties.

Select this tab to access Reported properties.

List of properties

Property
Specifies the properties that are displayed as columns in the bill of material.

For each property, a column is created in the bill of material.

The Property column displays the list of attributes available on a product.

By default, the bill of material is created using the default bill of material style which displays, among other properties, the product name, the item number and the quantity.

However, you can enrich the bill of material with as many properties as you want and/or you can change the existing ones.

Note that properties need to be valued/computed first before you can visualize them in the bill of material. For more information, see the relevant documentation on these properties.

You can select and modify an existing property. To do so, select the appropriate filter under Product Type Filter and the property associated with this filter under Property. Enter a title for this property in the Property Title box and specify a Column Width.

Notes:

  • The Product Type Filter lists all the defined PLM extensions.
  • If you select All Types under the Product Type Filter, all the properties across all PLM extensions are listed alphabetically.
  • Reported properties are not supported for heterogeneous selection of bill of materials whose list of reported properties are different, either in term of content or in term of order.
  • Reported properties are supported if the selected bill of material has the same list of reported properties and same order even if their titles and column widths are different.

Title
  • You can modify the title of the selected property.
  • For each property, a default name is set. However, you can change the default name using this field.
Column width
  • You can modify the column width of the selected property.
  • This column displays 3 decimal digits.
  • The width value is set using the drafting standard unit (mm/m/inch as specified in the Length list, see Paper Unit).
  • The minimum value is equal to twice the horizontal column margin (included).
  • The maximum value is equal to 1 meter (included).
  • You can type the value or use arrows to change the width value by 1mm.
New
Allows you to add a new reported property.

The new reported property is inserted at the end of the list and is initialized from the previously selected property.

Important:

This command is available only if a single line is selected.

Delete
Deletes one or more selected properties.
Important:

This command is available only if a single line is selected.

Move up
Moves the selected property up.
You can modify the order of properties. The first line matches with the first column of the bill of material (the first column of the bill of material displays the values of the property displayed here in the first line of the bill of material customization).
Important:
  • This command is available only if a single line is selected.
  • However, this command is unavailable if the selected line is the first one.
Move down
Moves the selected property down.
Important:
  • This command is available only if a single line is selected.
  • However, this command is unavailable if the selected line is the first one.
