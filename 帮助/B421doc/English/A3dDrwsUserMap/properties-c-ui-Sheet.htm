Sheet Properties

Sheet lets you modify sheet properties.

Select the Sheet tab to access properties.

  • Format
  • Print Area

Name
Defines a meaningful name for the sheet.
Scale

Specifies the scale applicable to all views in the sheet.

  • This implies that the scale of all existing views is multiplied by that of the sheet. For example, if existing views already have a scale of 1/10, and if you change the sheet scale to 1/10, then existing views will now have a scale of 1/100.
  • The scale does not determine the position of the views or any other object contained in the sheet.

    When the grid is displayed, it does not determine the position of the view.

The following sheet scale values must be considered:

  • Normal range: Scale must be greater than 1.10-5 and lower than 1.104
  • Large range: Scale must be greater than 1.10-7 and lower than 1.102

Note: If you specify a scale outside this scale range, an error appears displaying the scale limits for the current representation.

Format

Format
Contains the list of format names defined by the administrator in the Standards Editor.

For more details, see Sheet Format Definition.

You can also create your own user-defined formats, defined locally for a given drawing or layout. To create your own format, type a name for the format in the Format field. Then set values for the Width and theHeight fields. The name of the newly created format must be different from those in the user-defined lists and in the standard list of formats. If not, a warning message is displayed to inform that the format name is invalid.

Display
Displays the frame representing the format of the sheet.
Width
Specifies the width of the selected format. This field is available for user-defined formats only.
Height
Specifies the height of the selected format. This field is available for user-defined formats only.
Portrait
Applies the portrait orientation.

This option is available only if the selected format allows you to modify the orientation type, see Sheet Format Definition.

Landscape

Applies the landscape orientation.

This option is available only if the selected format allows you to modify the orientation type, see Sheet Format Definition.

Print Area

Activate
Specifies a specific area of the sheet that should be printed.

Doing this activates the associated fields so that you can define the print area.

In addition, you must select Document area option as the Print area in the Print dialog box so that the print area is printed.

If you do not select the Document area option, the whole document is printed.

X

Specifies the X coordinate of the lower left-hand corner of the print area.

Y

Specifies the Y coordinate of the lower left-hand corner of the print area.

Width

Specifies the width of the print area.

Height

Specifies the height of the print area.

(apply a format)

Defines a print format for a particular sheet size and its standard.
