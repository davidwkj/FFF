Tolerance Properties

The Tolerance lets you modify tolerance dimension properties.

Select this tab to access Tolerance properties.

Main Value, Dual Value

Main value and Dual value define respectively the main value and the dual value tolerance display.

Important:
  • These properties are only available with the Show dual value property of Value properties.
  • For a full description of the selected tolerance type, click information icon on the left of the box.
  • When you choose the same tolerance type for main and for dual value, then the values for this tolerance will also be the same.
Upper value
Defines the tolerance upper value. This option is only available for numerical tolerance display.
Lower value
Defines the tolerance lower value. This option is only available for numerical tolerance display.
First value
Defines the tolerance first value. This option is only available for alphanumeric tolerance display.
Second value
Defines the tolerance second value. This option is only available for alphanumeric tolerance display.
Important:
  • When the base unit of the dimension value and the Tolerance Display Factor correspond to a supported unit, see Dimension Tolerance Formats reference, you can edit the values in the Upper value and Lower value lists.

    However, when the base unit of the dimension value and the Tolerance Display Factor do not correspond to a supported unit, no unit is displayed in the Upper value and Lower value lists.

  • For dimensions with alphanumerical tolerances, you can display the corresponding numerical equivalents in the representation, simply by selecting the dimension and placing the cursor over the tolerance in the representation.

    The numerical equivalents are displayed in a tooltip.

  • For dimensions with tolerance js and JS, there is no correspondence between the numerical and alpha numerical value.

    The numerical value displayed is +-0 or the previous numerical value applied to the dimension.
