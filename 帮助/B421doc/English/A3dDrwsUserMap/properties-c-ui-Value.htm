Value Properties

The Value lets you modify dimension properties.

Select the Value tab to access properties.

  • Dimension Type
  • Value Orientation
  • Dual Value
  • Format
  • Fake Dimension

Dimension Type

Displays the dimension type name, projected dimension in our example.

Driving

Value Orientation

Reference
Defines the dimension value reference orientation according to the following:
  • Screen
  • View
  • Dimension line
Orientation
Defines the dimension value orientation:
  • Parallel
  • Perpendicular
  • Fixed angle
Angle
Defines the dimension value orientation angle.
Important:

These properties are only available with the Fixed Angle property of Orientation.

Position
Defines the dimension value position:
  • Auto
  • Inside
  • Outside
Offset
Defines the dimension value offset relative to the dimension line.

Dual Value

Show dual value
Defines whether a dual dimension value is displayed.
Location
Available location for the dual value are as follows:
  • Below
  • Fractional
  • Side-by-side

Format

Main Value, Dual Value
Defines the dimension format for the main and dual value.
Important:
  • These properties are only available with the Show dual value property of Value properties.
  • For a full description of the selected tolerance type, click information icon on the left of the box.
Description
Defines the numerical display format applied to the dimension.
Display
Defines the factor applied to the dimension.
Format
Defines the representation format for the decimal part of value, applied to the dimension and the tolerance:
  • Decimal
  • Fractional
Precision
Defines the precision format for the decimal part of value, applied to the dimension and the tolerance.

Fake Dimension

Defines whether the dimension value is faked.

Important:

Properties in this section are applicable for non-semantic dimensions only.

Representation
Defines fake dimension representation:
  • Numerical
  • Alphanumerical
Important:

When the Numerical representation is selected, the dimension value is updated according to the dimension geometrical support modification.

Nevertheless if you want to define a specific dimension value independently of its geometrical support, you have to select the Alphanumerical and type in your value.

Main Value, Dual Value
Defines fake dimension representation for the main and dual value.
Important:
  • These properties are only available with the Show dual value property of Value properties.
  • Texts for numerical fake dimensions are restricted to six characters. If you need to insert a text containing more than six characters, simply use the alphanumeric fake dimension.
