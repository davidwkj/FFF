Line Thicknesses Definition

Line thickness definitions are located in the Standard Definition dialog box, in the drafting category > Line Thickness node, available via Me > Preferences > Standards.

Related Topics
About Line Thicknesses Definition
Parameter Description
Availability Indicates whether this definition is available for users to choose from: 1 = available 0 = unavailable.
Pixels Specifies the size in pixels, with a maximum of 16; reflects the result displayed on screen.
mm Specifies the size in millimeters; corresponds to the printed version.

0 mm corresponds to the smallest thickness available on the device used for printing.
