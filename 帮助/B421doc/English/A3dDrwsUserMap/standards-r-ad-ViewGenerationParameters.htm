View Generation Definition

View generation definition parameters let you configure options that should be applied when generating views in a Drawings context only. These parameters are located in the View > Generation node of the Standard Definition editor, available via Me > Preferences > Standards.

Parameter Description
Thickness Index Lets you configure the line thickness for geometry which is automatically generated in views (this includes all geometry except fillet edges). Specify the number of the line thickness definition parameter, as specified in the Line Thickness node of the Standards editor. For more information, see Line Thicknesses Definition.
Material Cut Pattern This parameter is used when generating views from parts which use a material to which a specific pattern is associated.

  • Select Material to use the pattern associated to a given material (instead of the patterns defined in the standards), even if this pattern is not defined in the standards.
  • Select Standard to use standard patterns only, instead of the pattern associated to a given material. See Pattern Definition for more information about defining standard patterns.
