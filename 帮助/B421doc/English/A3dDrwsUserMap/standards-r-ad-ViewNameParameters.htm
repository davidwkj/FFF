View Name Definition

View Name stands for the name of the view. It is composed of a Name, an ID and a suffix. This section deals with the view name definition parameters, located in the Views > View Name node of the Standard Definition editor, available via Me > Preferences > Standards.

Related Topics
View Text Definition

You can configure the view name using the following parameters:

  • Automatic: Available values are:
    • Yes: The Name is automatically managed. It corresponds to the string associated to the View Type.
    • No: The Name follows the standard file definition. It corresponds to the Name parameter that is used.
  • Name: When Automatic parameter is set to No, it is used to fill the Name character string which is part of the View Name. If it is empty, the Name will be empty too.
