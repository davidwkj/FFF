Freeform Work Area Shortcuts

When working in the Freeform action bar, several handy shortcuts are available to you in the work area.

  • Freeform Work Area Options
  • Freeform Keyboard Shortcuts
  • Freeform Quick Command Context Toolbars

Freeform Work Area Options

The Freeform work area provides options to display, hide, or select freeform mesh items used to modify the shape.

You can drag the mesh points and faces to customize the shape. To assist with selecting the mesh entities, the following options are available.

Button Function
Any Entities. Selects any mesh entities.
Vertices. Selects vertices.
Edges. Selects edges.
Faces. Selects faces.

Note: Press o to select the faces around the freeform. Click near an edge of the face, to select the direction of the loop.
Edge Rings. Selects the ring edges of a loop.

Note: Press i to select the ring around the freeform.

Edge Loops. Selects the edge of a loop.
Face Loops. Selects the faces of a loop.

Note: Press o to toggle the direction of the loop.
Show Only Visible Elements. Filters out the freeform mesh items in back, not visible in the display.
Show Mesh Cage. Displays the mesh cage.
Transparency. Sets the transparency of the freeform.

When Transparency is selected, a slider appears allowing you to set the transparency from 0 to 50 percent.

Note: When the Transparency is not selected, the freeform is 100 percent opaque.

Freeform Keyboard Shortcuts

The Freeform keyboard shortcuts are key combinations that are only available when working with Freeform items in the work area.

Shortcut Function
Esc Exits the command.
Delete Quickly removes faces or edge loops.

When you select an edge and press Delete, the entire loop belonging to that edge is deleted.
Double-click
Freeform Action Bar Commands
Applies the current command and enables you to use the command again.
Freeform Entity Selections
Double-clicking an edge of the active freeform body selects the entre edge loop.
Double-clicking a face of the active freeform body selects the entre face loop.

Tip: Click the edge of the face to select the direction of the loop.
q
Freeform Action Bar Commands
When creating freeforms, cycles through the orientation of the freeform on the various planes.

After you click the 3D or 2D freeform button in the action bar and see the freeform in the work area, repeatedly press q to cycle through the orientation of the freeform on the various planes.

Freeform Entity Selections
When selecting mesh entities, cycles through alternative solutions with your selection.

After shift selecting freeform entities in the work area, press q to select alternative solutions.
m
Freeform Action Bar Commands
When creating freeforms, press m to toggle the freeform shape in and out of the mid-plane position.
i
Freeform Entity Selections
Selects the ring of edges after selecting the first freeform edge.
Selects the ring of faces after selecting the first freeform face.
o
Freeform Entity Selections
Selects the loop of edges after selecting the first freeform edge.
Selects the loop of faces after selecting the first freeform face.

Tip: Press o again to select the faces going in the other direction.
Shift + face
Freeform Entity Selections
Select a face and press the Shift key and click on non-consecutive faces to automatically select the faces between the selections.

After shift selecting freeform entities in the work area, press q to select alternative solutions.
Ctrl + entity
Freeform Entity Selections
Press the control (Ctrl) key and click any combination of entities.
Multiple loops can be selected by pressing the Ctrl key and clicking loops.

Freeform Quick Command Context Toolbars

The Freeform work area provides context toolbars for quick access to common functions available with the selected mesh item.

Depending on what you select in the freeform mesh, various context menus are available to you. As product enhancements are released, other options might be available. The following are a few of the possible context toolbars.

Context Toolbar Availability
Select from these options when you have multiple faces selected in the freeform mesh.
  • Crease Edges.
  • Uncrease Edges is available when a creased edge is selected.
  • Extrude Faces.
  • Subdivide Faces.
  • Opens the Quick Align dialog box, draw a spline used for the quick align and close the dialog box.
  • Opens the Quick Align dialog box, drag the quick align handles and close the dialog box.

  • Crease Edges.
  • Uncrease Edges is available when a creased edge is selected.
  • Insert Loops.
  • Opens the Quick Align dialog box, draw a spline used for the quick align and close the dialog box.
  • Opens the Quick Align dialog box, drag the quick align handles and close the dialog box.
  • Quick Fill is available when an edge of a hole is selected.
