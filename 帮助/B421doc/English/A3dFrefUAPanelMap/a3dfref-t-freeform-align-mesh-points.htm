Align and Quick Align Points on the Freeform Mesh

You can align freeform mesh points on objects using commands from the action bar or the context toolbar.

This task shows you how to:

  • Aligning Mesh Points
  • Quick Align

Aligning Mesh Points

You can align mesh points on a freeform object with other objects in the work area.

  1. From the Freeform section of the action bar, create a 2D or 3D freeform object.

  2. From the Freeform section of the action bar, click Align Mesh Points .

    Note: Press Esc to exit the command, double-click to apply the current command and use the command again.

  3. From the Align Mesh Points dialog box, click one of the following alignment options:

    Option Description
    Align Subdivision Elements. Aligns edges and faces to subdivision entities.
    Align Vertices. Aligns the selected vertices using the first two entities.
    Align Vertices to Curve. Aligns the vertices to the curve entity.
    Align Vertices to Entity. Aligns the vertices to the entity.
    Align Vertices to Body. Aligns the vertices to the body.

  4. Depending on your alignment option, select the entities to align.

    Option Description
    Selects vertices, edges, or faces to align with the entity.
    Selects vertices. The alignment is with the first two vertices or to the entity.
    Selects edges or faces. The alignment is with the first selection.
    Selects axis or curve entity to align with vertices.


    		Selects entity to align with vertices.
    Selects body to align with vertices.

  5. Optional: When Align Subdivision Elements is selected in the Align Mesh Points dialog box, click one of the following:

    Option Description
    Collinear or Coplanar. Aligns all selected items to the first selected edge or face.
    Parallel. Aligns parallel to the first selected edge or face.
    Orthogonal. Aligns orthogonal to the first selected edge or face.
    Coincident. Aligns all selected items to the same line or plane.

Quick Align

When working with 2D or 3D freeform objects, you can Quick Align multiple vertices using the context toolbar.

  1. Select two or more vertices in the mesh you want to align.

  2. From the context toolbar, click Quick Align .

  3. Drag the dynamic alignment line to the position you want.
