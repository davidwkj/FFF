Creasing Edges on a Freeform Face

You can adjust the crease between mesh faces on freeform objects.

  1. From the Freeform section of the action bar, create a 2D or 3D freeform object.

    Note: A 3D Freeform Box is used in this procedure.



  2. From the Freeform section of the action bar, click Crease Edges .

    Note: Press Esc to exit the command, double-click to apply the current command and use the command again.

  3. Select one or more faces in the freeform mesh you want to modify.

  4. Drag the Crease Edges slider to adjust the corners between rounded and square.
