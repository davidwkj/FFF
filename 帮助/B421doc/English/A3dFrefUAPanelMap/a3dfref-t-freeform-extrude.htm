Extruded Freeform from Sketch

You can extrude a sketch entity into a freeform object, then customize that shape into a unique design.

  1. Create a sketch entity.

    Note: The following 2D entity is used in this procedure.



  2. From the Freeform section of the action bar, click Extrude .

    The Extruded Freeform from Sketch dialog box is displayed.

  3. From the work area, click the following command options to adjust end conditions of the feature:

    Option Description
    Midplane. Extends the feature equally in both directions from the sketch plane. Specify overall length.
    Flip. Reverses the orientation of the feature and direction of the pull.

  4. From Profile Selection box, select the sketch entity.

  5. Click in the text boxes and specify the number of profile segments.



  6. Click in Extruded Freeform from Sketch dialog box.

  7. Optional: Drag the Robot arrows to position the freeform object.



  8. Optional: Drag the mesh points and faces to customize the shape of the freeform object.
